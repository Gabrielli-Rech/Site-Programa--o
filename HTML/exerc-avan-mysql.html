Exercícios Avançados de MySQL

  • 1. Crie uma consulta para combinar dados de duas tabelas usando `JOIN` com condições complexas.
  • 2. Utilize a função `CASE` em uma consulta `SELECT` para criar uma coluna condicional.
  • 3. Implemente uma subconsulta correlacionada que retorne os resultados de uma tabela com base em outra.
  • 4. Use a função `GROUP BY` e `HAVING` para encontrar registros que atendam a uma condição de agregação.
  • 5. Crie uma consulta que utilize `UNION` para combinar os resultados de duas ou mais tabelas.
  • 6. Implemente uma consulta que use a função `ROW_NUMBER()` para atribuir um número de linha a cada registro.
  • 7. Crie uma consulta que use a função `CONCAT()` para unir várias colunas em uma só.
  • 8. Use a função `DATE_FORMAT()` para formatar uma data em um formato específico.
  • 9. Realize uma consulta para retornar os `N` registros mais recentes de uma tabela com `ORDER BY` e `LIMIT`.
  • 10. Crie uma consulta que utilize `LEFT JOIN` para retornar todos os registros de uma tabela e os correspondentes de outra.
  • 11. Utilize `DISTINCT` para encontrar registros únicos em uma tabela.
  • 12. Crie uma consulta com `IN` e `NOT IN` para filtrar registros em uma tabela.
  • 13. Implemente uma consulta com a função `SUM()` e `COUNT()` para calcular a soma e a contagem de registros.
  • 14. Use a função `IF()` para criar uma coluna com valores condicionais em uma consulta `SELECT`.
  • 15. Realize uma consulta com `BETWEEN` para selecionar valores dentro de um intervalo.
  • 16. Crie uma consulta que use a função `LAG()` para comparar valores de uma linha com a linha anterior.
  • 17. Utilize `EXISTS` para verificar a existência de uma subconsulta.
  • 18. Implemente uma consulta para atualizar uma tabela com base nos valores de outra tabela.
  • 19. Crie uma consulta com a função `GROUP_CONCAT()` para concatenar valores de uma coluna em uma string separada por vírgulas.
  • 20. Realize uma consulta com `JOIN` que use uma condição de junção complexa e múltiplas tabelas.
  • 21. Utilize `CASE` e `GROUP BY` para criar categorias customizadas em uma consulta de agregação.
  • 22. Implemente uma consulta que use a função `COUNT()` e `GROUP BY` para contar a quantidade de registros por grupo.
  • 23. Realize uma consulta para encontrar os 5 produtos mais vendidos em uma tabela de vendas com `ORDER BY` e `LIMIT`.
  • 24. Crie uma consulta que utilize `SELECT INTO` para criar uma nova tabela com base nos resultados de uma consulta.
  • 25. Use `COALESCE()` para tratar valores nulos e fornecer valores alternativos.
  • 26. Implemente uma consulta para deletar registros de uma tabela que não existem em outra com `NOT EXISTS`.
  • 27. Utilize a cláusula `ROLLUP` para criar subtotais em uma consulta de agregação.
  • 28. Crie uma consulta que utilize `HAVING` e `GROUP BY` para filtrar grupos após a agregação.
  • 29. Realize uma consulta que utilize `SUBSTRING()` para extrair uma parte de uma string.
  • 30. Use a função `REPLACE()` para substituir substrings em uma coluna de uma tabela.
  • 31. Implemente uma consulta que utilize a função `STR_TO_DATE()` para converter strings em datas.
  • 32. Crie uma consulta para criar um índice único em uma tabela para uma ou mais colunas.
  • 33. Utilize a função `CURDATE()` e `DATEDIFF()` para calcular a diferença entre datas.
  • 34. Implemente uma consulta para criar uma tabela temporária com `CREATE TEMPORARY TABLE`.
  • 35. Crie uma consulta que utilize `UNION ALL` para combinar todas as linhas de duas consultas, incluindo duplicatas.
  • 36. Utilize a cláusula `LIMIT` para paginação de resultados em uma consulta complexa.
  • 37. Crie uma consulta que use a função `FIND_IN_SET()` para encontrar um valor em uma lista separada por vírgulas.
  • 38. Implemente uma consulta para encontrar duplicatas em uma tabela usando `GROUP BY` e `HAVING`.
  • 39. Realize uma consulta que utilize a função `DATE_ADD()` para adicionar um intervalo de tempo a uma data.
  • 40. Use `CASE` e `WHEN` em uma consulta `UPDATE` para aplicar condições específicas a atualizações de registros.
  • 41. Implemente uma consulta que use `ORDER BY` e `RAND()` para retornar uma lista aleatória de registros.
  • 42. Crie uma consulta que utilize `SELECT` e `DISTINCT` para encontrar entradas únicas de várias colunas.
  • 43. Utilize `LOCK TABLES` e `UNLOCK TABLES` para gerenciar a concorrência em uma consulta de atualização.
  • 44. Implemente uma consulta que utilize `CAST()` para alterar o tipo de dado de uma coluna em uma consulta.
  • 45. Crie uma consulta para usar `GROUP BY` em colunas calculadas com `CONCAT()` ou funções agregadas.
  • 46. Use `SELECT` e `ORDER BY` com `NULLS FIRST` e `NULLS LAST` para ordenar resultados com valores nulos.
  • 47. Realize uma consulta que utilize `ALTER TABLE` para renomear uma coluna de uma tabela.
  • 48. Crie uma consulta que use `JOIN` com uma condição de união com subconsultas.
  • 49. Utilize `EXPLAIN` para analisar o plano de execução de uma consulta e otimizar seu desempenho.
  • 50. Implemente uma consulta que use `IFNULL()` para substituir valores nulos por um valor default.
