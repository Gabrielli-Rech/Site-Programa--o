O que são Laços de Repetição?

Os laços de repetição em Python são usados para executar um bloco de código várias vezes de forma iterativa. Eles são fundamentais para automatizar tarefas repetitivas e são essenciais em muitas aplicações de programação.

Principais Laços de Repetição em Python

Em Python, os laços de repetição mais comuns são:

  • for: Usado para iterar sobre uma sequência (ex.: listas, tuplas, strings) ou um intervalo de números.
  • while: Executa um bloco de código enquanto a condição for verdadeira.

Exemplo de um laço for:

for i in range(5):
    print(i)  # Imprime os números de 0 a 4

Exemplo de um laço while:

count = 0
while count < 5:
    print(count)
    count += 1  # Incrementa count de 1 em 1

Exemplo de uso de break e continue:

for i in range(10):
    if i == 5:
        break  # Interrompe o laço quando i é 5
    if i % 2 == 0:
        continue  # Pula a iteração se i for par
    print(i)  # Imprime apenas números ímpares menores que 5
