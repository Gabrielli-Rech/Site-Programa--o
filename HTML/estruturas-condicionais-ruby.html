O que são Estruturas Condicionais?

Estruturas condicionais são usadas para executar diferentes blocos de código com base em uma condição específica. Em Ruby, isso permite que você controle o fluxo do programa de forma lógica e dinâmica.

Estrutura if

A estrutura if é usada para executar um bloco de código se uma condição for verdadeira.

if condicao
  puts "A condição é verdadeira"
end

Se a condição for falsa, o bloco de código não será executado.

Estrutura if-else

A estrutura if-else permite executar um bloco de código se a condição for verdadeira e outro bloco se for falsa.

if condicao
  puts "A condição é verdadeira"
else
  puts "A condição é falsa"
end

Estrutura elsif

A estrutura elsif é usada quando você precisa verificar múltiplas condições. Se a primeira condição for falsa, o Ruby testa a condição elsif e assim por diante.

if condicao1
  puts "Condição 1 é verdadeira"
elsif condicao2
  puts "Condição 2 é verdadeira"
else
  puts "Nenhuma das condições é verdadeira"
end

Estrutura case

A estrutura case é uma alternativa para usar múltiplos if-elsif quando você precisa comparar um valor com várias opções diferentes.

case variavel
when "valor1"
  puts "A variável é igual a valor1"
when "valor2"
  puts "A variável é igual a valor2"
else
  puts "A variável não é igual a nenhum dos valores"
end

Operador Ternário

O operador ternário é uma forma compacta de escrever uma estrutura if-else. Ele é usado para uma única expressão condicional.

resultado = condicao ? "A condição é verdadeira" : "A condição é falsa"
puts resultado

É uma maneira prática de atribuir um valor a uma variável com base em uma condição.
