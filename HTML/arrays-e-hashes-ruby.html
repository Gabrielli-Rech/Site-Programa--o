Exercícios de Arrays em Ruby

Fáceis

  • Crie um array de números de 1 a 10 e imprima todos os elementos.
  • Adicione um elemento ao final de um array.
  • Remova o último elemento de um array e imprima-o.
  • Verifique se um determinado elemento está presente em um array.
  • Imprima todos os elementos de um array usando um loop `each`.
  • Crie um array de strings e imprima o tamanho de cada string.
  • Filtre os números pares de um array de números inteiros.
  • Crie um array de palavras e imprima cada palavra em maiúsculas.
  • Imprima todos os elementos de um array na ordem inversa.
  • Concatene dois arrays e imprima o resultado.
  • Crie um array de números e calcule a soma de todos os elementos.
  • Crie um array e imprima o primeiro e o último elemento.
  • Verifique o comprimento de um array.
  • Crie um array de cores e imprima a terceira cor.
  • Remova um elemento específico de um array.
  • Crie um array de objetos e imprima suas propriedades.

Médios

  • Crie um array de strings e use `map` para adicionar um sufixo a cada string.
  • Use `select` para filtrar elementos que contenham uma letra específica.
  • Crie um array e use `sort` para ordenar os elementos em ordem crescente.
  • Crie um array de números e calcule a média de todos os elementos.
  • Use `reject` para remover todos os elementos que são menores que 10 de um array.
  • Crie um array de números e use `reduce` para multiplicar todos os elementos.
  • Crie um array e use `uniq` para remover elementos duplicados.
  • Crie um array de palavras e use `each_with_index` para imprimir o índice e a palavra.
  • Crie um array de objetos e use `each` para imprimir a propriedade de cada objeto.
  • Crie um array de números e use `map` para elevar cada número ao quadrado.
  • Implemente uma função que retorna a soma de todos os elementos de um array passado como parâmetro.
  • Crie um array de strings e use `find` para encontrar a primeira string com mais de 5 caracteres.
  • Use `slice` para extrair um subarray de um array original.
  • Crie um array de números e use `min` e `max` para encontrar os elementos mínimo e máximo.
  • Implemente uma função que inverte um array sem usar `reverse`.
  • Crie um array de strings e use `join` para concatenar todas as strings em uma única string.

Difíceis

  • Crie um método que encontre todos os números primos em um array.
  • Implemente uma função que verifique se um array é um palíndromo.
  • Crie um método que receba dois arrays e retorne a interseção entre eles.
  • Crie um array de strings e use `group_by` para agrupar as palavras pelo comprimento.
  • Crie um array de objetos e use `sort_by` para ordená-los por uma de suas propriedades.
  • Implemente uma função que divida um array em subarrays de tamanho específico.
  • Crie um método que receba um array de strings e retorne a string mais longa.
  • Crie um método que encontre a soma dos elementos únicos em um array.
  • Crie um método que remova elementos duplicados em um array mantendo a ordem original.
  • Crie uma função que encontre os números que aparecem mais de uma vez em um array.
  • Implemente uma função que receba dois arrays e retorne um array com os elementos que estão em um e não no outro.
  • Crie um método que encontre o elemento que ocorre mais vezes em um array.
  • Crie um método que inverta os elementos de um array e imprima cada elemento em uma linha.
  • Crie uma função que retorne a soma dos elementos que estão em posições pares de um array.
  • Implemente uma função que encontre os elementos comuns entre três arrays.
  • Crie um método que receba um array e retorne um array com apenas os elementos que são múltiplos de 3.

Desafios

  • Crie um método que encontre todos os anagramas de uma palavra em um array de palavras.
  • Crie uma função que gere todas as combinações possíveis de elementos de um array.
  • Implemente uma função que receba um array e retorne um novo array em que os elementos estejam ordenados pelo número de vezes que aparecem.
  • Crie um método que use `inject` para calcular o produto de todos os elementos de um array.
  • Crie um método que encontre o menor número ímpar em um array de números.
  • Implemente uma função que transforme um array de objetos em um hash.
  • Crie um método que gere a sequência de Fibonacci em um array de tamanho especificado.
  • Crie um método que remova os elementos do meio de um array e retorne-os em um novo array.
  • Implemente uma função que encontre o maior produto de dois elementos em um array de números.
  • Crie um método que verifique se um array contém um número específico de formas diferentes.
  • Crie uma função que use `map` para criar um array de strings formatadas de acordo com uma condição.
  • Implemente um método que divida um array em dois subarrays de tamanhos iguais, se possível.
  • Crie uma função que encontre o elemento do array que aparece na maior quantidade de subarrays.
  • Crie uma função que encontre todos os pares de elementos cujas somas são iguais a um valor especificado.
  • Implemente uma função que use `each_with_index` para imprimir os elementos de um array e seus índices em ordem reversa.
  • Crie uma função que remova elementos duplicados de um array sem usar `uniq`.

Exercícios de Hashes em Ruby

Fáceis

  • Crie um hash de nomes e idades e imprima todos os valores.
  • Adicione um novo par chave-valor a um hash existente.
  • Remova um par chave-valor de um hash.
  • Verifique se uma chave existe em um hash.
  • Imprima todas as chaves de um hash.
  • Imprima todos os valores de um hash.
  • Acesse um valor específico em um hash.
  • Crie um hash de contatos e imprima os nomes de todos os contatos.
  • Crie um hash e use `each` para imprimir chave e valor juntos.
  • Crie um hash e imprima a soma dos valores numéricos.
  • Implemente uma função que inverta as chaves e valores de um hash.
  • Verifique se um valor específico existe em um hash.
  • Crie um hash e adicione um novo elemento com `merge`.
  • Imprima um hash formatado com `to_s`.
  • Implemente uma função que retorne o valor de uma chave específica ou uma mensagem padrão se a chave não existir.
  • Crie um hash e use `select` para filtrar pares chave-valor específicos.

Médios

  • Crie um hash e use `keys` para imprimir apenas as chaves.
  • Use `values` para imprimir apenas os valores de um hash.
  • Crie um método que receba um hash e retorne a chave com o maior valor.
  • Crie um método que receba um hash e retorne a chave com o menor valor.
  • Implemente uma função que receba um hash e retorne os pares chave-valor em ordem alfabética das chaves.
  • Crie um método que use `each_with_index` para imprimir chave, valor e índice em um hash.
  • Implemente uma função que calcule o número de elementos em um hash.
  • Crie um método que use `map` para transformar os valores de um hash em um novo hash.
  • Implemente uma função que retorne um novo hash com todas as chaves em maiúsculas.
  • Crie uma função que filtre um hash removendo pares com valores nulos.
  • Use `delete_if` para remover pares chave-valor de um hash que atendem a uma condição específica.
  • Implemente uma função que adicione um par chave-valor a um hash se a chave não existir.
  • Crie um hash de estudantes e use `each` para imprimir o nome e a nota de cada um.
  • Crie um método que retorne um novo hash com as chaves invertidas.
  • Implemente uma função que retorne um hash com apenas as chaves que têm valores numéricos.
  • Crie uma função que encontre todas as chaves que possuem valores que são arrays.

Difíceis

  • Crie uma função que agrupe um hash por um valor específico.
  • Crie um método que conte quantas vezes cada valor aparece em um hash.
  • Implemente uma função que calcule a média de todos os valores numéricos em um hash.
  • Crie um método que combine dois hashes, somando os valores de chaves iguais.
  • Crie uma função que modifique um hash de forma que os valores sejam transformados em arrays.
  • Implemente uma função que filtre um hash e retorne apenas as chaves com valores de tipo string.
  • Crie um método que encontre a chave de um hash que possui um valor específico.
  • Implemente uma função que divida um hash em dois, baseado em um critério específico.
  • Crie uma função que retorne um hash com chaves e valores trocados e ordenados pela chave.
  • Implemente uma função que verifique se todos os valores em um hash são únicos.
  • Crie um método que imprima todas as combinações possíveis de chaves e valores em um hash.
  • Implemente uma função que agrupe um hash por um determinado critério de valor.
  • Crie uma função que retorne o hash interseção entre dois hashes.
  • Crie um método que calcule a soma dos valores em um hash e retorne o valor da chave com a maior soma.
  • Implemente uma função que modifique um hash para que todas as chaves sejam numéricas.
  • Crie uma função que ordene um hash por valores de forma decrescente.

Desafios

  • Crie uma função que agrupe um array de hashes por um campo específico.
  • Implemente uma função que retorne o hash de um array de pares chave-valor.
  • Crie um método que encontre a chave com o maior valor num hash aninhado.
  • Implemente uma função que retorne um novo hash onde cada chave seja uma palavra de uma frase.
  • Crie uma função que percorra um hash aninhado e imprima todas as chaves e valores.
  • Crie um método que receba um hash e retorne apenas as chaves que estão em um array de interesse.
  • Implemente uma função que mescle dois hashes e preserve as chaves únicas.
  • Crie um método que percorra um hash e modifique os valores para seus quadrados.
  • Implemente uma função que calcule a diferença entre dois hashes em termos de chaves e valores.
  • Crie uma função que retorne um hash filtrado por um valor mínimo.
  • Crie um método que crie um hash de contagem de elementos de um array de strings.
  • Implemente uma função que transforme um hash em um array de arrays de pares chave-valor.
  • Crie uma função que verifique se um hash possui chaves duplicadas.
  • Implemente uma função que atualize um hash de acordo com uma condição específica.
  • Crie uma função que inverta um hash mantendo a chave como valor e o valor como chave.
  • Crie uma função que receba um hash de hashes e retorne o hash que possui o maior valor em uma chave específica.
