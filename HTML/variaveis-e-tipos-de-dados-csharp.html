O que são Variáveis e Tipos de Dados?

Em C#, variáveis são usadas para armazenar valores que podem ser manipulados em um programa. C# é uma linguagem de tipagem estática e forte, o que significa que o tipo de dado de uma variável deve ser declarado de forma explícita e não pode ser alterado.

Declaração de Variáveis

Para declarar uma variável em C#, você especifica o tipo de dado seguido pelo nome da variável. A variável pode ser inicializada ao mesmo tempo da declaração ou em uma etapa separada:

string nome = "Ana";  // string
int idade = 28;  // inteiro
double altura = 1.65;  // double
bool ativo = true;  // booleano

O tipo da variável deve ser conhecido em tempo de compilação e não pode ser alterado.

Tipos de Dados em C#

Os tipos de dados em C# podem ser classificados como primitivos e complexos:

  • Primitivos:
    • int: Números inteiros de 32 bits. Exemplo: 42, -10
    • long: Números inteiros de 64 bits. Exemplo: 1234567890
    • short: Números inteiros de 16 bits. Exemplo: 100
    • byte: Números inteiros de 8 bits, variando de 0 a 255. Exemplo: 255
    • double: Números de ponto flutuante de precisão dupla de 64 bits. Exemplo: 3.14
    • float: Números de ponto flutuante de precisão simples de 32 bits. Exemplo: 1.5
    • decimal: Números de ponto flutuante de 128 bits, usados para cálculos financeiros. Exemplo: 10.5m
    • char: Um único caractere Unicode de 16 bits. Exemplo: 'A'
    • bool: Valores booleanos. Exemplo: true, false
    • string: Sequência de caracteres (texto). Exemplo: "Olá, mundo!"
    • object: Tipo base de todos os tipos em C#. Pode armazenar qualquer tipo de dado. Exemplo: object obj = "texto";
  • Tipos Complexos: Tipos que podem conter múltiplos valores e oferecer mais funcionalidades, como classes, arrays, structs e enums.

Exemplo de diferentes tipos de dados:

string nome = "Carlos";  // string
int idade = 30;  // int
double altura = 1.78;  // double
bool ativo = false;  // booleano
object endereco = null;  // object
