O que é o MongoDB?

O MongoDB é um banco de dados NoSQL orientado a documentos. Ele armazena dados em formato JSON-like, usando documentos BSON (Binary JSON), o que permite maior flexibilidade e escalabilidade em comparação aos bancos de dados relacionais tradicionais.

Inserindo Documentos em uma Coleção

Para inserir documentos em uma coleção do MongoDB, você pode usar os métodos insertOne() ou insertMany().

Exemplo de inserção de um único documento:

db.colecao.insertOne({
    nome: "João",
    idade: 30,
    cidade: "São Paulo"
});

Exemplo de inserção de múltiplos documentos:

db.colecao.insertMany([
    { nome: "Maria", idade: 25, cidade: "Rio de Janeiro" },
    { nome: "Carlos", idade: 28, cidade: "Belo Horizonte" }
]);

Manipulando Documentos

Após inserir documentos, você pode realizar operações como atualizar, deletar e consultar os documentos.

Atualizando Documentos

Para atualizar um documento, use o método updateOne() ou updateMany().

db.colecao.updateOne(
    { nome: "João" },
    { $set: { idade: 31 } }
);

Explicação: O operador $set é usado para definir novos valores para os campos existentes.

Deletando Documentos

Para deletar documentos, use o método deleteOne() ou deleteMany().

db.colecao.deleteOne({ nome: "Carlos" });

Explicação: O método deleteOne() remove o primeiro documento que corresponde ao filtro, enquanto deleteMany() remove todos os documentos que atendem ao filtro.

Consultando Documentos

Para consultar documentos, use o método find().

db.colecao.find({ idade: { $gte: 25 } });

Explicação: O operador $gte significa "maior ou igual a".
