O que é Git?

Git é um sistema de controle de versão distribuído de código aberto que permite rastrear mudanças em arquivos e colaborar em projetos de software. Criado por Linus Torvalds, é amplamente utilizado no desenvolvimento de software moderno.

Comandos Básicos do Git

Comandos essenciais para começar a usar o Git:

  • git init – Inicializa um novo repositório Git.
  • git clone [url] – Clona um repositório remoto para a sua máquina.
  • git add . – Adiciona todos os arquivos modificados ao estágio de commit.
  • git commit -m "mensagem" – Cria um novo commit com a mensagem especificada.
  • git push origin main – Envia suas mudanças para o repositório remoto.
