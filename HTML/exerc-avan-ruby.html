Exercícios Avançados de Ruby

  • Voltar
  • Início
  • 1. Manipulação de Arrays: Crie um método que receba um array e retorne um novo array contendo todos os elementos que são comuns entre dois arrays diferentes.
  • 2. Métodos Recursivos: Escreva um método recursivo para calcular o fatorial de um número.
  • 3. Filtros e Map: Dado um array de números, use métodos select e map para retornar apenas os números pares, multiplicados por 2.
  • 4. Filtros e Reduções: Escreva um método que, dado um array de números, retorne a soma dos números que são divisíveis por 3.
  • 5. Classes e Objetos: Crie uma classe Pessoa com atributos nome, idade e email, e métodos para exibir a idade da pessoa em dias e verificar se a pessoa é maior de idade.
  • 6. Herança: Crie uma classe Animal com métodos som e mover. Em seguida, crie classes derivadas como Cachorro e Gato que sobrescrevem o método som.
  • 7. Módulos: Crie um módulo Calculadora com métodos soma, subtracao, multiplicacao, e divisao. Inclua esse módulo em uma classe Operacoes.
  • 8. Iteradores: Escreva um método que percorra um array e imprima todos os elementos que são maiores que a média do array.
  • 9. Procs e Lambdas: Crie uma função que receba um proc ou lambda como argumento e a aplique a um array de números.
  • 10. Blocks: Escreva um método que aceite um bloco e use o método each para aplicar esse bloco a todos os elementos de um array.
  • 11. Exceções: Crie um método que leia um arquivo e trate possíveis exceções caso o arquivo não exista ou não possa ser lido.
  • 12. Arquivos: Crie um método que escreva um array de strings em um arquivo e depois leia o arquivo para exibir seu conteúdo.
  • 13. Expressões Regulares: Escreva um método que use expressões regulares para validar se uma string é um endereço de e-mail válido.
  • 14. Classes e Métodos Estáticos: Crie uma classe Matematica com métodos de classe estáticos para calcular a soma e a média de dois números.
  • 15. Enumerable: Crie um método que use Enumerable#all? para verificar se todos os elementos de um array são positivos.
  • 16. Enumeração com each_with_index: Escreva um método que percorra um array e imprima o índice e o valor de cada elemento.
  • 17. Date e Time: Crie um método que calcule a diferença em dias entre duas datas.
  • 18. Métodos Encadeados: Crie uma cadeia de métodos em uma classe Pessoa para calcular o índice de massa corporal (IMC) e exibir se a pessoa está com sobrepeso, peso normal ou abaixo do peso.
  • 19. Herança Múltipla: Crie uma classe Mammal e uma classe Swimming e depois uma classe Dolphin que herde de ambas.
  • 20. Constantes: Crie uma classe que tenha uma constante PI e um método que calcule a área de um círculo dado seu raio.
  • 21. Destructuring: Escreva um método que receba um array e use destruição para atribuir os valores a variáveis individuais.
  • 22. Memorização: Crie um método que use a técnica de memorização para armazenar resultados de cálculos e melhorar a eficiência de chamadas subsequentes.
  • 23. Caminhos de Arquivo: Escreva um método que use File.join para criar um caminho de arquivo que funcione em qualquer sistema operacional.
  • 24. Comandos de Linha de Comando: Crie um método que receba um comando de shell como argumento e o execute usando system.
  • 25. Testes Unitários: Escreva testes para uma função que verifica se um número é primo.
  • 26. Métodos Privados: Crie uma classe que tenha métodos privados que só possam ser acessados de dentro da classe.
  • 27. Classe Anônima: Crie uma classe anônima que herde de uma classe base e defina um método sobrescrito.
  • 28. Simulação de Banco de Dados: Crie uma classe que simule um banco de dados simples com operações CRUD.
  • 29. Métodos de Instância: Escreva um método que altere o valor de um atributo em uma instância de uma classe.
  • 30. Reflexão: Use a reflexão para criar um método que liste os métodos e atributos de uma classe.
  • 31. Regex Avançado: Escreva um método que extraia todas as URLs de uma string usando expressões regulares.
  • 32. Manejo de Processos: Crie um script que inicie um processo externo, capture sua saída e a imprima.
  • 33. Processamento Paralelo: Escreva um método que use threads para calcular a soma de vários arrays simultaneamente.
  • 34. Evolução de Arrays: Crie um método que, dado um array de palavras, retorne uma hash onde as chaves sejam o tamanho das palavras e os valores sejam arrays de palavras desse tamanho.
  • 35. Construção de Hashes: Crie um método que aceite uma lista de pares chave-valor e construa um hash.
  • 36. Personificação de Dados: Crie uma classe Produto e use to_s para retornar uma representação em string de um objeto da classe.
  • 37. Busca Binária: Implemente o algoritmo de busca binária em um array ordenado.
  • 38. Geradores de Senhas: Crie um método que gere senhas aleatórias com caracteres, números e símbolos.
  • 39. Conversão de Tipos: Escreva um método que converta uma string de data em um objeto Date e vice-versa.
  • 40. Mapeamento de Objetos: Crie um método que receba um array de objetos e retorne um array com um atributo específico de cada objeto.
  • 41. Desafio de Desempenho: Escreva um método que calcule o tempo de execução de outro método usando Time.now.
  • 42. Desestruturação de Hashes: Escreva um método que use a desestruturação de hashes para extrair dados específicos.
  • 43. Validação de Senha: Crie um método que valide uma senha para garantir que ela tenha pelo menos uma letra maiúscula, uma minúscula, um número e um caractere especial.
  • 44. Manipulação de Strings: Escreva um método que converta um texto em snake_case a partir de CamelCase.
  • 45. Scraping de Web: Escreva um script simples que use open-uri e nokogiri para fazer scraping em uma página da web.
  • 46. Desafio de Recursão: Implemente uma função recursiva para inverter uma string.
  • 47. Composição de Métodos: Crie um método que combine dois métodos em uma nova função usando Proc.
  • 48. Jogo de Adivinhação: Crie um método que implemente um jogo simples de adivinhação de números com feedback.
  • 49. Depuração de Código: Escreva um método que use binding.pry para inspecionar variáveis durante a execução.
  • 50. Simulação de Eventos: Crie uma classe que simule um evento de clique de botão e acione um bloco de código ao ser "clicado".
