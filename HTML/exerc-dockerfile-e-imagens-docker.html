Exercícios Fáceis

  • Crie um Dockerfile simples que use uma imagem base do Ubuntu e instale o `curl`.
  • Crie um Dockerfile que use a imagem base `node:14` e copie um arquivo `app.js` para o contêiner.
  • Crie uma imagem Docker que execute um servidor HTTP usando o `python3 -m http.server`.
  • Escreva um Dockerfile para uma aplicação Node.js com um script `start` que inicie o servidor.
  • Use a imagem base `nginx` e adicione um arquivo `index.html` customizado.
  • Crie um Dockerfile para uma aplicação Python que use `flask` para exibir "Hello, World!".
  • Crie uma imagem Docker com uma versão específica do Python, como `python:3.8`.
  • Use a imagem `alpine` e instale o `vim` no contêiner.
  • Crie um Dockerfile que faça a instalação de um pacote com `apt-get`.
  • Escreva um Dockerfile que copie um arquivo de configuração `nginx.conf` para um contêiner.
  • Utilize `docker build` para construir uma imagem a partir de um Dockerfile.
  • Crie uma imagem e execute um comando `echo` simples dentro dela.
  • Escreva um Dockerfile que defina uma variável de ambiente e imprima-a.
  • Crie uma imagem com um contêiner que utilize a versão mais recente do `ubuntu`.
  • Crie uma imagem Docker que execute um script bash (`setup.sh`) no momento da inicialização.
  • Escreva um Dockerfile que instale o `git` e clone um repositório no contêiner.
  • Crie uma imagem Docker com `openjdk:11` e execute um programa Java simples.
  • Escreva um Dockerfile que exponha uma porta específica do contêiner.
  • Crie uma imagem Docker com o `node:16` e instale uma dependência (`express`).
  • Use `docker run` para iniciar um contêiner de uma imagem customizada.
