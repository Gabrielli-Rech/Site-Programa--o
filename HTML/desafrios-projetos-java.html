Desafios e Projetos Práticos em Java

Desafios

  • 1. Crie um programa que receba uma lista de números e retorne a soma de todos os números ímpares.
  • 2. Desenvolva uma aplicação que implemente um algoritmo de busca binária em um array de inteiros.
  • 3. Escreva um programa que verifique se uma string é um palíndromo.
  • 4. Crie uma função que receba um array de strings e retorne a string mais longa.
  • 5. Implemente um programa que conte a quantidade de palavras em um arquivo de texto.
  • 6. Escreva um programa que simule a execução de um jogo de pedra, papel e tesoura entre dois jogadores.
  • 7. Desenvolva uma aplicação que gere a sequência de Fibonacci até um número N.
  • 8. Crie um programa que use a classe `HashMap` para contar a frequência de palavras em uma frase.
  • 9. Implemente um algoritmo de ordenação simples, como bubble sort ou selection sort.
  • 10. Escreva um programa que verifique se um número é primo.
  • 11. Crie uma função que calcule o fatorial de um número usando recursão.
  • 12. Implemente um programa que gere um número aleatório entre 1 e 100 e permita ao usuário adivinhar o número.
  • 13. Crie uma aplicação que leia um arquivo CSV e exiba seus dados em formato de tabela.
  • 14. Desenvolva uma função que use `Arrays` para encontrar o segundo maior elemento em um array.
  • 15. Escreva um programa que verifique se duas strings são anagramas.
  • 16. Implemente um programa que leia uma lista de números e retorne a soma dos elementos duplicados.
  • 17. Crie uma função que inverta uma lista de strings.
  • 18. Desenvolva um programa que converta um número decimal para binário.
  • 19. Escreva um programa que implemente a busca linear em um array.
  • 20. Crie uma classe `Pessoa` com atributos como nome, idade e endereço, e métodos para exibir as informações.

Projetos Práticos

  • 1. Desenvolva um sistema de cadastro de clientes com um banco de dados em Java e JDBC.
  • 2. Crie uma aplicação de controle de estoque que permita adicionar, remover e consultar produtos.
  • 3. Implemente um jogo de xadrez com uma interface gráfica simples usando JavaFX.
  • 4. Desenvolva um sistema de gerenciamento de tarefas com persistência em arquivo.
  • 5. Crie um chatbot simples que utilize processamento de linguagem natural (NLP) para responder perguntas básicas.
  • 6. Implemente uma aplicação de chat em tempo real usando sockets em Java.
  • 7. Crie um programa que simule o funcionamento de um banco, com operações de saque, depósito e transferência.
  • 8. Desenvolva um sistema de reservas de hotel que permita consultar e reservar quartos.
  • 9. Escreva uma aplicação para calcular o índice de massa corporal (IMC) e recomendar um plano de saúde.
  • 10. Implemente uma aplicação de conversor de moedas usando uma API de câmbio em tempo real.
  • 11. Crie um sistema de recomendação de filmes baseado nas preferências do usuário.
  • 12. Desenvolva uma calculadora científica em Java que suporte operações avançadas, como trigonometria e logaritmos.
  • 13. Implemente um sistema de rastreamento de despesas pessoais com relatórios mensais.
  • 14. Crie um jogo de trivia com perguntas e respostas em Java.
  • 15. Desenvolva uma aplicação de geração de senhas seguras com diferentes critérios de complexidade.
  • 16. Implemente uma aplicação de gerenciamento de biblioteca, com funcionalidades de cadastro, busca e remoção de livros.
  • 17. Crie um sistema de armazenamento de notas de alunos e cálculo de médias em Java.
  • 18. Desenvolva uma aplicação de previsão do tempo usando uma API de dados meteorológicos.
  • 19. Implemente um sistema de cadastro e consulta de contatos com uma interface de usuário em Swing.
  • 20. Crie uma aplicação de leitura e escrita de arquivos XML em Java usando `JAXB`.
