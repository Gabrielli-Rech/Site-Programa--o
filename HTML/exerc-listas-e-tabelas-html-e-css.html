Exercícios de Listas

  • Crie uma lista ordenada (ol) de 5 elementos representando uma lista de tarefas.
  • Crie uma lista não ordenada (ul) de frutas com pelo menos 4 itens.
  • Adicione uma lista aninhada dentro de uma lista não ordenada, onde o sub-item representa diferentes tipos de um item.
  • Crie uma lista de links (ul) que leva a diferentes páginas de exemplo.
  • Crie uma lista de compras com 5 itens, usando uma lista não ordenada e marcadores.

Exercícios de Tabelas

  • Crie uma tabela com cabeçalhos que liste as 5 principais cidades do mundo por população, com colunas para nome, país e população.
  • Implemente uma tabela com 3 linhas e 3 colunas, onde uma das células contém uma imagem.
  • Crie uma tabela de horários de aulas com colunas para horário, disciplina e professor.
  • Adicione uma tabela com uma linha de cabeçalho e uma linha de rodapé, com 4 colunas e 2 linhas de dados.
  • Crie uma tabela com uma célula mesclada (colspan) para representar um título de coluna que abranja duas colunas.
  • Crie uma tabela de preços de produtos com 4 colunas: produto, preço unitário, quantidade e preço total.
  • Implemente uma tabela com uma linha de cabeçalho, onde uma das células tem um fundo colorido.
  • Crie uma tabela com 3 linhas e 2 colunas, onde as células da primeira coluna têm um destaque em negrito.
  • Adicione uma tabela com uma coluna de datas e uma coluna de eventos.
  • Crie uma tabela de contatos com colunas para nome, telefone, e-mail e endereço.
