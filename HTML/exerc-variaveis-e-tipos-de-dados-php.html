Exercícios Práticos

  • Crie uma variável chamada $nome e atribua seu nome a ela. Imprima a variável na tela.
  • Crie uma variável $idade e atribua a sua idade. Imprima a idade com uma mensagem, por exemplo: "Eu tenho X anos".
  • Crie uma variável $altura e atribua um valor de ponto flutuante. Imprima a altura com uma mensagem formatada, como "Minha altura é X metros".
  • Crie uma variável $isEstudante do tipo booleano e atribua um valor true ou false. Imprima a variável com uma mensagem como "Sou estudante?"
  • Crie um array $frutas com pelo menos três elementos (por exemplo: "maçã", "banana", "laranja"). Imprima todos os elementos usando um loop.
  • Crie uma variável $pessoa como um array associativo com chaves "nome" e "idade". Imprima a chave "nome".
  • Crie uma variável $preco e atribua um valor monetário a ela. Imprima o valor formatado como moeda (por exemplo, "R$ 100,00").
  • Verifique e imprima se a variável $idade é do tipo inteiro usando a função is_int().
  • Crie uma variável $mensagem do tipo string e use a função strlen() para imprimir o comprimento da string.
  • Crie duas variáveis numéricas, $num1 e $num2, e imprima a soma, subtração, multiplicação e divisão de ambas.
  • Crie uma variável $texto e use a função strtoupper() para imprimir o texto em maiúsculas.
  • Crie uma variável $nomeCompleto usando a concatenação de duas strings. Imprima o resultado com uma mensagem.
  • Crie uma variável $dataAtual usando a função date() para exibir a data atual em formato "dd/mm/aaaa".
  • Crie uma variável $numero e use a função abs() para imprimir o valor absoluto do número.
  • Crie uma variável $tempCelsius e converta para Fahrenheit usando a fórmula: (C × 9/5) + 32.
  • Crie uma variável $frase e use a função str_replace() para substituir uma palavra específica por outra.
  • Crie uma variável $senha do tipo string e use a função md5() para imprimir o hash MD5 da senha.
  • Crie uma variável $var de tipo misto e imprima o tipo de dado usando a função gettype().
  • Crie uma variável $numeroGrande do tipo inteiro e imprima o número formatado com a função number_format().
  • Crie uma variável $nome e use a função ucwords() para capitalizar a primeira letra de cada palavra.
  • Crie uma variável $email e use a função filter_var() para validar se é um email válido.
  • Crie uma variável $mensagem e use a função substr() para imprimir os primeiros 5 caracteres da string.
  • Crie uma variável $quantidade e use a função round() para arredondar o valor para o inteiro mais próximo.
  • Crie uma variável $pessoa e use a função is_array() para verificar se a variável é um array.
  • Crie uma variável $mensagem e use a função strrev() para imprimir a string ao contrário.
  • Crie uma variável $valor e use a função is_numeric() para verificar se é um número.
  • Crie uma variável $vazio e use a função empty() para verificar se a variável está vazia.
  • Crie uma variável $divisao e use a função intdiv() para fazer a divisão inteira de dois números.
