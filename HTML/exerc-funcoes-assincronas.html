O que são Funções Assíncronas?

Funções assíncronas em JavaScript são usadas para lidar com operações que demoram a serem executadas, como chamadas a APIs ou temporizadores. Elas permitem que o código continue sua execução enquanto aguardam a resposta de uma operação.

Exercícios de Funções Assíncronas

  • Crie uma função assíncrona que use `fetch` para fazer uma requisição a um endpoint e retorne o JSON da resposta.
  • Crie uma função que use `setTimeout` para simular uma tarefa assíncrona e imprima uma mensagem após 3 segundos.

  • Crie uma função assíncrona que simule um erro de conexão e utilize um `catch` para tratá-lo.
  • Crie uma função assíncrona que retorne o resultado de uma operação de soma após 1 segundo.
