Exercícios Fáceis

  • Crie uma tabela chamada `usuarios` com as colunas `id` (chave primária, serial), `nome` (texto) e `email` (texto).
  • Adicione uma coluna `idade` (inteiro) à tabela `usuarios`.
  • Renomeie a coluna `nome` para `nome_completo` na tabela `usuarios`.
  • Remova a coluna `idade` da tabela `usuarios`.
  • Crie uma tabela chamada `produtos` com as colunas `id`, `nome` e `preco`.
  • Adicione uma restrição para que a coluna `email` da tabela `usuarios` seja única.
  • Insira 3 registros fictícios na tabela `usuarios`.
  • Exclua todos os registros da tabela `usuarios` sem apagar a tabela.
  • Crie uma tabela chamada `pedidos` com as colunas `id`, `usuario_id` e `total`.
  • Exclua a tabela `pedidos`.

Exercícios Médios

  • Crie uma tabela chamada `enderecos` com as colunas `id`, `usuario_id`, `rua`, `cidade` e `estado`. Configure `usuario_id` como chave estrangeira para a tabela `usuarios`.
  • Adicione uma restrição para que a coluna `preco` da tabela `produtos` seja maior que 0.
  • Altere o tipo da coluna `total` da tabela `pedidos` para `numeric` com 2 casas decimais.
  • Crie um índice na coluna `email` da tabela `usuarios` para melhorar a performance das consultas.
  • Escreva um comando para listar todas as tabelas do banco de dados atual.
  • Crie uma tabela chamada `categorias` com as colunas `id` e `nome`. Adicione uma chave estrangeira na tabela `produtos` referenciando `categorias`.
  • Crie uma tabela com uma coluna `data_criacao` que armazene a data e hora do registro automaticamente.
  • Adicione uma restrição para que a coluna `nome` da tabela `produtos` não aceite valores nulos.
  • Crie uma tabela chamada `avaliacoes` com as colunas `id`, `produto_id`, `usuario_id` e `nota`. Configure `produto_id` e `usuario_id` como chaves estrangeiras.
  • Exclua todas as chaves estrangeiras da tabela `enderecos`.

Exercícios Difíceis

  • Crie uma tabela chamada `estoques` com as colunas `id`, `produto_id` e `quantidade`. Configure `produto_id` como chave estrangeira.
  • Adicione uma restrição que limite a `quantidade` da tabela `estoques` a valores entre 0 e 1000.
  • Crie uma tabela chamada `transacoes` com as colunas `id`, `tipo` e `valor`, onde `tipo` só pode ser "compra" ou "venda".
  • Adicione uma restrição de chave composta para as colunas `produto_id` e `usuario_id` na tabela `avaliacoes`.
  • Implemente um comando que exclua todos os registros de uma tabela, mas mantenha o próximo valor da chave primária intacto.
  • Escreva um comando para clonar a estrutura de uma tabela (sem copiar os dados).
  • Crie uma tabela com uma coluna que armazene JSON, representando os atributos de um produto.
  • Escreva um comando para listar todas as colunas de uma tabela e seus tipos de dados.
  • Implemente um comando para mover uma tabela para outro schema no mesmo banco de dados.
  • Crie uma tabela chamada `logs` que registre a data e hora de cada inserção de dados automaticamente.

Desafios

  • Crie um sistema de tabelas que represente um sistema de biblioteca com `livros`, `autores`, `emprestimos` e `usuarios`.
  • Crie um sistema de tabelas que represente um sistema de vendas, incluindo `clientes`, `pedidos`, `produtos` e `itens_pedido`.
  • Implemente uma tabela de histórico que registre alterações feitas em outra tabela, incluindo a data da alteração.
  • Crie uma tabela que utilize partições para organizar dados por ano.
  • Desenhe um modelo de tabelas que represente uma rede social com usuários, postagens, comentários e curtidas.
  • Implemente uma tabela de auditoria que registre todas as operações (inserções, atualizações, exclusões) feitas em uma tabela específica.
  • Escreva um comando que compare a estrutura de duas tabelas e liste as diferenças.
  • Crie uma tabela que utilize herança para armazenar informações genéricas e específicas de diferentes tipos de veículos.
  • Projete um sistema de tabelas para uma aplicação de gerenciamento escolar com `alunos`, `professores`, `cursos` e `notas`.
  • Implemente uma tabela com colunas que utilizem o tipo de dados `ARRAY` para armazenar listas de valores.
