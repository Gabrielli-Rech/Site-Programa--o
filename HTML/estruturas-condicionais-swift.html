O que são Estruturas Condicionais?

Estruturas condicionais são usadas para tomar decisões em Swift com base em condições. Elas permitem que o código seja executado dependendo de valores ou condições específicas.

Estrutura if

A estrutura if executa um bloco de código se a condição especificada for verdadeira.

let idade = 18
if idade >= 18 {
    print("Você é maior de idade.")
}

Neste exemplo, a mensagem será exibida se a idade for maior ou igual a 18.

Estrutura if-else

Use a estrutura if-else para executar um bloco de código se a condição for verdadeira e outro bloco se ela for falsa.

let temperatura = 30
if temperatura > 25 {
    print("Está quente lá fora!")
} else {
    print("O tempo está agradável.")
}

Estrutura if-else if

Quando há múltiplas condições, use a estrutura if-else if para verificar cada uma delas em sequência.

let nota = 85
if nota >= 90 {
    print("Excelente!")
} else if nota >= 70 {
    print("Bom trabalho.")
} else {
    print("Você precisa melhorar.")
}

Estrutura switch

O switch é uma alternativa ao if para verificar múltiplos casos. Ele é mais limpo e eficiente quando há muitas condições.

let diaDaSemana = 3
switch diaDaSemana {
case 1:
    print("Domingo")
case 2:
    print("Segunda-feira")
case 3:
    print("Terça-feira")
default:
    print("Outro dia")
}

Neste exemplo, o código imprime o dia correspondente ao número fornecido.

Quando usar if ou switch?

Use if para condições simples ou verificações booleanas. Use switch para comparar um valor com múltiplos casos.
