Exercícios de Laços de Repetição

  • Crie um loop `for` que imprime os números de 1 a 10 no console.
  • Crie um loop `while` que imprime os números de 10 a 1 no console.
  • Escreva um loop `do-while` que continue imprimindo a palavra "PHP" enquanto uma variável for menor que 5.
  • Use um loop `for` para somar todos os números de 1 a 100 e imprima o resultado.
  • Crie um loop `for` que imprime apenas os números pares de 1 a 20.
  • Escreva um loop `while` que imprime a tabuada do 2 (de 2 x 1 a 2 x 10).
  • Crie um loop `do-while` que continue pedindo um número ao usuário até que ele insira um número positivo.
  • Use um loop `for` para imprimir uma lista de strings em ordem reversa.
  • Escreva um loop que conte de 0 a 50, pulando de 5 em 5.
  • Crie um loop `for` que imprima os elementos de um array de strings.
  • Implemente um loop `while` que conta quantas vezes um número específico aparece em um array.
  • Crie um loop `do-while` que peça uma senha até que o usuário insira a senha correta.
  • Use um loop `for` para contar quantas vezes a letra 'a' aparece em uma string dada.
  • Escreva um loop `while` que imprime os números de 100 a 1 em ordem decrescente e para quando atingir 50.
  • Crie um loop que imprime uma pirâmide de asteriscos de altura 5.
  • Implemente um loop `for` que percorre um array associativo e imprime suas chaves e valores.
  • Crie um loop `while` que soma os números de um array até alcançar um valor de soma maior que 100.
  • Use um loop `do-while` para gerar uma senha aleatória de 8 caracteres e imprima no console.
  • Escreva um loop `for` que imprime a sequência de Fibonacci até o décimo termo.
  • Crie um loop `while` que simula um contador regressivo de 10 a 1 com um intervalo de 1 segundo entre cada número.
