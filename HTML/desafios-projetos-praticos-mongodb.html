Desafios Fáceis

  • 1. Conectar a um banco de dados MongoDB local e listar os bancos de dados existentes.
  • 2. Criar uma coleção e inserir um documento simples.
  • 3. Consultar todos os documentos de uma coleção usando find().
  • 4. Atualizar um documento específico em uma coleção usando updateOne().
  • 5. Deletar um documento de uma coleção usando deleteOne().
  • 6. Criar um índice em um campo específico de uma coleção.
  • 7. Contar o número de documentos em uma coleção usando countDocuments().
  • 8. Inserir múltiplos documentos em uma coleção com insertMany().
  • 9. Realizar uma consulta com filtro específico em uma coleção.
  • 10. Obter todos os campos de um documento específico sem a necessidade de filtros.

Desafios Médios

  • 11. Realizar consultas complexas com operadores de comparação como $gt e $lt.
  • 12. Utilizar aggregate() para realizar uma agregação simples com $match e $group.
  • 13. Adicionar um campo calculado em um documento existente usando updateOne() e $set.
  • 14. Criar uma coleção com um esquema de validação usando Mongoose.
  • 15. Realizar consultas de projeção para retornar apenas campos específicos.
  • 16. Utilizar findOneAndUpdate() para realizar uma consulta de atualização com retorno.
  • 17. Criar um sistema de log que armazena as atividades de um usuário em uma coleção separada.
  • 18. Criar uma consulta que utilize $or para combinar múltiplas condições de filtro.
  • 19. Utilizar populate() com Mongoose para fazer uma junção de coleções.
  • 20. Criar uma página de relatórios usando aggregate() para calcular a média de valores de um campo.

Desafios Difíceis

  • 21. Criar um índice composto em uma coleção e explicar como ele melhora a performance.
  • 22. Implementar uma consulta com aggregation pipeline para filtrar, agrupar e ordenar documentos.
  • 23. Implementar um sistema de ranking de usuários com contagem e ordenação de documentos em uma coleção.
  • 24. Utilizar a operação bulkWrite() para inserir, atualizar e deletar múltiplos documentos em uma única operação.
  • 25. Criar uma função que utilize aggregate() para analisar e agrupar dados de vendas.
  • 26. Criar uma consulta para encontrar os documentos mais recentes em uma coleção usando $sort e $limit.
  • 27. Realizar consultas com operadores lógicos como $and e $nor.
  • 28. Implementar uma atualização em massa com a função updateMany() e filtro complexo.
  • 29. Criar uma consulta com projeção e uso de $slice para retornar apenas uma parte de um array.
  • 30. Criar uma solução para armazenar e consultar dados de logs em tempo real.

Desafios Avançados

  • 31. Criar um banco de dados de aplicações em tempo real com change streams para rastrear alterações em uma coleção.
  • 32. Criar um sistema de recomendação de produtos com base em compras anteriores usando aggregate().
  • 33. Implementar a replicação de dados entre múltiplos bancos de dados MongoDB.
  • 34. Realizar uma análise de desempenho de consultas usando o explain().
  • 35. Criar uma consulta para agrupar dados e calcular percentuais com a operação $project.
  • 36. Configurar e usar um shard cluster em MongoDB para escalabilidade horizontal.
  • 37. Implementar um sistema de autenticação de usuários com dados armazenados em MongoDB.
  • 38. Configurar uma integração com uma API para importar e exportar dados em formato JSON.
  • 39. Criar um sistema de backups automáticos usando mongodump e mongorestore.
  • 40. Implementar uma consulta que analisa e retorna documentos com dados temporais, usando $dateToString.
