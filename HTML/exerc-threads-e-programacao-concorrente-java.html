Exercícios de Threads e Programação Concorrente em Java

Exercícios sobre Threads

  • 1. Crie um programa que inicie uma thread para imprimir números de 1 a 10 com um intervalo de 1 segundo entre eles.
  • 2. Desenvolva um programa que crie duas threads que alternem a impressão de "Hello" e "World".
  • 3. Implemente uma thread que calcule a soma dos números de 1 a 100 e imprima o resultado.
  • 4. Escreva um programa que utilize múltiplas threads para imprimir a tabela de multiplicação de 1 a 10.
  • 5. Crie um exemplo de uma thread que pause sua execução por 2 segundos antes de continuar.
  • 6. Implemente um programa que crie uma thread para contar de 1 a 100 em intervalos de 0.5 segundos e outra para contar de 100 a 1.
  • 7. Desenvolva uma aplicação que utilize uma thread para baixar um arquivo de internet em segundo plano.
  • 8. Crie um exemplo de um programa com uma thread que calcule a média de uma lista de números e imprima o resultado.
  • 9. Escreva um programa que utilize `Thread.sleep()` para simular uma pausa de execução e depois continuar.
  • 10. Implemente um exemplo em que uma thread é interrompida e um tratamento de exceção é realizado.
  • 11. Desenvolva um programa que use `Runnable` para criar e iniciar threads.
  • 12. Crie uma thread que, ao ser executada, imprime uma mensagem em intervalos de 1 segundo até que a execução seja interrompida.
  • 13. Implemente um programa que crie uma thread que imprime uma mensagem e depois se torna "daemon" para terminar automaticamente.
  • 14. Desenvolva um exemplo de thread que leia dados de um arquivo e os imprima na tela.
  • 15. Crie um exemplo de um programa que utilize um contador em uma thread compartilhada e evite condições de corrida com `synchronized`.
  • 16. Escreva um programa que crie uma thread para simular um processo de renderização em uma aplicação gráfica.
  • 17. Implemente um exemplo que use `ExecutorService` para gerenciar múltiplas threads simultâneas.
  • 18. Crie uma thread que execute uma tarefa em um loop infinito até que um sinal de parada seja recebido.
  • 19. Desenvolva um programa que use `CountDownLatch` para aguardar que várias threads terminem suas tarefas antes de continuar.
  • 20. Escreva um exemplo que use `Semaphore` para limitar o acesso de threads a um recurso compartilhado.
  • 21. Implemente um programa que utilize `Future` e `Callable` para retornar um resultado de execução de thread.
  • 22. Crie uma aplicação que simule a execução de um sistema de vendas com threads para diferentes tipos de transações.
  • 23. Desenvolva um exemplo de um programa que use `ThreadLocal` para armazenar dados específicos de uma thread.
  • 24. Escreva um exemplo de uma thread que utilize um `ExecutorService` para agendar tarefas repetitivas com `ScheduledExecutorService`.
  • 25. Implemente um programa com threads que simule a execução de uma contagem regressiva de um temporizador.
  • 26. Crie um exemplo em que várias threads escrevem e leem de uma lista compartilhada com `ReentrantLock`.
  • 27. Desenvolva um programa que utilize `Thread.join()` para garantir que uma thread principal aguarde a conclusão de outras threads.
  • 28. Escreva um programa em que uma thread envie dados para uma fila e outra receba e processe esses dados.
  • 29. Implemente um exemplo de um programa com `ExecutorService` para gerenciar tarefas de execução assíncrona.
  • 30. Crie um exemplo em que uma thread de leitura e uma thread de escrita competem pelo mesmo recurso e utilize `ReadWriteLock`.
  • 31. Desenvolva um programa que utilize `Thread.sleep()` e um loop de verificação para processar tarefas em intervalos regulares.
  • 32. Escreva um exemplo de um programa que tenha uma thread para fazer download de imagens enquanto outra exibe uma barra de progresso.
  • 33. Implemente um programa que utilize `ForkJoinPool` para dividir e conquistar uma tarefa de processamento paralelo.
  • 34. Crie um exemplo de um produtor-consumidor usando `BlockingQueue`.
  • 35. Desenvolva um programa em que uma thread realize operações matemáticas intensivas e outra verifique a integridade do resultado.
  • 36. Escreva um exemplo de uma aplicação de simulação de um sistema de impressão com várias threads gerenciando filas de impressão.
  • 37. Implemente uma thread que monitore o uso de memória e envie alertas quando determinado limite for atingido.
  • 38. Crie um exemplo em que uma thread execute uma tarefa demorada enquanto outras realizam tarefas menores em paralelo.
  • 39. Desenvolva um programa que use `CompletableFuture` para encadear tarefas assíncronas e processar resultados.
  • 40. Implemente um sistema de logging em um programa multithreaded com uso de `synchronized` para evitar conflitos de escrita.
  • 41. Crie um exemplo de um sistema de votação em que uma thread representa um usuário que vota e outra computa os resultados.
  • 42. Escreva um programa com threads que gerenciem diferentes tipos de tarefas em uma fila de prioridade usando `PriorityQueue`.
  • 43. Desenvolva uma aplicação que use `ExecutorService` para processar tarefas de forma assíncrona com um pool de threads.
  • 44. Implemente um exemplo de um jogo multiplayer em que cada jogador é representado por uma thread.
  • 45. Crie uma aplicação que utilize `Thread.sleep()` e `Thread.interrupt()` para gerenciar a interrupção de uma tarefa de longa execução.
  • 46. Escreva um exemplo que implemente um serviço de envio de notificações em tempo real com várias threads de background.
  • 47. Implemente um programa que utilize `Callable` e `ExecutorService` para processar tarefas paralelas e coletar resultados.
  • 48. Crie uma aplicação de backup em que uma thread faça a cópia de arquivos enquanto outra monitore a integridade do processo.
  • 49. Desenvolva um exemplo que use `Thread.sleep()` e `ThreadLocal` para gerenciar variáveis específicas de uma thread.
  • 50. Implemente um sistema de atendimento automatizado em uma central de chamadas com threads para cada cliente.
