O que são Laços de Repetição?

Laços de repetição são estruturas de controle que permitem a execução de um bloco de código múltiplas vezes, com base em uma condição. Em Ruby, existem diferentes tipos de laços que podem ser usados para criar loops em seu código.

Laço while

O laço while executa um bloco de código enquanto a condição especificada for verdadeira.

contador = 0
while contador < 5
  puts "Contador é #{contador}"
  contador += 1
end

Se a condição for falsa na primeira verificação, o bloco de código não será executado.

Laço until

O laço until é o oposto do while e executa um bloco de código enquanto a condição for falsa.

contador = 0
until contador >= 5
  puts "Contador é #{contador}"
  contador += 1
end

O loop termina assim que a condição se tornar verdadeira.

Laço for

O laço for é usado para iterar sobre uma faixa de valores. É uma maneira de repetir um bloco de código um número específico de vezes.

for i in 0..4
  puts "Valor de i: #{i}"
end

Esse loop percorre todos os valores de 0 a 4, inclusive.

Método each

O método each é usado para iterar sobre elementos de uma coleção, como arrays e hashes.

[1, 2, 3, 4, 5].each do |numero|
  puts "Número: #{numero}"
end

Esse método é muito útil para percorrer elementos de uma coleção de forma elegante e fácil de ler.

Comandos break e next

Os comandos break e next podem ser usados para controlar a execução dos laços. 

for i in 0..10
  break if i == 5
  next if i.even?
  puts "Valor ímpar: #{i}"
end

break interrompe o loop quando a condição é atendida, e next pula a iteração atual quando a condição é atendida.
