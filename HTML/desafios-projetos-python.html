Desafios e Projetos Práticos em Python

Desafios

  • 1. Crie um jogo de adivinhação em Python onde o usuário deve tentar adivinhar um número aleatório gerado pelo computador.
  • 2. Desenvolva um script para automatizar o envio de e-mails em massa com anexos usando a biblioteca `smtplib`.
  • 3. Implemente uma função que faça a verificação de um número primo e retorne todos os números primos até um número especificado pelo usuário.
  • 4. Escreva um programa para simular um jogo da velha (tic-tac-toe) que permita a dois jogadores jogarem em um terminal.
  • 5. Crie um script que analise um arquivo de log e conte quantas vezes cada tipo de erro (por exemplo, 404, 500) aparece.
  • 6. Desenvolva uma função que simule um algoritmo de busca em profundidade (DFS) em um grafo representado por uma lista de adjacência.
  • 7. Implemente uma função que faça a conversão de números romanos para números arábicos.
  • 8. Escreva um programa que receba uma lista de strings e retorne a maior palavra, a menor palavra e a palavra mais frequente.
  • 9. Crie uma função que receba uma string e retorne uma versão da string com as palavras embaralhadas aleatoriamente.
  • 10. Desenvolva um algoritmo que encontre o caminho mais curto entre duas cidades em um mapa usando o algoritmo de Dijkstra.

Projetos Práticos

  • 1. Crie um gerenciador de tarefas em Python com uma interface de linha de comando que permita adicionar, remover e marcar tarefas como concluídas.
  • 2. Desenvolva uma aplicação em Python que consulte uma API de clima e exiba as condições meteorológicas de uma cidade em tempo real.
  • 3. Implemente um programa de reconhecimento de voz que capture a entrada do microfone e converta a fala em texto.
  • 4. Crie uma aplicação de rastreamento de despesas financeiras que permita ao usuário inserir despesas e gerar relatórios mensais.
  • 5. Desenvolva um chatbot simples usando a biblioteca `NLTK` para responder a perguntas comuns com base em palavras-chave.
  • 6. Escreva um programa de web scraping para coletar informações de um site de notícias e exibir as manchetes mais recentes em um arquivo de texto.
  • 7. Implemente um sistema de recomendação de filmes baseado em análise de dados de filmes e preferências do usuário usando bibliotecas como `pandas` e `scikit-learn`.
  • 8. Crie um aplicativo de conversão de moeda que utilize uma API de câmbio para converter valores entre diferentes moedas em tempo real.
  • 9. Desenvolva um simulador de jogo de cartas em Python com regras básicas de um jogo de poker ou blackjack.
  • 10. Escreva um programa de análise de sentimentos que utilize uma API de processamento de linguagem natural para classificar tweets como positivos, negativos ou neutros.
