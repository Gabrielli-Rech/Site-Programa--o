O que são Estruturas Condicionais?

Estruturas condicionais são usadas para executar diferentes blocos de código dependendo de uma condição. Em Swift, isso é feito principalmente com as instruções if, else e switch.

Estrutura if-else

A estrutura if-else é usada para executar um bloco de código se a condição for verdadeira e outro bloco se for falsa.

let numero = 10
if numero > 0 {
    print("O número é positivo")
} else if numero < 0 {
    print("O número é negativo")
} else {
    print("O número é zero")
}

Estrutura switch

A estrutura switch é usada para comparar um valor com múltiplas possibilidades de forma mais legível e eficiente. Ela é mais poderosa que a estrutura if-else quando há muitas condições a serem verificadas.

let diaDaSemana = "Terça-feira"
switch diaDaSemana {
case "Segunda-feira":
    print("Hoje é segunda-feira")
case "Terça-feira":
    print("Hoje é terça-feira")
case "Quarta-feira":
    print("Hoje é quarta-feira")
default:
    print("Outro dia da semana")
}

Observação: O switch em Swift deve ser completo, ou seja, deve cobrir todos os casos possíveis. Se não for possível, você deve usar o default.

Operador Ternário

O operador ternário é uma forma concisa de escrever uma estrutura if-else. Ele é utilizado quando a lógica for simples.

let numero = 5
let resultado = numero > 0 ? "Positivo" : "Negativo"
print(resultado)

O operador ternário ? verifica a condição e, se for verdadeira, retorna o valor antes do :. Se for falsa, retorna o valor após o :.
