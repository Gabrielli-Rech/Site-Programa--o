O que são Estruturas Condicionais?

As estruturas condicionais em Python são usadas para executar diferentes blocos de código com base em uma condição. Elas são fundamentais para a lógica de programação, permitindo que você tome decisões com base em condições específicas.

Principais Estruturas Condicionais

Em Python, as estruturas condicionais mais comuns são:

  • if: Executa um bloco de código se uma condição for verdadeira.
  • elif: Verifica uma condição adicional se a condição do if for falsa.
  • else: Executa um bloco de código se a condição do if e as condições do elif forem falsas.

Exemplo de uma estrutura condicional if:

idade = 18
if idade >= 18:
    print('Você é maior de idade.')
else:
    print('Você é menor de idade.')
