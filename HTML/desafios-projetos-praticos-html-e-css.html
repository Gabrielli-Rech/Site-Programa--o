Exercícios Desafios e Projetos Práticos

Desafios Fáceis

  • 1. Criar uma página HTML básica com uma estrutura de cabeçalho, corpo e rodapé.
  • 2. Adicionar uma lista ordenada e não ordenada em uma página HTML.
  • 3. Criar um formulário simples com campos de texto, botão de envio e caixas de seleção.
  • 4. Estilizar um parágrafo com a tag <style> no cabeçalho da página.
  • 5. Inserir uma imagem na página e ajustar seu tamanho usando CSS.
  • 6. Criar um botão que muda de cor ao passar o mouse (hover) usando CSS.
  • 7. Criar uma tabela com três colunas e três linhas, e aplicar estilos básicos.
  • 8. Adicionar um link que muda de cor ao ser clicado usando CSS.
  • 9. Criar um layout de duas colunas usando flexbox.
  • 10. Estilizar uma lista de navegação horizontal com links.

Desafios Médios

  • 11. Criar um card de produto com uma imagem, título e descrição usando HTML e CSS.
  • 12. Criar uma seção de contato com um formulário que inclui campos de texto e um botão de envio.
  • 13. Criar uma galeria de imagens responsiva usando grid.
  • 14. Adicionar animações simples, como um efeito de desvanecimento em uma imagem com CSS.
  • 15. Criar um cabeçalho fixo no topo da página usando CSS.
  • 16. Fazer um layout de página de blog com um cabeçalho, conteúdo e rodapé.
  • 17. Criar um botão de "voltar ao topo" que aparece quando o usuário rola a página.
  • 18. Adicionar um ícone de "hambúrguer" que abre um menu de navegação em dispositivos móveis.
  • 19. Criar um formulário de login estilizado com validação de entrada em CSS.
  • 20. Criar uma animação de rotação em uma imagem ao passar o mouse (hover).

Desafios Difíceis

  • 21. Criar um layout de site completo com cabeçalho, barra lateral, conteúdo principal e rodapé usando CSS Grid.
  • 22. Implementar um modal que pode ser aberto e fechado com um botão usando HTML, CSS e JavaScript.
  • 23. Criar um efeito de paralaxe de rolagem em uma página usando CSS.
  • 24. Criar um formulário com validação personalizada usando apenas HTML e CSS (sem JavaScript).
  • 25. Estilizar um site de notícias com colunas diferentes para a manchete e artigos relacionados.
  • 26. Criar uma animação de "fade-in" para um texto quando a página for carregada.
  • 27. Criar um layout de cartão interativo com um efeito de "flip" (virar) usando CSS.
  • 28. Criar um slideshow de imagens responsivo que se adapta a diferentes tamanhos de tela.
  • 29. Criar um efeito de hover que aumenta o tamanho e a opacidade de uma imagem.
  • 30. Desenvolver um menu de navegação que desliza para dentro e para fora da tela.

Desafios Avançados

  • 31. Criar um layout de página com várias seções, usando CSS Flexbox e Grid para um design responsivo.
  • 32. Criar uma animação de escrita simulando o efeito de digitação em um texto usando CSS.
  • 33. Implementar uma barra de progresso interativa que aumenta conforme o usuário interage com a página.
  • 34. Criar uma interface de aplicativo de tarefas com drag-and-drop usando apenas HTML e CSS (pouco uso de JavaScript).
  • 35. Criar um efeito de luz interativa que segue o cursor do mouse.
  • 36. Criar um layout de página com um efeito de navegação horizontal.
  • 37. Desenvolver uma página com um fundo animado que muda de cor de forma cíclica.
  • 38. Criar uma animação que faz um texto "dançar" ao ser exibido na página.
  • 39. Criar um formulário de múltiplas etapas com navegação entre etapas usando apenas HTML e CSS.
  • 40. Desenvolver um layout de galeria de imagens com filtros e efeitos de hover personalizados.
