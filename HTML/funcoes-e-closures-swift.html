Exercícios Fáceis

  • Crie uma função simples que recebe um número inteiro e imprime "Olá, Swift!".
  • Crie uma função que receba dois números e retorne a soma deles.
  • Escreva uma função que retorne o maior de dois números passados como parâmetros.
  • Crie uma função que recebe um array de inteiros e imprime cada elemento em uma linha.
  • Defina uma função que recebe uma string e retorna a mesma string em letras maiúsculas.
  • Escreva uma função que calcule a área de um retângulo com base e altura passadas como parâmetros.
  • Crie uma função que receba um array de strings e retorne o número de elementos.
  • Faça uma função que recebe um número e imprime "Número par" ou "Número ímpar" dependendo do caso.
  • Implemente uma função que retorne o quadrado de um número.
  • Crie uma função que verifique se uma palavra é palíndromo.

Exercícios Médios

  • Crie uma função que aceite um array de inteiros e retorne a soma de todos os elementos.
  • Escreva uma função que conte o número de vogais em uma string.
  • Crie uma função que receba um array de strings e retorne a string mais longa.
  • Implemente uma função que receba uma função como parâmetro e a execute.
  • Escreva uma função que verifique se um número é primo.
  • Crie uma função de ordenação personalizada para um array de inteiros.
  • Desenvolva uma função que combine duas listas de strings em uma única lista.
  • Crie uma função que, usando closures, multiplique cada elemento de um array por 2.
  • Escreva uma função que aceite uma string e um número e repita a string o número de vezes indicado.
  • Implemente uma função que receba um array de inteiros e retorne o maior valor ímpar.

Exercícios Difíceis

  • Crie uma função de ordenação de um array de inteiros que utilize closures como parâmetro de comparação.
  • Implemente uma closure que receba dois números e retorne a média deles.
  • Crie uma função recursiva que calcule o fatorial de um número.
  • Desenvolva uma função que retorne a Fibonacci de um número, usando recursão e closures.
  • Implemente uma closure que percorra um array de números e retorne o menor número.
  • Crie uma função de busca em um array de strings e utilize uma closure para determinar se a string contém uma palavra específica.
  • Desenvolva uma função que filtre todos os números pares de um array de inteiros usando uma closure.
  • Escreva uma função que retorne a soma de todos os elementos de um array, usando recursão e closures.
  • Implemente uma closure que seja capaz de calcular a potência de um número.
  • Crie uma função que aceita uma closure como parâmetro e utilize-a para calcular a soma de um array de números.

Desafios

  • Crie uma função que combine duas arrays de inteiros e remova elementos duplicados, utilizando closures.
  • Implemente uma closure que verifique se uma string é um anagrama de outra.
  • Crie uma função que receba um array de números e uma closure que determine se um número é primo, e retorne todos os números primos do array.
  • Desenvolva uma função de busca binária recursiva em Swift.
  • Crie uma closure que aceite um array e retorne o elemento mais frequente.
  • Implemente uma função de mapeamento que converta uma array de strings em uma array de números inteiros.
  • Escreva uma função que aceita uma string e retorne uma array com todas as palavras que contêm uma vogal específica.
  • Desenvolva uma função que calcule a soma de uma série infinita, usando closures e recursão.
  • Crie uma closure que retorne o maior divisor comum de dois números.
  • Implemente uma função que utilize closures para realizar operações de adição, subtração, multiplicação e divisão de forma dinâmica.
