Exercícios de Consultas e Operações em PostgreSQL

Fáceis

  • 1. Escreva uma consulta SQL para selecionar todos os campos de uma tabela específica.
  • 2. Realize uma consulta SQL que retorne apenas as linhas de uma tabela onde um campo específico tenha um valor específico.
  • 3. Use a cláusula `WHERE` para filtrar registros de uma tabela com um operador de comparação (`=`, `>`, `<`).
  • 4. Escreva uma consulta SQL para selecionar apenas campos específicos de uma tabela.
  • 5. Realize uma consulta com a cláusula `ORDER BY` para ordenar os resultados de uma tabela em ordem crescente de um campo específico.
  • 6. Use `ORDER BY` com `DESC` para ordenar os resultados em ordem decrescente.
  • 7. Escreva uma consulta para contar o número de linhas de uma tabela usando `COUNT()`.
  • 8. Realize uma consulta para encontrar o maior valor em uma coluna usando `MAX()`.
  • 9. Use a função `SUM()` para calcular a soma de uma coluna numérica.
  • 10. Escreva uma consulta que use a função `AVG()` para encontrar a média de uma coluna numérica.
  • 11. Utilize `LIMIT` para limitar o número de linhas retornadas em uma consulta.
  • 12. Escreva uma consulta que use `DISTINCT` para retornar apenas valores únicos de uma coluna.
  • 13. Realize uma consulta que utilize `GROUP BY` para agrupar resultados de uma tabela por um campo específico.
  • 14. Use `HAVING` para filtrar grupos em uma consulta `GROUP BY`.
  • 15. Escreva uma consulta que utilize `JOIN` para combinar dados de duas tabelas diferentes.
  • 16. Crie uma consulta que use `INNER JOIN` para combinar registros de duas tabelas que possuem valores correspondentes.
  • 17. Realize uma consulta com `LEFT JOIN` para incluir todos os registros da tabela da esquerda, mesmo que não haja correspondência na tabela da direita.
  • 18. Escreva uma consulta usando `RIGHT JOIN` para incluir todos os registros da tabela da direita, mesmo que não haja correspondência na tabela da esquerda.
  • 19. Use `BETWEEN` para encontrar registros dentro de um intervalo de valores.
  • 20. Escreva uma consulta usando `IN` para filtrar registros com base em uma lista de valores.
