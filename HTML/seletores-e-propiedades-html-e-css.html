O que são Seletores e Propriedades em CSS?

Em CSS, seletores são usados para identificar elementos HTML aos quais as regras de estilo serão aplicadas. As propriedades definem as características de estilo desses elementos, como cor, tamanho, layout e muito mais.

Seletores

Os seletores são a forma de apontar para os elementos HTML que você deseja estilizar. Alguns dos seletores mais comuns incluem:

  • Seletores de Elemento: Apontam para todos os elementos de um tipo específico. Exemplo: p {} aplica-se a todos os elementos <p>.
  • Seletores de Classe: Apontam para elementos com uma classe específica. Exemplo: .exemplo {} aplica-se a todos os elementos com a classe "exemplo".
  • Seletores de ID: Apontam para um elemento específico com um ID. Exemplo: #titulo {} aplica-se ao elemento com o ID "titulo".
  • Seletores de Atributo: Apontam para elementos que possuem um atributo específico. Exemplo: [type="text"] {} aplica-se a elementos com o atributo type="text".
  • Seletores de Descendentes: Apontam para elementos que são descendentes de outro elemento. Exemplo: div p {} aplica-se a todos os elementos <p> dentro de <div>.

Propriedades

As propriedades são usadas para definir o estilo de um elemento selecionado. Algumas das propriedades mais comuns incluem:

  • Cor: Define a cor do texto. Exemplo: color: blue;
  • Fonte: Define o tipo de fonte. Exemplo: font-family: 'Arial', sans-serif;
  • Tamanho da Fonte: Define o tamanho do texto. Exemplo: font-size: 16px;
  • Cor de Fundo: Define a cor de fundo de um elemento. Exemplo: background-color: lightgray;
  • Margem: Define o espaço fora do elemento. Exemplo: margin: 10px;
  • Preenchimento: Define o espaço dentro do elemento. Exemplo: padding: 5px;
  • Borda: Define a borda do elemento. Exemplo: border: 1px solid black;

Exemplo de Uso de Seletores e Propriedades

Veja como aplicar seletores e propriedades em um exemplo prático:

/* Seleciona todos os elementos 
 e aplica as propriedades */ 
p {
    color: blue;
    font-size: 16px;
    margin: 10px;
    padding: 5px;
    background-color: lightgray;
}

Este exemplo faz com que todos os elementos <p> tenham texto azul, tamanho de fonte de 16px, margem de 10px, preenchimento de 5px e um fundo cinza claro.
