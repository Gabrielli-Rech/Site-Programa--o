Desafios Fáceis

  • 1. Criar uma função que soma dois números com validação de tipos.
  • 2. Implementar uma interface para descrever um objeto `Pessoa` com `nome` e `idade`.
  • 3. Escrever uma função que verifica se um número é par ou ímpar.
  • 4. Criar um array tipado de strings e exibir os valores no console.
  • 5. Desenvolver uma função que retorna o comprimento de uma string.
  • 6. Criar um tipo union para representar um status que pode ser "ativo", "inativo" ou "pendente".
  • 7. Implementar um enum para representar os dias da semana.
  • 8. Criar uma função que recebe um array de números e retorna o maior número.
  • 9. Desenvolver uma classe simples `Carro` com propriedades `marca` e `modelo`.
  • 10. Escrever uma função que converte um número em uma string formatada.

Desafios Médios

  • 11. Criar uma classe base `Animal` com métodos genéricos e heranças para subclasses como `Cachorro` e `Gato`.
  • 12. Implementar uma função genérica que aceita um array de qualquer tipo e retorna seu tamanho.
  • 13. Desenvolver um programa que utiliza um map para contar a frequência de palavras em uma string.
  • 14. Criar um tipo personalizado para validar dados de um formulário de login.
  • 15. Implementar uma interface para definir os métodos e propriedades de uma calculadora.
  • 16. Escrever uma classe abstrata `Forma` com métodos para calcular área e perímetro.
  • 17. Desenvolver uma função que converte uma lista de objetos em um mapa indexado por uma chave específica.
  • 18. Criar uma aplicação simples que consome uma API e exibe os dados usando `fetch`.
  • 19. Implementar uma solução para ordenar um array de objetos por uma propriedade específica.
  • 20. Desenvolver um programa que utiliza decorators para registrar o tempo de execução de funções.

Desafios Difíceis

  • 21. Criar um sistema de gerenciamento de tarefas com CRUD básico usando classes e interfaces.
  • 22. Implementar um programa que usa generics avançados para manipular listas de dados.
  • 23. Desenvolver uma função que valida tipos e lança erros personalizados para tipos inválidos.
  • 24. Criar uma aplicação que utiliza RxJS para gerenciar fluxos de eventos em tempo real.
  • 25. Implementar um decorator para verificar permissões de métodos em uma classe.
  • 26. Criar uma biblioteca personalizada para realizar operações matemáticas complexas com validação de tipos.
  • 27. Desenvolver uma função que faz o merge de dois objetos mantendo as propriedades e tipos de ambos.
  • 28. Criar um sistema de eventos com listeners e disparos, implementado com classes.
  • 29. Implementar um algoritmo que utiliza `Promises` para simular tarefas assíncronas.
  • 30. Criar um programa que utiliza decorators para registrar logs de execução de métodos.
