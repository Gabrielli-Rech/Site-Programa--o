Exercícios Práticos

  • Crie uma página com uma tag <header> que contenha um título <h1>.
  • Adicione um parágrafo <p> com algum texto de exemplo.
  • Insira uma lista ordenada <ol> com pelo menos três itens.
  • Crie uma lista não ordenada <ul> com pelo menos três itens e adicione um item de lista <li> aninhado.
  • Insira uma tabela <table> com pelo menos duas linhas e duas colunas, incluindo os elementos <tr>, <th> e <td>.
  • Adicione uma imagem usando a tag <img> e defina um caminho para a imagem local ou uma URL de uma imagem da web.
  • Crie um link de navegação com a tag <a> que leva a uma página externa.
  • Adicione uma seção <section> com um título e um parágrafo.
  • Crie um formulário <form> com um campo de texto, um botão de envio e um rótulo usando a tag <label>.
  • Insira uma tag de vídeo <video> e adicione um botão de reprodução.
  • Adicione um elemento de áudio <audio> com um arquivo de áudio local ou de URL e defina a opção de controle.
  • Crie uma tag de definição de descrição <dl> com uma definição de termo e sua descrição usando <dt> e <dd>.
  • Utilize a tag <blockquote> para incluir uma citação em destaque em sua página.
  • Adicione um elemento de divisão <div> para criar uma área separada em sua página e insira algum texto dentro dela.
  • Crie um elemento <span> e aplique um estilo CSS embutido para alterar a cor do texto.
  • Use a tag <strong> para destacar um trecho de texto como importante e a tag <em> para enfatizar um trecho.
  • Adicione um botão <button> com um texto de sua escolha e uma ação básica ao clicar.
  • Insira uma tag de cabeçalho secundário <h2> com um texto de exemplo.
  • Crie uma lista de links <ul> que levem a três páginas diferentes.
  • Adicione uma tag de código <code> para exibir um pequeno trecho de código em linha.
  • Crie uma tag de comentário HTML <!-- --> para incluir uma anotação que não será exibida na página.
