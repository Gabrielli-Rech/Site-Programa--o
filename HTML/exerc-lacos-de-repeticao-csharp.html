Exercícios Básicos

  1. Imprima os números de 1 a 10 usando um laço `for`.
  2. Crie um programa que imprima os números de 10 a 1 usando um laço `while`.
  3. Use um laço `do-while` para imprimir os números de 1 a 5.
  4. Imprima os números pares de 1 a 20 usando um laço `for`.
  5. Imprima os números ímpares de 1 a 20 usando um laço `while`.
  6. Calcule a soma dos números de 1 a 100 usando um laço `for`.
  7. Imprima a tabuada do número 5 usando um laço `for`.
  8. Imprima os múltiplos de 3 de 1 a 30 usando um laço `for`.
  9. Peça ao usuário um número e imprima a tabuada desse número usando um laço `for`.
  10. Use um laço `while` para contar de 50 a 0 de 5 em 5.
  11. Peça ao usuário para inserir números até que ele digite um número negativo.
  12. Imprima os números de 1 a 100, mas pule os múltiplos de 5 usando a instrução `continue`.
  13. Use um laço `for` para imprimir a sequência de Fibonacci até o 10º termo.
  14. Peça ao usuário para digitar números até que a soma deles seja maior que 100.
  15. Use um laço `do-while` para pedir uma senha ao usuário até que ele insira a senha correta.
  16. Imprima os primeiros 10 números da sequência de Fibonacci usando um laço `while`.
  17. Use um laço para contar quantos números entre 1 e 100 são divisíveis por 7.
  18. Imprima os quadrados dos números de 1 a 20 usando um laço `for`.
  19. Calcule o fatorial de um número inserido pelo usuário usando um laço `for`.
  20. Use um laço para inverter uma string inserida pelo usuário.

Exercícios Intermediários

  1. Peça ao usuário um número e verifique se ele é primo usando um laço.
  2. Imprima todos os números primos entre 1 e 100 usando um laço aninhado.
  3. Crie um programa que simule um caixa eletrônico que aceita depósitos e saques até que o usuário escolha sair.
  4. Imprima os números de 1 a 100, substituindo múltiplos de 3 por "Fizz", múltiplos de 5 por "Buzz" e múltiplos de ambos por "FizzBuzz".
  5. Calcule a soma dos dígitos de um número usando um laço `while`.
  6. Peça ao usuário um número e desenhe uma pirâmide com `*` até esse número usando laços.
  7. Imprima os números de 1 a 100 em ordem reversa, mas exclua os múltiplos de 7 usando `continue`.
  8. Use um laço para verificar se uma palavra é um palíndromo.
  9. Crie um programa que conte quantas vogais existem em uma string fornecida pelo usuário.
  10. Simule o lançamento de um dado 10 vezes e exiba os resultados usando um laço.
  11. Calcule a soma dos números ímpares de 1 a 100 usando um laço.
  12. Peça ao usuário um número e determine quantos dígitos ele tem usando um laço.
  13. Imprima um triângulo invertido de `*` com base em um número inserido pelo usuário.
  14. Imprima a soma de todos os números na diagonal principal de uma matriz 3x3.
  15. Use um laço para encontrar o maior número em uma lista de números inseridos pelo usuário.
  16. Imprima um padrão de losango com base em um número de linhas fornecido pelo usuário.
  17. Peça ao usuário para inserir 5 números e armazene-os em uma matriz. Imprima os números em ordem inversa.
  18. Simule um jogo de adivinhação onde o usuário deve adivinhar um número gerado aleatoriamente.
  19. Use um laço para encontrar o menor número em um array de números inseridos pelo usuário.
  20. Crie um programa que exiba os números de 1 a 100 e conte quantos são divisíveis por 3 e 5.

Exercícios Avançados

  1. Crie um programa que gere todos os números possíveis de um dado de 4 faces e some os resultados.
  2. Simule o cálculo de juros compostos para 5 anos com um valor inicial fornecido pelo usuário.
  3. Crie um programa que armazene nomes em uma lista e permita a remoção ou adição de elementos até que o usuário decida sair.
  4. Calcule a soma de todos os números em um array de números inteiros fornecidos pelo usuário.
  5. Crie uma calculadora que opere até que o usuário escolha sair.
  6. Simule uma fila de banco onde os clientes são atendidos em ordem de chegada.
  7. Crie um algoritmo para reverter a ordem de um array usando um laço.
  8. Use um laço para implementar uma função que encontre o maior divisor comum (MDC) entre dois números.
  9. Calcule o valor de um número elevado a uma potência usando apenas laços.
  10. Simule uma matriz de 5x5 e imprima a soma de todos os seus elementos.
  11. Peça ao usuário para digitar números até que a média de todos eles seja maior que 50.
  12. Implemente um algoritmo que gere as combinações possíveis de duas listas de strings.
  13. Use laços para calcular o número de combinações possíveis para um grupo de 5 pessoas escolhidas de um grupo maior.
  14. Implemente um programa que encontre a sequência de Collatz para qualquer número inteiro.
  15. Use laços para desenhar um padrão como um tabuleiro de xadrez com `*` e `-`.
  16. Crie um algoritmo para determinar se um número é um número perfeito (a soma de seus divisores é igual ao número).
  17. Simule um contador binário de 8 bits usando laços.
  18. Desenhe um padrão espiral em uma matriz de números inteiros.
  19. Crie um algoritmo que converta números decimais em binários usando apenas laços.
  20. Implemente o método de Newton-Raphson para encontrar a raiz quadrada de um número usando laços.
