O que são Coleções no MongoDB?

No MongoDB, uma coleção é um agrupamento de documentos, equivalente a uma tabela em bancos de dados relacionais. Coleções são armazenadas em bases de dados e podem conter documentos com estruturas diferentes.

Criação de Coleções

Para criar uma coleção no MongoDB, você pode usar o método db.createCollection() ou simplesmente inserir um documento em uma coleção inexistente, que será criada automaticamente.

Usando db.createCollection():

db.createCollection("nome_da_colecao");

Exemplo de criação de uma coleção chamada "clientes":

db.createCollection("clientes");

Inserindo um documento em uma coleção inexistente:

db.clientes.insertOne({ nome: "Ana", idade: 30, cidade: "São Paulo" });

Se a coleção "clientes" não existir, ela será criada automaticamente ao inserir o primeiro documento.

Estrutura de Coleções

As coleções podem conter documentos com diferentes estruturas. Cada documento é armazenado em formato JSON-like, e pode ter campos e tipos de dados variados.

Exemplo de documentos em uma coleção "clientes":

db.clientes.insertMany([
    { nome: "Carlos", idade: 25, cidade: "Rio de Janeiro" },
    { nome: "Mariana", idade: 28, cidade: "Belo Horizonte" },
    { nome: "Lucas", idade: 22, cidade: "Porto Alegre" }
]);

Os documentos podem ter diferentes campos e tipos de dados, permitindo flexibilidade na modelagem de dados.

Listando e Verificando Coleções

Para listar todas as coleções em uma base de dados, use o comando show collections:

show collections;

Para verificar a existência de uma coleção específica, use o método db.getCollectionNames():

db.getCollectionNames();

Exclusão de Coleções

Para excluir uma coleção existente, use o comando db.collection.drop().

db.clientes.drop();

Isso removerá a coleção "clientes" e todos os seus documentos.
