Exercícios Avançados PostgreSQL

Nível Fácil

  1. Escreva uma consulta que liste todas as tabelas em um banco de dados PostgreSQL.
  2. Crie uma função que retorne o número de linhas em uma tabela específica.
  3. Use a cláusula DISTINCT para listar valores únicos de uma coluna.
  4. Escreva uma consulta que utilize a função COUNT para contar o número de registros em uma tabela.
  5. Crie uma consulta que utilize a função SUM para calcular o total de uma coluna numérica.
  6. Escreva uma consulta que utilize a cláusula GROUP BY para agrupar resultados de uma tabela.
  7. Utilize a função AVG para calcular a média de uma coluna de valores numéricos.
  8. Crie uma consulta que use a cláusula WHERE para filtrar resultados de acordo com um critério.
  9. Escreva uma consulta com a função MAX para encontrar o maior valor em uma coluna.
  10. Use a função MIN para encontrar o menor valor em uma coluna de dados.
  11. Crie uma consulta com JOIN para combinar duas tabelas com base em uma chave estrangeira.
  12. Utilize a cláusula ORDER BY para ordenar os resultados de uma consulta em ordem crescente.
  13. Escreva uma consulta que utilize a função COALESCE para lidar com valores NULL.
  14. Crie uma consulta que use a cláusula LIMIT para retornar apenas um número específico de linhas.
  15. Escreva uma consulta que utilize INNER JOIN e LEFT JOIN em uma única consulta.
  16. Crie uma tabela com um índice em uma coluna específica.
  17. Escreva uma consulta que utilize a função TO_CHAR para formatar uma data.
  18. Crie uma consulta que utilize a função DATE_PART para extrair partes de uma data.
  19. Escreva uma consulta que use a função STRING_AGG para concatenar valores de uma coluna.
  20. Utilize a cláusula EXCEPT para encontrar registros que estão em uma tabela, mas não na outra.

Nível Médio

  1. Crie uma subconsulta para encontrar a média de uma coluna e filtrar registros acima dessa média.
  2. Escreva uma consulta que use a função ARRAY para armazenar e acessar elementos de um array.
  3. Crie uma função armazenada para calcular o fatorial de um número.
  4. Escreva uma consulta que utilize a função ROW_NUMBER() para numerar os registros de uma tabela.
  5. Crie uma consulta que utilize a cláusula CASE para retornar diferentes valores de acordo com condições.
  6. Escreva uma consulta que utilize a função LAG() para comparar valores em linhas anteriores.
  7. Crie uma consulta com a função CROSS JOIN e explique o resultado.
  8. Utilize a função JSONB para consultar um campo JSONB em uma tabela.
  9. Escreva uma consulta com DISTINCT ON para selecionar registros únicos com base em uma coluna específica.
  10. Crie uma consulta que utilize a função HAVING para filtrar grupos de registros.
  11. Escreva uma consulta que utilize a função STRING_TO_ARRAY para dividir uma string em um array.
  12. Crie uma consulta que utilize a função TO_TIMESTAMP para converter uma string em um timestamp.
  13. Utilize a cláusula UNION para combinar o resultado de duas consultas diferentes.
  14. Escreva uma consulta para encontrar os registros que aparecem mais de uma vez em uma tabela.
  15. Crie uma consulta que utilize a função GENERATE_SERIES para gerar uma série de números.
  16. Escreva uma consulta que utilize FILTER para calcular somas condicionais.
  17. Crie uma função para atualizar um registro em uma tabela específica.
  18. Utilize a cláusula INTERSECT para encontrar registros que estão em ambas as tabelas.
  19. Escreva uma consulta que utilize a função COUNT com a cláusula GROUP BY.
  20. Crie uma consulta que utilize a função RANGE para trabalhar com intervalos de datas.

Nível Difícil

  1. Escreva uma consulta que utilize a função PL/pgSQL para criar um bloco anônimo de código.
  2. Crie uma tabela que tenha uma chave primária composta por duas colunas.
  3. Utilize uma subconsulta correlacionada para encontrar registros com base em outra tabela.
  4. Crie uma função que calcule a diferença entre duas datas em dias úteis.
  5. Escreva uma consulta que utilize WITH RECURSIVE para criar uma árvore hierárquica de dados.
  6. Crie uma consulta que utilize a função pg_stat_activity para monitorar consultas em execução.
  7. Escreva uma consulta para buscar dados em uma tabela particionada.
  8. Utilize a função pg_trgm para encontrar correspondências de texto aproximadas em uma tabela.
  9. Crie um índice GIN em uma coluna que armazena dados JSONB.
  10. Escreva uma consulta que utilize EXPLAIN ANALYZE para analisar o plano de execução de uma consulta.
  11. Crie um script para realizar backup e restauração de um banco de dados PostgreSQL.
  12. Escreva uma consulta que utilize pg_cron para agendar tarefas de manutenção do banco de dados.
  13. Crie um procedimento armazenado que faça a importação de dados de um arquivo CSV para uma tabela.
  14. Utilize a função pg_stat_statements para encontrar as consultas mais lentas do banco de dados.
  15. Escreva uma consulta para realizar a replicação de dados entre duas tabelas diferentes.
  16. Crie uma função que faça a compressão de dados em uma tabela para economizar espaço.
  17. Utilize PL/pgSQL para criar uma função que faça a verificação de integridade referencial entre tabelas.
  18. Escreva uma consulta que utilize a função pg_locks para identificar bloqueios de banco de dados.
  19. Crie um script de migração para atualizar a estrutura de uma tabela existente.
  20. Use a função pg_restore para restaurar um banco de dados a partir de um arquivo de backup.

Nível Desafio

  1. Escreva uma consulta que utilize a extensão postgis para encontrar pontos próximos a uma localização.
  2. Crie uma tabela temporária com dados de uma consulta e use-a em outra consulta subsequente.
  3. Escreva uma função que utiliza o operador @@ para buscar por dados de texto com um índice GIN.
