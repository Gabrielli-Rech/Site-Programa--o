Eleve Suas Habilidades com Projetos Reais

Os desafios e projetos práticos ajudam a consolidar seus conhecimentos em JavaScript. Abaixo, você encontrará uma lista de desafios e ideias de projetos para criar aplicações funcionais.

Desafios e Projetos Práticos em JavaScript

Desafios

Fáceis

  • Crie uma função que valide um número de CPF ou CNPJ com base em regras de formatação e dígitos verificadores.
  • Desenvolva uma função que converta uma string de formato "camelCase" para "snake_case".
  • Implemente um contador em tempo real que exiba a quantidade de cliques em um botão na página.
  • Crie uma função que gere senhas aleatórias de um comprimento especificado pelo usuário.
  • Desenvolva um relógio digital em tempo real que exiba a data e a hora no formato `DD/MM/AAAA HH:mm:ss`.

Médios

  • Crie um jogo simples de adivinhação de números onde o usuário tenha três tentativas para acertar um número gerado aleatoriamente.
  • Desenvolva uma aplicação que leia uma lista de tarefas, permita adicionar, editar e remover tarefas, e salve o estado no armazenamento local do navegador.
  • Implemente uma calculadora científica com funcionalidades básicas e avançadas como raiz quadrada, potência, e logaritmos.
  • Crie uma função que consuma uma API pública (como uma API de clima) e exiba os dados de forma organizada no navegador.
  • Desenvolva uma aplicação que calcule a quantidade de dias, horas, minutos e segundos entre duas datas fornecidas pelo usuário.

Difíceis

  • Implemente um editor de texto em HTML que suporte funcionalidades como negrito, itálico, e lista com bullets.
  • Crie um jogo da memória onde os usuários devem encontrar pares de cartas iguais, com um cronômetro para registrar o tempo de conclusão.
  • Desenvolva um sistema de login e cadastro utilizando armazenamento local, validando se o usuário já existe antes de criar uma nova conta.
  • Crie uma visualização de gráficos simples usando a biblioteca Chart.js para exibir dados de uma API ou de entrada do usuário.
  • Desenvolva uma aplicação que permita ao usuário enviar arquivos e exiba o progresso do upload em tempo real.

Projetos Práticos

Fáceis

  • Desenvolva um site de portfólio com uma lista de projetos, um currículo e uma seção de contato com validação de formulário.
  • Implemente uma página de conversão de moedas que consuma uma API para obter as taxas de câmbio atuais.
  • Crie uma página que permita ao usuário pesquisar e filtrar receitas com base em palavras-chave e categorias.
  • Desenvolva um gerador de código QR simples que converta texto em um QR code exibido na página.
  • Implemente um sistema de votação que permita ao usuário votar em opções e exiba o total de votos em tempo real.

Médios

  • Desenvolva um sistema de carrinho de compras com itens, preços, cálculo de total e persistência no armazenamento local.
  • Crie uma aplicação de cronômetro com funcionalidades de iniciar, pausar, reiniciar e salvar os tempos registrados.
  • Implemente uma aplicação de leitura de notícias que consuma uma API pública e exiba artigos em diferentes categorias.
  • Crie um sistema de busca com filtro avançado que permita ao usuário pesquisar por dados em um conjunto de registros simulados.
  • Desenvolva uma aplicação de calendário que permita ao usuário adicionar e visualizar eventos em datas específicas.

Difíceis

  1. Desenvolva uma aplicação de gerenciamento financeiro que permita cadastrar despesas, receitas e exiba gráficos detalhados.
  2. Crie um jogo completo de Pedra, Papel e Tesoura com um placar que armazene as pontuações entre o usuário e o computador.
  3. Implemente um sistema de chat em tempo real usando WebSocket ou Firebase como backend.
  4. Desenvolva um jogo do tipo Quiz com perguntas aleatórias, temporizador e placar final.
  5. Crie um editor de imagens básico com funcionalidades como corte, redimensionamento e filtros de cor.
  6. Calculadora de Números Romanos: Crie uma função que converta números decimais em números romanos e vice-versa.
  7. Jogo da Forca: Desenvolva um jogo da forca que escolha palavras aleatórias de um array e permita o jogador adivinhar.
  8. Validador de CPF: Implemente uma função que valide números de CPF, verificando se os dígitos verificadores estão corretos.
  9. Gerador de Senhas: Crie uma aplicação que gere senhas aleatórias com opções para incluir números, letras e caracteres especiais.
  10. Criptografia Simples: Implemente uma função que use o método de substituição (como a cifra de César) para criptografar e descriptografar textos.

Projetos Práticos

  1. Lista de Tarefas: Desenvolva uma aplicação de lista de tarefas com funcionalidades para adicionar, editar e remover itens, além de salvar os dados no localStorage.
  2. Jogo da Memória: Crie um jogo da memória interativo, onde os usuários devem combinar cartas com base em imagens ou texto.
  3. Conversor de Moedas: Desenvolva um aplicativo que converta valores entre diferentes moedas, utilizando uma API externa para as taxas de câmbio.
  4. Simulador de Quiz: Crie um jogo de perguntas e respostas, com pontuação baseada em respostas corretas e incorretas.
  5. Chat em Tempo Real: Desenvolva um sistema básico de chat utilizando WebSockets para comunicação em tempo real.
