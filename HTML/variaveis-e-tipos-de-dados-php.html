O que são Variáveis e Tipos de Dados?

Em PHP, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. PHP é uma linguagem de tipagem fraca, o que significa que o tipo de dado de uma variável é determinado automaticamente pelo valor atribuído a ela.

Declaração de Variáveis

Em PHP, as variáveis são declaradas com o prefixo $ seguido do nome da variável. Por exemplo:

$nome = "João"; // string
$idade = 25; // inteiro
$altura = 1.75; // float
$ativo = true; // boolean

É importante lembrar que os nomes das variáveis em PHP são case-sensitive (diferenciam maiúsculas de minúsculas).

Tipos de Dados em PHP

PHP suporta os seguintes tipos de dados primitivos:

  • string: Cadeia de caracteres. Exemplo: "Olá, mundo!"
  • int: Números inteiros. Exemplo: 42, -10
  • float: Números de ponto flutuante. Exemplo: 3.14, -0.001
  • bool: Valores booleanos. Exemplo: true, false
  • null: Representa uma variável sem valor. Exemplo: null
  • array: Estruturas que podem conter múltiplos valores. Exemplo: [1, 2, 3]
  • object: Instância de uma classe.

Exemplo de diferentes tipos de dados em PHP:

$nome = "Maria"; // string
$idade = 30; // int
$ativo = false; // boolean
$altura = 1.68; // float
$dados = null; // null
