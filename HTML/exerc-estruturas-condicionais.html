Exercícios de Estruturas Condicionais

Básicos (Nível Fácil)

  1. Estrutura Condicional Simples: Verifique se um número é positivo e imprima "Positivo" ou "Negativo".
  2. Verificação de Maioridade: Crie uma função que receba a idade e imprima "Maior de idade" ou "Menor de idade".
  3. Classificação de Nota: Verifique se a nota do aluno é maior ou igual a 7 e imprima "Aprovado" ou "Reprovado".
  4. Verificação de Paridade: Implemente uma função que verifique se um número é par ou ímpar e imprima o resultado.
  5. Verificação de Divisibilidade: Crie um sistema que verifique se um número é divisível por 3 e imprima "Divisível por 3".
  6. Comparação de Idades: Verifique se uma pessoa é mais velha do que outra e imprima quem é mais velho.
  7. Verificação de Palíndromo: Implemente uma função que verifique se uma palavra é um palíndromo e imprima o resultado.
  8. Classificação de Idade: Crie uma função que receba a idade e imprima "Criança", "Adolescente", "Adulto" ou "Idoso".
  9. Verificação de Ano Bissexto: Implemente uma função que determine se um ano é bissexto e imprima "Bissexto" ou "Não bissexto".
  10. Verificação de Paridade de um Array: Crie uma função que verifique se todos os elementos de um array são pares e imprima "Todos são pares" ou "Nem todos são pares".
  11. Jogo de Adivinhação: Crie um jogo em que o usuário adivinhe um número gerado aleatoriamente e imprima mensagens de "Maior", "Menor" ou "Acertou!".
  12. Verificação de Categoria de Produto: Crie um sistema que classifique um produto como "Econômico", "Médio" ou "Luxo" com base no seu preço.
  13. Verificação de Número Primo: Implemente uma função que verifique se um número é primo e imprima "Primo" ou "Não primo".
  14. Verificação de Temperatura: Crie um sistema que imprima "Frio", "Agradável" ou "Quente" com base na temperatura fornecida.
  15. Verificação de Votação: Implemente uma função que verifique se a pessoa é elegível para votar (idade >= 16) e imprima "Elegível" ou "Não elegível".
  16. Classificação de Nota: Crie um sistema que imprima "Excelente", "Bom", "Regular" ou "Ruim" com base na nota do aluno (0-10).
  17. Verificação de Maior que 100: Implemente uma função que verifique se um número é maior que 100 e imprima uma mensagem correspondente.
  18. Verificação de Números Iguais: Crie um sistema que compare dois números e imprima "Igual" ou "Diferente".
  19. Verificação de Sinal de um Número: Crie uma função que verifique se um número é positivo, negativo ou zero e imprima a mensagem correspondente.
  20. Verificação de Nível de Água: Implemente uma função que verifique se o nível de água está abaixo, no nível ou acima do nível crítico e imprima a mensagem correspondente.
  21. Classificação de Temperatura com if-else: Implemente um sistema que imprima "Frio", "Morno" ou "Quente" com base na temperatura informada, usando a estrutura if-else.

Intermediário (Nível Médio)

  1. Função de Soma com Condição: Crie uma função que receba dois números como argumentos e retorne sua soma, mas apenas se a soma for maior que 10.
  2. Verificando o Maior Número em um Array: Crie uma função que receba um array de números e retorne o maior número usando uma estrutura condicional.
  3. Manipulação de Strings com Condições: Crie uma função que receba uma string e, se a string começar com uma vogal, retorne "Início com vogal"; caso contrário, retorne "Início com consoante".
  4. Desafio de Palíndromo com Condicional: Crie uma função que verifique se uma palavra é um palíndromo e retorne "É um palíndromo" ou "Não é um palíndromo".
  5. Contagem de Caracteres Condicional: Crie uma função que conte quantas vezes um determinado caractere aparece em uma string e, se for maior que 3, imprima "Muitas ocorrências"; caso contrário, imprima "Poucas ocorrências".
  6. Filtrando Números Pares: Crie uma função que receba um array e retorne um novo array apenas com os números pares, usando uma estrutura de decisão.
  7. Classificação de Faixa Etária: Crie uma função que retorne uma classificação de faixa etária (criança, adolescente, adulto, idoso) com base em uma idade fornecida.
  8. Conversão de String para Número com Validação: Crie uma função que converta uma string que representa um número em um tipo `number` e imprima uma mensagem se a conversão falhar.
  9. Função de Contagem de Palavras com Condição: Crie uma função que conte quantas palavras existem em uma frase e, se for maior que 10, imprima "Frase longa"; caso contrário, imprima "Frase curta".
  10. Classificação de Números em Ordem: Crie uma função que classifique um array de números em ordem crescente e, se o array for vazio, imprima "Array vazio".
  11. Destructuring com Condicional: Desestrutura um objeto pessoa e, se a idade for maior que 18, imprima "Maior de idade".
  12. Laço while com Condição: Use um `while` para imprimir os números de 1 a 10 e pare de imprimir se o número for 5.
  13. Função Recursiva de Fatorial com Condicional: Crie uma função recursiva para calcular o fatorial de um número e retorne "Número inválido" se o número for negativo.
  14. Uso de reduce() com Condicional: Crie uma função que receba um array de números e retorne a soma total usando `reduce()`, mas apenas se todos os números forem positivos.
  15. Verificando o Maior Número em um Array com Condicional: Crie uma função que encontre o maior número em um array e imprima "Não existe número maior" se o array for vazio.
  16. Removendo Duplicatas com Condição: Crie uma função que remova duplicatas de um array e imprima "Array com duplicatas removidas".
  17. Função de Formatação de Data com Validação: Crie uma função que receba uma data em formato YYYY-MM-DD e retorne DD/MM/YYYY. Verifique se a data é válida.
  18. Uso de Set com Condição: Crie um Set com alguns elementos e, se a quantidade de elementos únicos for maior que 5, imprima "Muitos elementos únicos".
  19. Uso de Map com Condição: Crie um Map com pares chave-valor e imprima uma mensagem se o valor de uma chave específica for maior que 10.
  20. Contagem de Caracteres com Condicional: Crie uma função que conte quantas vezes uma letra aparece em uma string e imprima "Encontrado" se for maior que 1.
  21. Uso de Array.find() com Validação: Crie uma função que encontre o primeiro número par em um array e imprima "Nenhum par encontrado" se não houver números pares.
  22. Objeto com Propriedades Dinâmicas e Condição: Crie um objeto e adicione uma nova propriedade de forma dinâmica, mas somente se a propriedade não existir.
  23. Uso de setTimeout() com Condição: Crie uma função que imprima "Olá após 3 segundos" somente se a hora atual for antes de 12h.
  24. Função para Imprimir Números Primos com Condição: Crie uma função que imprima todos os números primos de 1 a 100 e, se a quantidade de números primos for menor que 10, imprima "Poucos números primos".
  25. Desafio com for...in e Condição: Use um `for...in` para imprimir todas as chaves de um objeto e, se o valor da chave for um número, imprima "Número encontrado".
  26. Função para Verificar Anagramas com Condicional: Crie uma função que verifique se duas strings são anagramas e imprima "São anagramas" ou "Não são anagramas".
  27. Uso de Array.sort() com Condição: Crie uma função que classifique um array de números em ordem crescente e imprima "Array já ordenado" se ele já estiver em ordem.
  28. Função de Contagem de Palavras com Condicional: Crie uma função que conte quantas vezes uma palavra aparece em um texto e imprima "Mais de 5 vezes" se for maior que 5.
  29. Objetos Aninhados com Condição: Crie um objeto empresa com sub-objetos funcionários e departamentos e imprima "Tem mais de 5 funcionários" se a quantidade de funcionários for maior que 5.
  30. Função para Converter Temperatura com Validação: Crie uma função que converta Celsius para Fahrenheit e imprima "Temperatura inválida" se a entrada for menor que 0.
  31. Uso de String.split() com Condição: Crie uma função que quebre uma frase em palavras usando `split()` e, se o número de palavras for menor que 3, imprima "Frase muito curta".
  32. Função para Converter String em Array com Validação: Crie uma função que converta uma string separada por vírgulas em um array e imprima "String inválida" se a entrada não contiver vírgulas.
  33. Desafio de Programação Funcional com Condição: Crie uma função que use `map()` e `filter()` em conjunto e, se o array resultante estiver vazio, imprima "Nenhum elemento encontrado".
  34. Uso de Array.join() com Condição: Crie uma função que junte os elementos de um array em uma única string e imprima "Array vazio" se o array for vazio.
  35. Função para Substituir Texto com Condição: Crie uma função que substitua uma palavra em uma string por outra e imprima "Nenhuma substituição feita" se a palavra não for encontrada.
  36. Função para Converter String para Número com Validação: Crie uma função que converta uma string em um número com `Number()` e imprima "Conversão falhou" se o resultado for `NaN`.
  37. Função de Média de Array com Condição: Crie uma função que calcule a média dos elementos de um array e imprima "Média baixa" se a média for menor que 50.
  38. Uso de Array.every() com Condição: Crie uma função que verifique se todos os elementos de um array são positivos e imprima "Todos são positivos" ou "Há elementos negativos".
  39. Função de Contagem de Vogais com Condição: Crie uma função que conte o número de vogais em uma string e imprima "Muitas vogais" se for maior que 5.
  40. Uso de Array.includes() com Condição: Crie uma função que verifique se um array contém um elemento específico e imprima "Elemento encontrado" ou "Elemento não encontrado".
  41. Criação de Função de Callback com Condição: Crie uma função `processarDados()` que aceita uma função de callback como parâmetro e imprima "Callback não executado" se a função não for passada.
  42. Uso de Object.keys() com Condição: Crie uma função que imprima todas as chaves de um objeto e imprima "Objeto vazio" se não houver chaves.
  43. Função para Exibir Números Pares com Condição: Crie uma função que imprima todos os números pares de 1 a 100 e imprima "Nenhum par encontrado" se não houver pares.
  44. Uso de Array.reverse() com Condição: Crie uma função que inverta a ordem dos elementos de um array e imprima "Array já invertido" se estiver em ordem inversa.
  45. Função de Transformação de Texto com Validação: Crie uma função que transforme uma string em maiúsculas e imprima "Texto inválido" se a entrada estiver vazia.
  46. Uso de Object.values() com Condição: Crie uma função que imprima todos os valores de um objeto e imprima "Objeto sem valores" se não houver valores.
  47. Função para Contar Números Negativos com Condição: Crie uma função que conte quantos números negativos existem em um array e imprima "Nenhum número negativo" se for zero.
  48. Função de Conversão de Moeda com Validação: Crie uma função que converta um valor em reais para dólares e imprima "Valor inválido" se o valor for negativo.
  49. Função de Subtração de Arrays com Condição: Crie uma função que subtraia dois arrays e retorne um array com os elementos diferentes, imprimindo "Arrays iguais" se não houver diferença.
  50. Uso de Object.entries() com Condição: Crie uma função que imprima todas as chaves e valores de um objeto e imprima "Objeto vazio" se não houver entradas.
  51. Função para Verificar Se um Ano é Bissexto com Condição: Crie uma função que verifique se um ano é bissexto e imprima "Ano bissexto" ou "Não é bissexto".

Avançado (Nível Difícil)

  1. Verificação de Número Par ou Ímpar: Crie uma função que verifique se um número é par ou ímpar e retorne a mensagem correspondente.
  2. Classificação de Faixa Etária: Crie uma função que receba a idade de uma pessoa e retorne se ela é uma criança (0-12 anos), adolescente (13-17 anos), adulta (18-64 anos) ou idosa (65 anos ou mais).
  3. Verificação de Palíndromos Complexos: Crie uma função que verifique se uma string é palíndromo, ignorando espaços, pontuações e diferenças de maiúsculas/minúsculas.
  4. Desconto de Produto com Condições: Crie uma função que aplique um desconto de 10% a um produto se o preço for acima de R$ 100,00; caso contrário, retorne o preço original.
  5. Verificação de Número Primo: Crie uma função que verifique se um número é primo e retorne uma mensagem apropriada.
  6. Classificação de Nota Escolar: Crie uma função que receba uma nota de 0 a 100 e retorne a classificação do aluno: "A" (90-100), "B" (80-89), "C" (70-79), "D" (60-69), "F" (abaixo de 60).
  7. Verificação de Triângulo: Crie uma função que receba três lados e determine se eles podem formar um triângulo, usando as propriedades de desigualdade do triângulo.
  8. Cálculo de Raiz Quadrada com Condições: Crie uma função que verifique se um número é negativo e, se for, retorne uma mensagem de erro; caso contrário, calcule a raiz quadrada.
  9. Verificação de Ordem Crescente em um Array: Crie uma função que receba um array e retorne se ele está em ordem crescente.
  10. Verificação de Anagramas com Condições: Crie uma função que verifique se duas palavras são anagramas usando uma estrutura condicional para comparar as contagens de caracteres.
  11. Determinação de Categoria de Produto: Crie uma função que receba um preço e retorne "Produto barato" se for menor que R$ 50, "Produto médio" se for entre R$ 50 e R$ 150, e "Produto caro" se for maior que R$ 150.
  12. Verificação de Palavras Únicas em uma Frase: Crie uma função que receba uma frase e retorne uma lista com as palavras que aparecem apenas uma vez.
  13. Verificação de Substring em String: Crie uma função que verifique se uma substring está presente em uma string e retorne true ou false.
  14. Determinação de Nível de Acesso: Crie uma função que, com base em uma pontuação de usuário, retorne o nível de acesso ("Baixo", "Médio", "Alto").
  15. Cálculo de Maior e Menor Número em um Array: Crie uma função que encontre o maior e o menor número em um array e retorne-os.
  16. Verificação de Ordem de Números: Crie uma função que verifique se todos os números de um array estão em ordem crescente ou decrescente.
  17. Verificação de Categoria de Idade com Condições: Crie uma função que retorne a categoria de idade de uma pessoa, levando em consideração faixas como "Criança", "Adolescente", "Adulto", "Idoso".
  18. Verificação de Palindromos com Regex: Crie uma função que use expressões regulares para verificar se uma string é palíndromo, ignorando espaços e pontuações.
  19. Verificação de Paridade em Array: Crie uma função que verifique se todos os elementos de um array são pares.
  20. Cálculo de Média com Condição: Crie uma função que calcule a média de um array e retorne se ela é "Alta", "Média" ou "Baixa" com base em um limiar pré-definido.
  21. Verificação de Nível de Usuário com Condição: Crie uma função que retorne um nível de acesso ("Admin", "Editor", "Viewer") com base em uma pontuação de usuário.
  22. Verificação de Número Positivo ou Negativo: Crie uma função que verifique se um número é positivo, negativo ou zero e retorne a mensagem correspondente.
  23. Classificação de Nomes por Comprimento: Crie uma função que classifique um nome como "Curto" (menos de 5 letras), "Médio" (5-8 letras), ou "Longo" (mais de 8 letras).
  24. Verificação de Aluno Aprovado ou Reprovado: Crie uma função que verifique se um aluno foi aprovado (nota >= 6) ou reprovado (nota < 6) em uma prova.
  25. Verificação de Palíndromos com Consideração de Maiúsculas: Crie uma função que verifique se uma string é um palíndromo, levando em consideração diferenças de maiúsculas e minúsculas.
