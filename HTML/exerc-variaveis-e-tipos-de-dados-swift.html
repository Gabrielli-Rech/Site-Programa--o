Exercícios Práticos

  • Crie uma variável do tipo String chamada nome e atribua seu nome a ela. Imprima na tela.
  • Crie uma variável idade do tipo Int e atribua sua idade. Imprima a idade com uma mensagem, como "Eu tenho X anos".
  • Crie uma variável altura do tipo Double e atribua um valor de ponto flutuante. Imprima a altura com uma mensagem formatada, como "Minha altura é X metros".
  • Crie uma variável isEstudante do tipo Bool e atribua um valor true ou false. Imprima a variável com uma mensagem, como "Sou estudante?"
  • Crie um array frutas do tipo [String] com pelo menos três elementos (por exemplo: "maçã", "banana", "laranja"). Imprima todos os elementos usando um loop.
  • Crie um dicionário pessoa do tipo [String: Any] com chaves "nome" e "idade". Imprima a chave "nome".
  • Crie uma variável preco do tipo Double e atribua um valor monetário a ela. Imprima o valor formatado como moeda (por exemplo, "R$ 100,00").
  • Crie uma variável numero e use o método isMultiple(of:) para verificar se é múltiplo de 3 e imprima o resultado.
  • Crie uma variável mensagem do tipo String e use o método count para imprimir o comprimento da string.
  • Crie duas variáveis numéricas, num1 e num2, e imprima a soma, subtração, multiplicação e divisão de ambas.
  • Crie uma variável texto e use o método uppercased() para imprimir o texto em maiúsculas.
  • Crie uma variável nomeCompleto usando a interpolação de strings. Imprima o resultado com uma mensagem.
  • Crie uma variável dataAtual usando o método Date() para exibir a data atual em formato "dd/mm/aaaa".
  • Crie uma variável numeroGrande do tipo Int e imprima o número formatado com o método formatted() para separá-lo em grupos de três dígitos.
  • Crie uma variável nome e use o método split(separator:) para separar uma string em um array de palavras.
  • Crie uma variável email e use uma expressão regular para validar se é um email válido.
  • Crie uma variável idade e use uma estrutura condicional if para imprimir se a pessoa é maior de idade (18 anos ou mais).
  • Crie uma variável temperaturaCelsius e converta para Fahrenheit usando a fórmula: (C × 9/5) + 32.
  • Crie uma variável valor e use o método abs() para imprimir o valor absoluto do número.
  • Crie uma variável mensagem e use o método replacingOccurrences(of:with:) para substituir uma palavra específica por outra.
  • Crie uma variável senha do tipo String e use o método hashValue para imprimir o valor hash da senha.
  • Crie uma variável pessoa e use o método isEmpty para verificar se é um valor vazio.
  • Crie uma variável valor e use o método round() para arredondar o valor para o inteiro mais próximo.
  • Crie uma variável pessoa e use a função type(of:) para imprimir o tipo da variável.
  • Crie uma variável nome e use o método reversed() para imprimir a string ao contrário.
  • Crie uma variável valor e use a propriedade isNumeric para verificar se é um número.
  • Crie uma variável texto e use o método contains() para verificar se uma palavra está presente na string.
  • Crie uma variável quantidade e use o método min() para encontrar o menor valor em um array de números.
