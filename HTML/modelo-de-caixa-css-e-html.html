O que é o Modelo de Caixa em CSS?

O modelo de caixa em CSS é a forma como os elementos HTML são exibidos e organizados na página. Ele inclui propriedades como largura, altura, padding, border e margin para definir o tamanho e o espaçamento dos elementos.

Componentes do Modelo de Caixa

Cada elemento na página é representado como uma caixa retangular, composta pelos seguintes componentes:

  • Content: O conteúdo real do elemento, como texto, imagens, etc.
  • Padding: O espaçamento entre o conteúdo e a borda do elemento.
  • Border: A linha ao redor do elemento, que pode ter diferentes estilos, larguras e cores.
  • Margin: O espaço fora da borda do elemento, que cria espaçamento entre o elemento e os outros elementos.

Exemplo de Modelo de Caixa

Veja como a propriedade box-sizing pode influenciar o modelo de caixa:

/* Exemplo de modelo de caixa padrão (content-box) */
.box {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
    box-sizing: content-box; /* Padrão */
}

/* Exemplo de modelo de caixa alterado para border-box */
.box-border-box {
    width: 200px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
    box-sizing: border-box; /* Inclui padding e border na largura total */
}
