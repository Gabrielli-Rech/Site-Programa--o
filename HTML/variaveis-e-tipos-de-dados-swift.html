O que são Variáveis e Tipos de Dados?

Em Swift, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. Swift é uma linguagem de tipagem forte e segura, o que significa que você precisa declarar explicitamente o tipo da variável ou permitir que o compilador infira o tipo com base no valor atribuído.

Declaração de Variáveis

Em Swift, você pode declarar variáveis e constantes usando as palavras-chave var e let, respectivamente. A diferença é que as constantes não podem ser alteradas após serem definidas:

var nome = "João"  // variável (mutável)
let idade = 25      // constante (imutável)

Você também pode especificar explicitamente o tipo ao declarar uma variável:

var altura: Double = 1.75
let ativo: Bool = true

Tipos de Dados em Swift

Swift suporta diversos tipos de dados primitivos e avançados. Os tipos básicos incluem:

  • String: Cadeias de caracteres. Exemplo: "Olá, mundo!"
  • Int: Números inteiros. Exemplo: 42, -10
  • Double: Números de ponto flutuante de alta precisão. Exemplo: 3.14
  • Float: Números de ponto flutuante com menos precisão. Exemplo: 3.14
  • Bool: Valores booleanos. Exemplo: true, false
  • Character: Um único caractere. Exemplo: "A"
  • Optional: Indica que uma variável pode ou não ter valor. Exemplo: String?

Exemplo de diferentes tipos de dados em Swift:

var nome: String = "Maria"
let idade: Int = 30
var ativo: Bool = false
let altura: Double = 1.68
