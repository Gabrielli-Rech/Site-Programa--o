O que é JavaScript?

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Com JavaScript, você pode criar aplicativos web dinâmicos, desenvolver servidores usando Node.js e até programar para dispositivos móveis. É a linguagem de programação de facto para desenvolvimento web e é suportada por todos os navegadores modernos.

Principais Características do JavaScript

  • Leve e Rápido: JavaScript é executado no lado do cliente, tornando a execução rápida e eficiente.
  • Versatilidade: Pode ser usado tanto para desenvolvimento front-end quanto back-end (Node.js).
  • Orientação a Objetos: Suporta programação orientada a objetos, facilitando a criação de código modular e reutilizável.
  • Comunidade e Ecossistema: Possui uma enorme comunidade e um vasto ecossistema de bibliotecas e frameworks, como React, Angular e Vue.js.
  • Compatibilidade: Funciona em todos os navegadores modernos sem a necessidade de plugins adicionais.

Sintaxe Básica

Exemplo de Declaração de Variáveis

// Declaração de variáveis
let nome = "João"; // Variável de escopo de bloco
const idade = 30; // Variável constante
var cidade = "São Paulo"; // Variável de escopo global ou função

Exemplo de Estruturas de Controle

// Estrutura de controle condicional
if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}

// Estrutura de repetição
for (let i = 0; i < 5; i++) {
    console.log("Número: " + i);
}

Funções Avançadas

Funções de Ordem Superior

// Função de ordem superior que recebe outra função como argumento
function executarOperacao(a, b, operacao) {
    return operacao(a, b);
}

function soma(x, y) {
    return x + y;
}

console.log(executarOperacao(5, 10, soma)); // Saída: 15

Arrow Functions

// Sintaxe de arrow function
const multiplicar = (a, b) => a * b;
console.log(multiplicar(5, 4)); // Saída: 20

Manipulação de DOM

JavaScript é frequentemente usado para manipular o Document Object Model (DOM) e criar interfaces dinâmicas. Aqui está um exemplo de como modificar elementos HTML com JavaScript:

// Seleciona um elemento e altera seu conteúdo
document.getElementById("demo").innerHTML = "Olá, JavaScript!";

// Adiciona um evento de clique
document.getElementById("btn").addEventListener("click", function() {
    alert("Botão clicado!");
});

Programação Assíncrona

JavaScript tem suporte integrado para operações assíncronas, o que permite que você faça chamadas de rede e aguarde resultados sem bloquear a execução do código.

Exemplo de Promises

// Exemplo de uma Promise
const minhaPromise = new Promise((resolve, reject) => {
    let sucesso = true; // Simulação de uma operação assíncrona
    if (sucesso) {
        resolve("Operação bem-sucedida!");
    } else {
        reject("Ocorreu um erro.");
    }
});

minhaPromise
    .then(resposta => console.log(resposta))
    .catch(erro => console.error(erro));
