Exercícios de Funções e Interfaces

Exercícios Fáceis

  • Crie uma função simples que receba um número e retorne seu dobro.
  • Implemente uma função que receba uma string e retorne a mesma string em maiúsculas.
  • Crie uma função que retorne a soma de dois números.
  • Defina uma função que receba um array de números e retorne a soma de todos os elementos.
  • Crie uma função que aceite um parâmetro e retorne "Par" se for par, ou "Ímpar" se for ímpar.
  • Implemente uma função que receba um número e verifique se é positivo, negativo ou zero.
  • Crie uma função que receba um array de strings e retorne o número de elementos no array.
  • Escreva uma função que retorne o quadrado de um número.
  • Implemente uma função que receba dois argumentos, um nome e uma idade, e retorne uma string formatada.
  • Crie uma função que recebe um array de números e retorne o maior valor do array.

Exercícios Médios

  • Defina uma interface `Pessoa` com propriedades `nome` e `idade` e crie uma função que receba um objeto dessa interface e retorne "Olá, [nome]!"
  • Implemente uma função que receba uma lista de números e retorne a média dos valores.
  • Crie uma interface `Carro` com propriedades `marca`, `modelo` e `ano` e escreva uma função que aceite um objeto do tipo `Carro` e imprima suas propriedades.
  • Desenvolva uma função que receba um array de objetos `Produto` com propriedades `nome` e `preco` e retorne o total dos preços.
  • Crie uma função de ordenação que receba um array de números e o ordene em ordem crescente.
  • Defina uma interface `Endereco` e escreva uma função que imprima as informações de um endereço formatado.
  • Implemente uma função que receba um array de strings e retorne as strings que têm mais de 5 caracteres.
  • Crie uma função que receba uma interface `Animal` com propriedades `nome` e `especie` e imprima uma mensagem sobre o animal.
  • Desenvolva uma função que receba um array de números e retorne a soma dos números pares.
  • Escreva uma função que retorne uma string invertida.

Exercícios Difíceis

  • Crie uma interface `Pessoa` com propriedades obrigatórias e opcionais e escreva uma função que imprima as informações de uma pessoa.
  • Implemente uma função genérica que receba um array de qualquer tipo e retorne a quantidade de elementos únicos.
  • Desenvolva uma função que receba uma interface `Usuario` com propriedades `nome`, `email` e `telefone` e retorne true se todos os campos forem preenchidos.
  • Crie uma função que receba um array de objetos e um predicado de filtro (como uma função de callback) e retorne os elementos que atendem ao predicado.
  • Implemente uma função que combine duas interfaces em uma só e retorne um objeto com as propriedades combinadas.
  • Crie uma interface `Produto` com propriedades `id` e `descricao` e desenvolva uma função que receba um array de produtos e retorne a descrição do produto com o maior `id`.
  • Escreva uma função que utilize uma interface `Conta` e calcule o saldo médio de um array de contas.
  • Desenvolva uma função que receba uma interface `Pedido` e um array de produtos e calcule o total do pedido.
  • Crie uma função recursiva que utilize uma interface `Categoria` e retorne todas as subcategorias de uma categoria dada.
  • Implemente uma função que receba um array de objetos `Pessoa` e retorne os nomes de todas as pessoas com mais de 30 anos.

Desafios

  • Crie uma função genérica que aceite dois arrays e retorne um novo array contendo apenas os elementos que existem em ambos os arrays.
  • Implemente uma função de filtragem que utilize uma interface e uma closure para remover itens que não atendem a uma condição específica.
  • Desenvolva uma função que receba uma lista de tarefas e uma interface `Tarefa` e retorne apenas as tarefas que não estão completas.
  • Crie uma função que, usando interfaces e generics, receba uma lista de objetos e retorne o objeto que mais aparece.
  • Implemente uma função que utilize uma interface `Pessoa` e uma lista de amigos e retorne a lista de amigos de uma pessoa específica.
  • Desenvolva uma função que use uma interface `Endereco` e retorne o endereço formatado como uma string.
  • Escreva uma função que aceita uma lista de objetos `Produto` e retorne um array com as descrições dos produtos que custam mais de um certo valor.
  • Implemente uma função que receba uma interface `Aluno` e um array de notas, e calcule a média das notas, retornando "Aprovado" ou "Reprovado".
  • Crie uma função que receba um array de interfaces `Usuario` e retorne os usuários que têm idade superior a 18 anos.
  • Desenvolva uma função de mapeamento que utilize uma interface `Carro` para retornar uma lista com apenas as marcas dos carros.
