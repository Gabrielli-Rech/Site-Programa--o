Exercícios Avançados TypeScript

  • Voltar
  • Início
  • 1. Manipulação de Arrays: Crie uma função que receba um array e retorne um novo array contendo todos os elementos que são comuns entre dois arrays diferentes.
  • 2. Métodos Recursivos: Escreva uma função recursiva para calcular o fatorial de um número.
  • 3. Funções de Ordem Superior: Crie uma função que aceite outro como argumento e aplique a função a um array de números.
  • 4. Closure Capturing Values: Escreva uma função que use closures para capturar variáveis e manipular valores de fora da função.
  • 5. Classes e Objetos: Crie uma classe `Pessoa` com atributos `nome`, `idade` e `email`, e métodos para exibir a idade em dias e verificar se a pessoa é maior de idade.
  • 6. Herança: Crie uma classe `Animal` e subclasses `Cachorro` e `Gato` que sobrescrevem um método `emitirSom()`.
  • 7. Protocolos: Implemente um protocolo `Animável` e faça com que as classes `Cachorro` e `Gato` implementem este protocolo.
  • 8. Enumerações: Crie uma enum `DiaDaSemana` e escreva um método que retorne se é um dia útil ou fim de semana.
  • 9. Opções e Unwrapping: Escreva uma função que use `if let` e `guard let` para fazer unwrapping de valores opcionais.
  • 10. Extensões: Crie uma extensão para a classe `String` que adicione um método para verificar se uma string é um palíndromo.
  • 11. Protocol-Oriented Programming: Crie um protocolo `Forma` e implemente uma classe `Círculo` que conforma com esse protocolo.
  • 12. Destructuring: Desconstrua um array em variáveis individuais usando `let (primeiro, segundo, terceiro) = array`.
  • 13. Generics: Crie uma função genérica que receba um array de qualquer tipo e retorne o primeiro elemento.
  • 14. Closure como Argumento: Escreva uma função que receba uma closure como argumento e a execute dentro da função.
  • 15. Manipulação de Dicionários: Crie uma função que receba um dicionário e retorne apenas as chaves cujos valores sejam maiores que 10.
  • 16. Tipos Customizados: Defina um `struct` chamado `Pessoa` com propriedades `nome` e `idade` e um método que exiba uma mensagem de boas-vindas.
  • 17. Operadores Customizados: Crie um operador sobrecarregado para somar duas instâncias de uma classe `Pessoa` com base em uma propriedade específica.
  • 18. Trabalhando com Arquivos: Escreva um método que leia e escreva em um arquivo usando `FileManager`.
  • 19. Geradores de Senhas: Crie uma função que gere uma senha aleatória com letras, números e caracteres especiais.
  • 20. Manipulação de Strings: Crie um método que converta um texto em `snake_case` a partir de `CamelCase`.
  • 21. Tratamento de Erros: Crie uma função que lança um erro se uma condição não for atendida e trate esse erro com um `do-catch`.
  • 22. Type Casting: Escreva um método que use `as?` para verificar se um objeto pode ser convertido para um tipo específico.
  • 23. Protocol Extension: Adicione uma implementação padrão de um método a um protocolo `Animável` usando uma extensão.
  • 24. Simulação de Banco de Dados: Crie um `struct` que simule uma tabela com operações CRUD em um array de objetos.
  • 25. Desafio de Recursão: Escreva uma função recursiva para encontrar o maior elemento em um array.
  • 26. Métodos de Instância: Escreva uma função que altere uma propriedade em uma instância de uma classe e retorne o novo valor.
  • 27. Protocol com Associated Type: Crie um protocolo `Container` com um tipo associado e implemente uma classe `Cesta` que conforme com ele.
  • 28. Gerenciamento de Memória: Explique e implemente o uso de `weak` e `unowned` em referências de classe para evitar ciclos de referência.
  • 29. Operações Assíncronas: Use `async` e `await` para criar uma função assíncrona que retorne um valor após um atraso.
  • 30. Simulação de Eventos: Crie uma classe `Botao` que notifique quando um botão for clicado usando closures.
  • 31. Métodos Estáticos: Crie uma classe com métodos de classe estáticos para realizar cálculos matemáticos como soma e multiplicação.
  • 32. Unindo Arrays: Escreva uma função que una dois arrays de forma alternada (intercalando elementos de ambos).
  • 33. Integração com JSON: Crie uma função que converta um JSON em um objeto Swift usando `Codable`.
  • 34. Observer Pattern: Implemente o padrão de projeto Observer para notificar mudanças em um objeto.
  • 35. Manipulação de Data e Hora: Crie uma função que calcule a diferença entre duas datas em dias.
  • 36. Repetição com `while`: Crie um método que use um loop `while` para contar de 1 a 10.
  • 37. Testes Unitários: Escreva testes unitários para uma função que calcula o desconto em uma compra.
  • 38. Customizando Objetos: Use `CustomStringConvertible` para criar uma representação em string personalizada de um objeto.
  • 39. Programação Declarativa: Use `map`, `filter` e `reduce` para manipular arrays de forma funcional.
  • 40. Troca de Valores: Escreva uma função que receba duas variáveis e troque seus valores sem usar uma variável auxiliar.
  • 41. Protótipos de Teste: Crie um protótipo de uma classe que simule operações de um serviço web.
  • 42. Tópicos de Concorrência: Implemente um método que execute várias tarefas em paralelo usando `DispatchQueue`.
  • 43. Conectando a APIs: Crie uma função que faça uma requisição HTTP GET usando `URLSession` e processe a resposta.
  • 44. Desafios de SwiftUI: Implemente uma interface básica em SwiftUI que exiba uma lista de itens.
  • 45. Interoperabilidade com Objective-C: Escreva uma classe em Swift que use métodos de uma classe escrita em Objective-C.
  • 46. Adaptadores de Protocólo: Crie um adaptador de protocolo para modificar como métodos de um protocolo são usados.
  • 47. Nomenclatura Avançada: Use `Typealiases` para criar abreviações para tipos complexos de dados.
  • 48. Funções Variádicas: Crie uma função que aceite um número variável de argumentos e retorne a soma de todos eles.
  • 49. Encadeamento de Funções: Crie um método que encadeie a execução de duas ou mais funções para um processamento de dados eficiente.
  • 50. Conclusão de Desafio: Combine vários conceitos de Swift para desenvolver um mini-jogo que conte a quantidade de vezes que uma palavra aparece em um texto.
