O que são Laços de Repetição?

Os laços de repetição em JavaScript são usados para executar um bloco de código várias vezes. Eles são fundamentais para automatizar tarefas e iterar sobre coleções de dados.

Principais Tipos de Laços de Repetição

Os laços de repetição mais comuns em JavaScript são:

  • for: Usado para repetir um bloco de código um número específico de vezes.
  • while: Executa um bloco de código enquanto uma condição for verdadeira.
  • do...while: Executa um bloco de código pelo menos uma vez e continua enquanto a condição for verdadeira.

Exemplo de um laço for:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Exemplo de um laço while:

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Exemplo de um laço do...while:

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);
