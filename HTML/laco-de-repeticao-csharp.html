O que são Laços de Repetição?

Laços de repetição são estruturas de controle de fluxo usadas para executar um bloco de código repetidamente enquanto uma condição for verdadeira ou por um número específico de vezes. Eles são essenciais para automatizar tarefas repetitivas em programas.

Laço de Repetição for

O laço for é usado quando você sabe quantas vezes um bloco de código deve ser executado. Ele é composto por três partes: inicialização, condição e incremento/atualização.

for (int i = 0; i < 10; i++)
{
    // código a ser executado
}

Exemplo:

for (int i = 0; i < 5; i++)
{
    Console.WriteLine("Iteração número " + i);
}

Laço de Repetição while

O laço while é usado quando você não sabe quantas vezes o bloco de código será executado. Ele continua a executar enquanto a condição for verdadeira.

int i = 0;
while (i < 5)
{
    // código a ser executado
    i++;
}

Exemplo:

int i = 0;
while (i < 3)
{
    Console.WriteLine("Iteração número " + i);
    i++;
}

Laço de Repetição do-while

O laço do-while é similar ao while, mas garante que o bloco de código seja executado pelo menos uma vez antes de verificar a condição.

int i = 0;
do
{
    // código a ser executado
    i++;
} while (i < 5);

Exemplo:

int i = 0;
do
{
    Console.WriteLine("Iteração número " + i);
    i++;
} while (i < 3);

Laço de Repetição foreach

O laço foreach é usado para iterar sobre coleções, como arrays ou listas, sem a necessidade de um índice. Ele é ideal para percorrer elementos de uma coleção.

foreach (var item in colecao)
{
    // código a ser executado para cada item
}

Exemplo:

string[] frutas = { "maçã", "banana", "cereja" };
foreach (var fruta in frutas)
{
    Console.WriteLine("Fruta: " + fruta);
}
