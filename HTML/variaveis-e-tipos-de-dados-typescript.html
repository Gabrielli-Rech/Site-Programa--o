O que são Variáveis e Tipos de Dados?

Em TypeScript, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. TypeScript é uma linguagem com tipagem estática, o que significa que você pode (e deve) declarar o tipo das variáveis para evitar erros no código.

Declaração de Variáveis

Em TypeScript, você pode declarar variáveis usando as palavras-chave let, const e var. Recomenda-se usar let para variáveis mutáveis e const para constantes. Os tipos podem ser declarados explicitamente:

let nome: string = "João"; // variável (mutável)
const idade: number = 25;    // constante (imutável)

Você também pode omitir o tipo e deixar que TypeScript o infira com base no valor atribuído:

let ativo = true; // TypeScript infere como boolean

Tipos de Dados em TypeScript

TypeScript suporta uma ampla gama de tipos de dados para melhor controle e segurança do código. Os principais tipos incluem:

  • string: Cadeias de caracteres. Exemplo: "Olá, mundo!"
  • number: Números inteiros e de ponto flutuante. Exemplo: 42, 3.14
  • boolean: Valores booleanos. Exemplo: true, false
  • any: Permite armazenar qualquer tipo de dado. Exemplo: let variavel: any = "texto";
  • null e undefined: Representam a ausência de valor.
  • array: Conjunto ordenado de elementos. Exemplo: let numeros: number[] = [1, 2, 3];
  • tuple: Lista de elementos de tipos fixos. Exemplo: let tupla: [string, number] = ["texto", 42];
  • enum: Conjunto de valores nomeados. Exemplo: enum Cores { Vermelho, Verde, Azul }
  • void: Usado para funções que não retornam valor.

Exemplo de diferentes tipos de dados em TypeScript:

let nome: string = "Maria";
let idade: number = 30;
let ativo: boolean = false;
let opcional: string | null = null; // tipo opcional
