O que é Docker?

Docker é uma plataforma de contêinerização que permite criar, testar e implantar aplicações de forma isolada em contêineres. Isso garante consistência entre ambientes de desenvolvimento e produção.

Comandos Básicos do Docker

Comandos essenciais para começar a usar o Docker:

  • docker pull [imagem] – Baixa uma imagem do Docker Hub.
  • docker run -d -p 80:80 [imagem] – Executa um contêiner em segundo plano e mapeia portas.
  • docker ps – Lista os contêineres em execução.
  • docker stop [ID do contêiner] – Para um contêiner em execução.
  • docker rm [ID do contêiner] – Remove um contêiner parado.
