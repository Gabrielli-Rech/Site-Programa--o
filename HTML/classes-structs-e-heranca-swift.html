Exercícios de Classes, Structs e Herança

Fáceis

  • 1. Crie uma struct `Pessoa` com propriedades `nome` e `idade`. Adicione um método para exibir essas informações.
  • 2. Crie uma classe `Carro` com propriedades `marca`, `modelo` e `ano`. Adicione um método para exibir os detalhes do carro.
  • 3. Implemente uma struct `Retangulo` com propriedades `largura` e `altura`. Adicione um método para calcular a área.
  • 4. Crie uma classe `Animal` com uma propriedade `nome` e um método para emitir som.
  • 5. Crie uma struct `Círculo` com uma propriedade `raio` e um método para calcular a circunferência.
  • 6. Crie uma classe `Pessoa` com uma propriedade `endereco`. Adicione um método para atualizar o endereço.
  • 7. Crie uma struct `Produto` com propriedades `nome` e `preco`. Adicione um método para aplicar um desconto.
  • 8. Implemente uma classe `Aluno` com propriedades `nome` e `nota`. Adicione um método para verificar aprovação.
  • 9. Crie uma struct `Ponto` com coordenadas `x` e `y`. Adicione um método para calcular a distância de outro ponto.
  • 10. Desenvolva uma classe `Livro` com propriedades `titulo` e `autor`. Adicione um método para exibir as informações.
  • 11. Crie uma struct `Triangulo` com propriedades `base` e `altura`. Adicione um método para calcular a área.
  • 12. Crie uma classe `Pessoa` com um método que retorne uma saudação personalizada usando o nome da pessoa.
  • 13. Crie uma struct `ContaBancaria` com propriedades `saldo` e `numero`. Adicione um método para depositar dinheiro.
  • 14. Desenvolva uma classe `Aluno` com propriedades `matricula` e `curso`. Adicione um método para atualizar o curso.
  • 15. Implemente uma struct `Temperatura` com uma propriedade em Celsius e um método para converter para Fahrenheit.
  • 16. Crie uma classe `Cidade` com propriedades `nome` e `populacao`. Adicione um método para exibir a densidade demográfica.
  • 17. Crie uma struct `ContaCorrente` com uma propriedade `saldo`. Adicione um método para verificar se há fundos suficientes para um saque.
  • 18. Crie uma classe `Endereco` com propriedades `rua` e `numero`. Adicione um método para exibir o endereço completo.
  • 19. Desenvolva uma struct `Retangulo` com métodos para calcular área e perímetro.
  • 20. Crie uma classe `Produto` com propriedades `nome` e `estoque`. Adicione um método para reduzir o estoque ao vender.

Médios

  • 1. Crie uma classe `Veiculo` com propriedades como `marca` e `modelo`. Herde essa classe em `Carro` e `Moto`.
  • 2. Crie uma struct `Pessoa` que armazene informações pessoais e inclua um método para calcular a idade com base na data de nascimento.
  • 3. Crie uma classe abstrata `Forma` com métodos para calcular área e perímetro. Implemente essas funções em subclasses `Retangulo` e `Circulo`.
  • 4. Desenvolva uma classe `Funcionario` com propriedades `nome` e `salario`. Crie subclasses `Gerente` e `Estagiario` com métodos específicos.
  • 5. Implemente uma struct `Data` com validação para o formato correto de dia, mês e ano.
  • 6. Crie uma hierarquia de classes para um sistema escolar, incluindo classes `Professor` e `Estudante`.
  • 7. Desenvolva uma classe `Veiculo` com um método para calcular o consumo de combustível baseado na distância.
  • 8. Implemente uma struct `Endereco` com validação para o formato de CEP.
  • 9. Crie uma classe `Loja` que gerencie múltiplos objetos de uma struct `Produto`.
  • 10. Implemente uma classe `Pedido` que contenha uma lista de produtos e calcule o total do pedido.
  • 11. Desenvolva uma struct `Horario` que valide entradas como hora e minuto e permita conversão para formato de 12 horas.
  • 12. Crie uma classe `Funcionario` com métodos para calcular aumento de salário com base no tempo de serviço.
  • 13. Implemente uma classe `ContaBancaria` com métodos para transferência de valores entre contas.
  • 14. Crie uma struct `Livro` com métodos para verificar disponibilidade em uma biblioteca.
  • 15. Desenvolva uma classe `Produto` que inclua lógica para aplicar múltiplos tipos de desconto.
  • 16. Crie uma classe `Animal` com um método para emitir som e herde em classes como `Cachorro` e `Gato`.
  • 17. Implemente uma struct `Veiculo` com métodos para verificar a capacidade de carga.
  • 18. Crie uma classe `Turma` com uma lista de estudantes e métodos para calcular a média da turma.
  • 19. Desenvolva uma hierarquia de classes para um sistema de transporte com classes `Trem` e `Metro`.
  • 20. Implemente uma classe `Evento` com validação de datas e métodos para exibir o tempo restante.

Difíceis

  • 1. Desenvolva um sistema de contas bancárias com classes `Conta`, `ContaCorrente` e `ContaPoupanca`, implementando regras de saque e transferência.
  • 2. Implemente uma classe abstrata `Animal` com métodos específicos e herde em classes como `Ave` e `Mamifero`.
  • 3. Crie uma classe `Funcionario` com um método para calcular bônus baseado no desempenho.
  • 4. Implemente uma struct `Agenda` que permita adicionar, editar e remover compromissos.
  • 5. Desenvolva uma hierarquia de classes para um sistema de transporte público, incluindo classes `Passagem` e `Veiculo`.

Desafios

  • 1. Desenvolva um sistema de inventário com classes que suportem polimorfismo e sobrescrita de métodos.
  • 2. Implemente um gerenciador de tarefas com structs e classes, incluindo validação de prioridade e data de vencimento.
  • 3. Crie um sistema de autenticação com classes `Usuario` e `Permissao`, usando herança e encapsulamento.
  • 4. Desenvolva um simulador de loja virtual com classes `Produto`, `Carrinho` e `Pedido`.
