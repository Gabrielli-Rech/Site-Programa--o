Exercícios de APIs.

  • Voltar
  • Início
  • Crie uma função em PHP que consome uma API REST e exibe os resultados no navegador.
  • Implemente um sistema que consome uma API pública (ex: API de clima) e mostra as informações do tempo em uma página PHP.
  • Crie uma API RESTful em PHP que retorna uma lista de produtos em formato JSON.
  • Desenvolva uma API em PHP que aceite parâmetros de consulta e retorne dados filtrados em JSON.
  • Consuma uma API externa usando a biblioteca `cURL` em PHP e exiba a resposta no navegador.
  • Implemente autenticação usando tokens JWT em uma API PHP e proteja o acesso a rotas específicas.
  • Crie uma API em PHP que permita realizar operações CRUD em uma lista de tarefas (Create, Read, Update, Delete).
  • Desenvolva uma API que aceite dados em formato XML e retorne a resposta em JSON.
  • Crie um cliente em PHP que consome a API de um serviço de terceiros e processa a resposta com `json_decode()`.
  • Implemente uma API em PHP que permita upload de arquivos e retorne a URL do arquivo salvo.
  • Crie uma página PHP que consome uma API de autenticação e exibe uma mensagem de boas-vindas com base nos dados do usuário.
  • Desenvolva uma API em PHP que receba dados em JSON e armazene em um banco de dados MySQL.
  • Crie uma função PHP que use a biblioteca `file_get_contents()` para consumir uma API REST e exibir os resultados.
  • Implemente uma página PHP que exiba uma lista de usuários de uma API pública com paginação.
  • Crie um script PHP que consome uma API REST com autenticação básica e exibe a resposta de forma estruturada.
  • Desenvolva uma API em PHP que permita a busca de informações em um banco de dados usando filtros de consulta.
  • Implemente um serviço PHP que consome uma API para verificar se um endereço de email é válido.
  • Crie um script PHP que consome a API de uma plataforma de pagamento e processa uma transação de teste.
  • Desenvolva um sistema em PHP que utilize a API de um serviço de mapas para calcular a distância entre dois pontos.
  • Crie uma API em PHP que autentica usuários com base em um banco de dados de credenciais.
  • Implemente uma função PHP que faça uma solicitação POST para uma API e manipule os dados de resposta.

Exercícios de Requisitos e Validação PHP

  • Crie uma função em PHP que valide um endereço de email e retorne `true` ou `false`.
  • Desenvolva uma função PHP que verifique se um número é positivo, negativo ou zero.
  • Implemente uma função PHP que valide se um número é um CPF válido (considerando as regras de cálculo).
  • Crie um script PHP que valide a entrada de um formulário, verificando se todos os campos obrigatórios estão preenchidos.
  • Desenvolva uma função em PHP que valide se um campo de senha atende a requisitos de segurança (mínimo de 8 caracteres, pelo menos uma letra maiúscula, um número e um símbolo).
  • Implemente uma função PHP que verifique se uma URL é válida e retorne um valor booleano.
  • Crie uma função PHP que valide se um nome de usuário contém apenas caracteres alfanuméricos e tenha entre 3 e 15 caracteres.
  • Desenvolva uma função PHP que valide o formato de uma data no padrão `YYYY-MM-DD`.
  • Implemente uma função que receba um arquivo de imagem e valide seu tipo MIME (ex: `image/png`, `image/jpeg`).
  • Crie uma função PHP que verifique se uma string contém apenas caracteres ASCII válidos.
  • Desenvolva uma função PHP que valide se um número de telefone possui o formato correto, por exemplo, `+55 11 98765-4321`.
  • Implemente uma função PHP que verifique se um arquivo existe e se o usuário tem permissão de leitura.
  • Crie uma função PHP que receba um endereço e valide seu formato (ex: `Rua X, 123, Bairro Y, Cidade Z`).
  • Desenvolva uma função PHP que valide um código postal (CEP) brasileiro.
  • Implemente uma função PHP que verifique se uma string é um número válido de cartão de crédito com base no algoritmo de Luhn.
  • Crie uma função PHP que verifique se um arquivo de texto contém caracteres inválidos para o sistema de arquivos.
  • Desenvolva uma função PHP que receba uma string e retire espaços em branco do início e do fim, validando se a string resultante está vazia.
  • Implemente uma função PHP que valide se um endereço de site (URL) tem um protocolo válido (HTTP/HTTPS).
  • Crie uma função PHP que valide se a extensão de um arquivo é permitida, como `.jpg`, `.png`, `.txt`.
  • Desenvolva uma função PHP que verifique se um número de identificação (ID) é único em uma lista de IDs.
  • Implemente uma função PHP que valide se um campo de data de nascimento não é posterior à data atual.
