O que é o Visual Studio Code?

O Visual Studio Code (VSCode) é um editor de código-fonte leve, gratuito e altamente configurável. Ele suporta diversas linguagens de programação e é amplamente usado por desenvolvedores.

A seguir, listamos os principais comandos e atalhos para otimizar sua produtividade no VSCode.

Comandos Básicos

  • Abrir Paleta de Comandos: 
    Ctrl + Shift + P (ou F1)
  • Abrir Arquivo: 
    Ctrl + O
  • Salvar Arquivo: 
    Ctrl + S
  • Salvar Todos os Arquivos: 
    Ctrl + K + S
  • Fechar Editor Atual: 
    Ctrl + W
  • Fechar Janela: 
    Ctrl + Shift + W

Navegação

  • Ir para Linha: 
    Ctrl + G
  • Alternar entre Arquivos Abertos: 
    Ctrl + Tab
  • Pesquisar no Arquivo: 
    Ctrl + F
  • Pesquisar em Todos os Arquivos: 
    Ctrl + Shift + F
  • Abrir a Barra Lateral: 
    Ctrl + B

Edição de Código

  • Copiar Linha Atual: 
    Ctrl + C (sem seleção)
  • Excluir Linha Atual: 
    Ctrl + Shift + K
  • Duplicar Linha: 
    Alt + Shift + Down
  • Comentar Linha: 
    Ctrl + /
  • Indentar Linha: 
    Tab
  • Desfazer Indentação: 
    Shift + Tab

Gerenciamento de Terminais

  • Abrir Terminal: 
    Ctrl + `
  • Criar Novo Terminal: 
    Ctrl + Shift + `
  • Alternar entre Terminais: 
    Ctrl + PageUp/PageDown
  • Fechar Terminal: 
    Ctrl + Shift + W
