O que são Bases e Tabelas no MySQL?

No MySQL, uma base de dados (ou banco de dados) é um contêiner que armazena tabelas e outros objetos relacionados. Tabelas são a estrutura onde os dados são armazenados em linhas e colunas, semelhantes a uma planilha.

Criação de Bases

Para criar uma base de dados no MySQL, você utiliza o comando CREATE DATABASE seguido do nome da base.

CREATE DATABASE nome_da_base;

Exemplo:

CREATE DATABASE escola;

Para selecionar a base de dados que você deseja usar, use o comando USE:

USE escola;

Criação de Tabelas

Para criar uma tabela em uma base de dados, você utiliza o comando CREATE TABLE seguido do nome da tabela e a definição das colunas com seus tipos de dados e restrições.

CREATE TABLE nome_da_tabela (
    coluna1 tipo_de_dado restricoes,
    coluna2 tipo_de_dado restricoes,
    ...
);

Exemplo de criação de uma tabela chamada "alunos":

CREATE TABLE alunos (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    idade INT,
    curso VARCHAR(50)
);

Tipos de Dados em Tabelas

Os tipos de dados mais comuns para colunas de tabelas incluem:

  • INT: Números inteiros.
  • VARCHAR(n): Strings de comprimento variável (máximo de n caracteres).
  • TEXT: Texto de comprimento variável.
  • DATE: Datas.
  • DECIMAL(p, s): Números decimais com precisão p e escala s.
  • BOOLEAN: Valores verdadeiro ou falso.

Exemplo de tabela com diferentes tipos de dados:

CREATE TABLE funcionarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL,
    salario DECIMAL(10, 2),
    data_admissao DATE,
    ativo BOOLEAN
);

Alteração de Tabelas

Para modificar uma tabela existente, use o comando ALTER TABLE.

ALTER TABLE nome_da_tabela ADD coluna tipo_de_dado;

Exemplo de adição de uma coluna:

ALTER TABLE alunos ADD email VARCHAR(150);

Para remover uma coluna:

ALTER TABLE alunos DROP COLUMN email;

Exclusão de Tabelas e Bases

Para excluir uma tabela, use o comando DROP TABLE.

DROP TABLE nome_da_tabela;

Para excluir uma base de dados, use o comando DROP DATABASE.

DROP DATABASE nome_da_base;
