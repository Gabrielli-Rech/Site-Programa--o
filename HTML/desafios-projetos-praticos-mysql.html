Desafios Fáceis

  • 1. Conectar a um banco de dados MySQL local e listar todas as tabelas existentes.
  • 2. Criar uma tabela simples para armazenar informações de clientes.
  • 3. Inserir dados em uma tabela usando o comando INSERT.
  • 4. Consultar todos os registros de uma tabela usando SELECT.
  • 5. Atualizar informações de um registro específico usando UPDATE.
  • 6. Deletar um registro de uma tabela usando DELETE.
  • 7. Usar WHERE para filtrar dados em uma consulta simples.
  • 8. Criar uma consulta que ordena registros com ORDER BY.
  • 9. Contar o número de registros em uma tabela com COUNT().
  • 10. Usar o comando LIMIT para retornar um número específico de registros.

Desafios Médios

  • 11. Criar uma relação entre duas tabelas usando chaves estrangeiras.
  • 12. Realizar uma consulta JOIN para combinar dados de duas tabelas.
  • 13. Usar funções de agregação como SUM(), AVG() e MAX() para gerar relatórios.
  • 14. Criar um índice em uma tabela para melhorar o desempenho de consultas.
  • 15. Criar uma tabela com um esquema que inclui tipos de dados DATE e TIME.
  • 16. Usar GROUP BY para agrupar dados e calcular somas totais.
  • 17. Implementar uma consulta com subconsultas usando IN ou EXISTS.
  • 18. Criar uma consulta que utilize CASE para gerar colunas calculadas.
  • 19. Criar uma tabela temporária para armazenar dados intermediários.
  • 20. Realizar uma consulta para calcular a diferença entre datas com DATEDIFF().

Desafios Difíceis

  • 21. Criar um procedimento armazenado que executa uma série de operações em tabelas.
  • 22. Escrever um gatilho (trigger) para atualizar automaticamente uma tabela após uma operação INSERT.
  • 23. Criar uma função armazenada que retorna um valor calculado com base nos parâmetros de entrada.
  • 24. Realizar uma consulta recursiva usando o comando WITH RECURSIVE.
  • 25. Implementar a normalização de um banco de dados em até a terceira forma normal (3NF).
  • 26. Configurar replicação de dados entre dois servidores MySQL.
  • 27. Criar uma consulta que inclua junções complexas com múltiplas tabelas.
  • 28. Escrever um script para importar dados em massa de um arquivo CSV para uma tabela.
  • 29. Implementar uma consulta que gere um relatório com agrupamentos por intervalos de tempo.
  • 30. Criar uma estratégia de backup automático com scripts SQL.

Desafios Avançados

  • 31. Criar um banco de dados particionado para gerenciar grandes volumes de dados.
  • 32. Implementar uma estratégia de controle de acesso usando privilégios de usuário.
  • 33. Escrever um script para monitorar o desempenho de consultas usando o EXPLAIN.
  • 34. Criar um índice composto em uma tabela e explicar seu impacto na performance.
  • 35. Configurar e otimizar um banco de dados MySQL para operações de alta concorrência.
  • 36. Desenvolver um sistema de auditoria que registre alterações em tabelas em tempo real.
  • 37. Implementar uma consulta que utilize funções de janela como ROW_NUMBER().
  • 38. Criar um sistema de armazenamento e consulta de dados JSON em uma tabela MySQL.
  • 39. Configurar e executar um cluster MySQL para alta disponibilidade.
  • 40. Realizar tuning de uma consulta lenta e medir a diferença de desempenho.
