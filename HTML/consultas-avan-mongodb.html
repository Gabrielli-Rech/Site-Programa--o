Exercícios de Consultas Avançadas em MongoDB

Fáceis

  • 1. Liste todos os documentos de uma coleção usando o método `find()`.
  • 2. Realize uma consulta que retorne documentos de uma coleção com um campo específico igual a um valor.
  • 3. Use o operador `$gt` para encontrar documentos onde um campo numérico é maior que um valor específico.
  • 4. Utilize o operador `$lt` para consultar documentos com valores numéricos menores que um valor específico.
  • 5. Realize uma consulta que busque documentos com um campo específico nulo.
  • 6. Use a projeção para exibir apenas campos selecionados em um documento.
  • 7. Use o operador `$or` para buscar documentos que satisfaçam pelo menos uma das condições em um array.
  • 8. Realize uma consulta que retorne documentos em ordem crescente de um campo.
  • 9. Faça uma consulta com ordenação decrescente de um campo.
  • 10. Use o operador `$in` para buscar documentos onde um campo tem um dos vários valores possíveis.
  • 11. Realize uma consulta para contar o número de documentos em uma coleção.
  • 12. Filtre documentos usando expressões regulares com `$regex` para buscar padrões de texto.
  • 13. Realize uma consulta com `skip()` e `limit()` para retornar apenas um número específico de documentos.
  • 14. Use o operador `$exists` para encontrar documentos onde um campo específico existe ou não existe.
  • 15. Realize uma consulta usando `$and` para combinar múltiplas condições.
  • 16. Encontre documentos que contenham um array específico e use o operador `$size` para filtrar pelo número de elementos.
  • 17. Faça uma consulta que retorne documentos em uma faixa de datas usando o operador `$gte` e `$lte`.
  • 18. Use o operador `$elemMatch` para filtrar documentos em que um array contenha um subdocumento que satisfaça condições específicas.
  • 19. Faça uma consulta para atualizar um documento com o método `updateOne()`.
  • 20. Realize uma consulta para remover um documento usando o método `deleteOne()`.

Médios

  • 1. Use a agregação com `$match` e `$group` para contar a quantidade de documentos por um campo específico.
  • 2. Realize uma consulta que combine múltiplos operadores lógicos `$and`, `$or`, e `$not`.
  • 3. Use a operação de agregação `$project` para renomear e modificar campos.
  • 4. Realize uma consulta com `aggregate()` para ordenar e agrupar documentos ao mesmo tempo.
  • 5. Crie uma consulta de agregação que combine dados de duas coleções usando `$lookup`.
  • 6. Use o operador `$unwind` em uma consulta de agregação para desmembrar arrays em subdocumentos individuais.
  • 7. Utilize `$addFields` para adicionar novos campos calculados em um documento.
  • 8. Realize uma consulta de agregação para filtrar documentos em uma faixa de tempo usando `$match` e `$gte`.
  • 9. Use `$sort` em uma agregação para ordenar documentos de acordo com múltiplos critérios.
  • 10. Crie uma consulta para remover todos os documentos de uma coleção que não satisfaçam uma condição com `deleteMany()`.
  • 11. Utilize a função `countDocuments()` para contar documentos de uma coleção que atendam a critérios específicos.
  • 12. Implemente uma consulta com `$lookup` para fazer join entre duas coleções em MongoDB.
  • 13. Use `$match` e `$project` juntos para criar uma consulta complexa de filtragem e seleção de campos.
  • 14. Realize uma consulta com uma subconsulta que retorna documentos com um valor mínimo de campo.
  • 15. Utilize `$out` em uma agregação para exportar resultados para uma nova coleção.
  • 16. Realize uma consulta para buscar documentos onde um campo seja um array de valores distintos.
  • 17. Use `$map` e `$reduce` para transformar e combinar dados de arrays dentro de uma consulta de agregação.
  • 18. Crie uma consulta com `$merge` para armazenar resultados de uma agregação em uma coleção existente.
  • 19. Use a agregação `$graphLookup` para buscar dados de forma recursiva em uma coleção.
  • 20. Realize uma consulta com `$arrayElemAt` para acessar um elemento específico de um array.

Difíceis

  • 1. Implemente uma consulta de agregação para calcular a média de valores em um campo específico de uma coleção.
  • 2. Realize uma consulta complexa de agregação usando `$bucket` para agrupar documentos em intervalos de valores.
  • 3. Use `$graphLookup` para implementar uma consulta que percorre e retorna uma árvore de documentos.
  • 4. Crie uma consulta de agregação com `$sort` e `$limit` para encontrar os documentos mais recentes ou mais antigos em uma coleção.
  • 5. Implemente uma consulta com `$map` para aplicar transformações a elementos de um array dentro de uma agregação.
  • 6. Use a consulta de agregação para encontrar documentos com a menor e maior quantidade de valores em um campo numérico.
  • 7. Crie uma consulta com `$project` e `$arrayElemAt` para acessar e modificar elementos de um array aninhado.
  • 8. Realize uma consulta com `$match` e `$group` que calcule a soma dos valores de um campo em grupos.
  • 9. Implemente uma consulta de agregação usando `$lookup` e `$unwind` para fazer join entre duas coleções e expandir subdocumentos.
  • 10. Use o operador `$text` para realizar uma busca de texto completo em uma coleção e filtrar resultados por relevância.

Desafios

  • 1. Crie uma consulta de agregação que faça análise de sentimentos em campos de texto usando `$project` e expressões regulares.
  • 2. Implemente uma consulta que encontre todos os documentos com subdocumentos que atendem a uma condição específica em arrays aninhados.
  • 3. Realize uma consulta para calcular o tempo médio de resposta de uma API com dados armazenados em uma coleção.
  • 4. Desenvolva uma consulta complexa que agrupe documentos em intervalos de tempo e calcule estatísticas de cada grupo.
  • 5. Crie uma consulta que combine operações de `$lookup`, `$unwind` e `$group` para fazer relatórios detalhados de vendas.
