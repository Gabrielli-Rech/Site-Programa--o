Exercícios de Snippets e Personalização do VSCode

  • Crie um snippet personalizado em `settings.json` para gerar um bloco de código de função em JavaScript.
  • Personalize o tema do VSCode usando a opção de temas integrados ou criando um tema personalizado em `settings.json`.
  • Explore a criação de snippets com variáveis de espaço reservado, como `$1`, `$2`, para gerar templates mais interativos.
  • Crie um snippet para uma estrutura de HTML5 e use o atalho de teclado para inseri-lo em um arquivo HTML.
  • Edite o `settings.json` para alterar a formatação automática de código ao salvar um arquivo.
  • Adicione extensões como `Prettier` e `ESLint` e configure-os para serem executados automaticamente ao salvar o arquivo.
  • Crie um snippet que insere um loop `for` em Python com a variável de iteração e uma estrutura de código base.
  • Personalize as teclas de atalho para um comando específico, como `Ctrl+Shift+P`, para ajustar a formatação de código.
  • Configure o VSCode para utilizar uma fonte personalizada que melhore a legibilidade do código.
  • Crie um snippet de código CSS para uma classe de botão com propriedades básicas como `background-color`, `border`, e `padding`.
  • Explore e instale extensões de snippets que podem ser usadas para facilitar a escrita de código em linguagens como JavaScript, Python e PHP.
  • Configure o VSCode para que as linhas de código com mais de 80 caracteres sejam destacadas com uma cor específica.
  • Personalize a barra lateral do VSCode, removendo ou adicionando ícones e atalhos para otimizar o fluxo de trabalho.
  • Crie um snippet para uma estrutura de função em TypeScript com tipos de parâmetros e retorno.
  • Adicione um snippet para um comentário de documentação em JavaScript com os campos `@param`, `@return`, e `@example`.
  • Utilize variáveis no `settings.json` para automatizar a substituição de valores como o nome do projeto e o nome do autor nos snippets.
  • Configure a barra de status do VSCode para mostrar informações como o tempo de execução do código ou o número de erros no projeto.
  • Crie um snippet para um arquivo `Dockerfile` com comandos comuns como `FROM`, `RUN`, `COPY`, e `EXPOSE`.
  • Explore o uso de multi-cursors para aplicar alterações em várias linhas de código ao mesmo tempo e como isso pode ser personalizado.
  • Configure a paleta de cores do VSCode para um fundo claro ou escuro, de acordo com suas preferências de trabalho.
  • Crie um snippet para uma função de API REST em Node.js com parâmetros de requisição e resposta.
  • Teste a funcionalidade de `Live Server` e configure-o para iniciar automaticamente quando você salvar um arquivo HTML.
  • Explore e configure a opção de "Code Lens" para exibir referências de funções e métodos diretamente no editor.
