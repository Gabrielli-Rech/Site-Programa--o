Exercícios Fáceis

  • Escreva um programa que verifica se um número é par ou ímpar.
  • Crie um script que verifica se um número é positivo, negativo ou zero.
  • Implemente um programa que determina se uma idade é maior ou igual a 18 anos.
  • Escreva um programa que imprime "Bom dia", "Boa tarde" ou "Boa noite" com base na hora atual.
  • Faça um programa que verifica se um número é divisível por 2 ou 3.
  • Escreva um script que imprime "Maior que 10" se um número for maior que 10, caso contrário, imprime "Menor ou igual a 10".
  • Crie um programa que verifica se uma string contém a palavra "Swift".
  • Implemente um script que verifica se um caractere é uma vogal.
  • Escreva um programa que imprime "É um fim de semana" se o dia atual for sábado ou domingo.
  • Crie um programa que verifica se um número é igual a 0 e imprime uma mensagem se for.

Exercícios Médios

  • Crie um script que determina se um ano é bissexto.
  • Escreva um programa que determina o maior de três números fornecidos pelo usuário.
  • Implemente um programa que classifica uma nota em "Aprovado", "Recuperação" ou "Reprovado".
  • Crie um script que imprime "Verão" se a estação for entre junho e setembro.
  • Escreva um programa que classifica um número como "positivo", "negativo" ou "zero" e imprime a mensagem correspondente.
  • Faça um programa que verifica se um número é múltiplo de 5 e imprime uma mensagem de acordo.
  • Implemente um script que verifica se uma palavra tem mais de 5 caracteres e imprime uma mensagem correspondente.
  • Escreva um programa que determina se um número é um quadrado perfeito.
  • Crie um script que determina se um usuário tem permissão para acessar um recurso com base em sua idade e nível de acesso.
  • Implemente um programa que verifica se uma variável contém um valor nulo e imprime uma mensagem se for o caso.

Exercícios Difíceis

  • Implemente um programa que calcula o salário de um funcionário considerando horas extras (acima de 40 horas).
  • Crie um script que simula uma calculadora com as operações de soma, subtração, multiplicação e divisão.
  • Escreva um programa que verifica se uma string é um palíndromo.
  • Implemente um script que determina a categoria de uma pessoa com base na idade (criança, adolescente, adulto, idoso).
  • Crie um programa que simula um jogo de "pedra, papel e tesoura".
  • Escreva um programa que calcula a média de notas e imprime a classificação do aluno.
  • Faça um script que verifica se um número é primo.
  • Implemente um programa que gera um número aleatório entre 1 e 100 e solicita ao usuário para adivinhar.
  • Crie um programa que calcula o IMC (Índice de Massa Corporal) e imprime uma mensagem com a classificação.
  • Escreva um script que determina se uma string é um anagrama de outra.

Desafios

  • Implemente um sistema de login com diferentes níveis de acesso (usuário, moderador, administrador).
  • Crie um programa que simula um caixa eletrônico, determinando a quantidade de notas necessárias para um saque.
  • Escreva um script que calcula o fatorial de um número.
  • Implemente um programa que verifica se uma senha atende aos critérios de segurança (número, letra maiúscula, símbolo, etc.).
  • Crie um script que determina se uma frase é um pangrama (contém todas as letras do alfabeto).
  • Escreva um programa que verifica se uma expressão matemática está balanceada em relação aos parênteses.
  • Implemente um sistema de cadastro de usuários com verificação de dados obrigatórios.
  • Crie um programa que determina a maior palavra em uma frase.
  • Escreva um script que simula um jogo de adivinhação com múltiplas tentativas.
  • Implemente um programa que determina se uma palavra é um homônimo (mesma forma, diferentes significados).
