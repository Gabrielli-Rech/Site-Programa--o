O que é Swift?

Swift é uma linguagem de programação desenvolvida pela Apple para criar aplicativos para iOS, macOS, watchOS e tvOS. É moderna, rápida e segura, projetada para ser fácil de aprender e trabalhar, com uma sintaxe concisa e expressiva.

Características e Sintaxe do Swift

Swift é conhecida por sua segurança de tipo, performance e interoperabilidade com Objective-C. Ela possui uma sintaxe que permite um código limpo e fácil de entender.

Exemplo de Código Swift

let saudacao = "Olá, mundo!"
print(saudacao)
