Exercícios

  • 1. Conectar ao banco de dados MongoDB e selecionar uma coleção existente.
  • 2. Inserir um documento em uma coleção usando o método insertOne().
  • 3. Inserir múltiplos documentos em uma coleção usando o método insertMany().
  • 4. Buscar todos os documentos de uma coleção usando o método find().
  • 5. Buscar documentos com uma condição específica usando find({ campo: valor }).
  • 6. Limitar o número de documentos retornados usando find().limit(n).
  • 7. Ordenar os resultados de uma consulta com find().sort({ campo: 1 }) (crescente) e find().sort({ campo: -1 }) (decrescente).
  • 8. Buscar documentos usando a operação de comparação $eq (igual).
  • 9. Buscar documentos usando a operação de comparação $gt e $lt (maior que e menor que).
  • 10. Utilizar a operação $and para fazer consultas compostas com múltiplas condições.
  • 11. Buscar documentos usando a operação $or para atender a pelo menos uma condição.
  • 12. Buscar documentos em que um campo esteja ausente com find({ campo: { $exists: false }}).
  • 13. Buscar documentos onde um campo contém um valor específico com find({ campo: { $in: [valor1, valor2] }}).
  • 14. Buscar documentos em que um campo tenha um valor específico, mas não em uma lista específica com find({ campo: { $nin: [valor1, valor2] }}).
  • 15. Buscar documentos com um campo que contenha um valor específico usando o operador $regex.
  • 16. Atualizar um documento com updateOne() e uma condição específica.
  • 17. Atualizar múltiplos documentos com updateMany().
  • 18. Substituir um documento inteiro com replaceOne().
  • 19. Deletar um documento específico com deleteOne().
  • 20. Deletar todos os documentos que atendem a uma condição com deleteMany().
