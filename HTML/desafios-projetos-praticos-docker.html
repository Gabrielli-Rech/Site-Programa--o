Desafios Fáceis

  • 1. Criar um contêiner Docker com uma imagem básica do Ubuntu e executar um comando ls.
  • 2. Criar um contêiner Docker com a imagem oficial do Node.js e executar um script simples de "Hello World".
  • 3. Compartilhar arquivos entre o sistema host e um contêiner Docker usando volumes.
  • 4. Configurar um contêiner que execute um servidor web simples com Python e visualizar a página em um navegador.
  • 5. Criar um contêiner Docker com a imagem do MySQL e acessar o banco de dados usando o terminal do contêiner.
  • 6. Escrever um Dockerfile simples para uma aplicação em Node.js.
  • 7. Testar a criação de um contêiner Docker a partir de uma imagem de um aplicativo frontend básico (HTML/CSS/JS).
  • 8. Rodar um contêiner que tenha o servidor Apache instalado e configurado para exibir uma página HTML.
  • 9. Utilizar o comando docker exec para executar um comando dentro de um contêiner em execução.
  • 10. Parar e remover contêineres e imagens com os comandos docker stop, docker rm e docker rmi.

Desafios Médios

  • 11. Criar um contêiner Docker que executa uma aplicação em Python Flask e acessá-la no navegador.
  • 12. Configurar uma rede Docker personalizada e conectar dois contêineres para que se comuniquem entre si.
  • 13. Criar um docker-compose.yml para uma aplicação composta por um frontend e um backend em contêineres separados.
  • 14. Utilizar o comando docker logs para monitorar e resolver problemas em um contêiner em execução.
  • 15. Criar e executar um contêiner Docker para um banco de dados PostgreSQL e inserir dados usando o terminal.
  • 16. Montar um contêiner de desenvolvimento que tenha as ferramentas de depuração de código instaladas.
  • 17. Usar um contêiner Docker para rodar um servidor Redis e interagir com ele por meio de um cliente Redis CLI.
  • 18. Configurar um contêiner Docker que execute um servidor de desenvolvimento em React e verificar a atualização automática.
  • 19. Criar um contêiner de uma aplicação em PHP e configurá-lo com o servidor Apache.
  • 20. Criar um contêiner com uma aplicação Java Spring Boot e rodar a aplicação em um ambiente isolado.

Desafios Difíceis

  • 21. Criar um contêiner Docker com múltiplos serviços (por exemplo, um backend em Node.js e um banco de dados MongoDB) e conectá-los.
  • 22. Configurar um contêiner Docker para rodar um servidor Nginx que faz proxy reverso para outro contêiner de aplicação.
  • 23. Criar um Dockerfile para uma aplicação em Go, otimizando a imagem para produção.
  • 24. Configurar uma rede Docker com múltiplas sub-redes e atribuir diferentes contêineres a diferentes sub-redes.
  • 25. Integrar Docker com uma ferramenta de CI/CD para automatizar a construção e o deploy de contêineres.
  • 26. Usar Docker Swarm para criar uma aplicação distribuída com escalabilidade horizontal de contêineres.
  • 27. Configurar um contêiner Docker com uma stack completa (LAMP) e criar um site PHP dinâmico.
  • 28. Criar uma imagem Docker personalizada para uma aplicação em Rust, com otimizações de build e execução.
  • 29. Usar Docker e Docker Compose para rodar um ambiente de desenvolvimento de uma aplicação em microserviços.
  • 30. Implementar um contêiner Docker que execute um cluster Kubernetes em um único nó.

Desafios Avançados

  • 31. Criar um sistema de logging centralizado usando o ELK Stack (Elasticsearch, Logstash, Kibana) em contêineres Docker.
  • 32. Configurar uma aplicação com Docker Compose e orquestrá-la em um cluster Kubernetes.
  • 33. Rodar uma aplicação em contêiner Docker com suporte a multi-stage builds para otimização de imagens.
  • 34. Implementar um ambiente de CI/CD usando Docker com Jenkins e GitHub Actions para build e deploy de contêineres.
  • 35. Configurar um contêiner para executar testes automatizados com Selenium WebDriver.
  • 36. Criar uma aplicação em Python que roda em contêineres Docker e se conecta a uma API de terceiros, gerenciando os dados com um banco de dados SQLite.
  • 37. Implementar um serviço de monitoramento em tempo real usando Prometheus e Grafana em contêineres Docker.
  • 38. Criar uma imagem Docker com uma aplicação de machine learning em Python que utiliza bibliotecas como TensorFlow.
  • 39. Configurar um ambiente de desenvolvimento com Docker que inclui emuladores de dispositivos móveis para teste de aplicativos.
  • 40. Criar um contêiner Docker que faça deploy de uma aplicação serverless com AWS Lambda usando o framework Serverless Framework.
