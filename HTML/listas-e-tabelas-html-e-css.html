O que são Listas e Tabelas?

Listas e tabelas são elementos fundamentais em HTML para organizar e apresentar dados de forma clara e estruturada. Listas são usadas para exibir coleções de itens, enquanto tabelas são ideais para dados tabulares.

Listas em HTML

Existem três principais tipos de listas em HTML:

Lista Não Ordenada (ul)

  • Item 1
  • Item 2
  • Item 3

Exemplo de lista com marcadores.

Lista Ordenada (ol)

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Exemplo de lista numerada.

Lista de Definição (dl)

HTML
Linguagem de marcação usada para estruturar páginas web.
CSS
Linguagem de estilo para definir a aparência de uma página web.

Exemplo de lista de termos e definições.

Tabelas em HTML

Tabelas são usadas para organizar dados em linhas e colunas. Veja como criar uma tabela em HTML:

Nome Idade Cidade
João 25 São Paulo
Maria 30 Rio de Janeiro
Pedro 22 Belo Horizonte

Para estilizar a tabela, você pode usar CSS e adicionar bordas, cores e outros efeitos.
