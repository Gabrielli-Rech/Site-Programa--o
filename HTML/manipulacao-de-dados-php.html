Exercícios Fáceis

  • Crie uma variável com uma string e use a função `strlen()` para encontrar o comprimento da string.
  • Crie um array de números e use `count()` para contar quantos elementos ele contém.
  • Crie uma string e use `strtoupper()` para convertê-la para maiúsculas.
  • Crie uma string e use `strtolower()` para convertê-la para minúsculas.
  • Crie uma variável com um número e use `abs()` para encontrar o valor absoluto.
  • Crie um array com alguns valores e use `array_merge()` para combinar com outro array.
  • Use a função `explode()` para dividir uma string em um array de substrings.
  • Use a função `implode()` para unir um array de strings em uma única string.
  • Crie uma variável de tipo inteiro e use `is_int()` para verificar se é um inteiro.
  • Use a função `trim()` para remover espaços em branco no início e no fim de uma string.

Exercícios Médios

  • Crie um array associativo e use `foreach()` para imprimir todas as chaves e valores.
  • Crie uma função que receba uma string e retorne a string reversa.
  • Use a função `substr()` para extrair uma substring de uma string.
  • Crie uma função que receba um array e retorne o maior valor.
  • Use a função `array_filter()` para remover todos os elementos vazios de um array.
  • Crie um array de strings e use `array_map()` para converter todos os elementos em maiúsculas.
  • Use `array_slice()` para extrair uma parte de um array e imprima o resultado.
  • Crie uma função que use `str_replace()` para substituir todas as ocorrências de uma palavra em uma string.
  • Crie uma string com espaços e use `str_pad()` para completar a string com caracteres até um tamanho desejado.
  • Use a função `explode()` para dividir uma string em um array com base em um delimitador específico.

Exercícios Difíceis

  • Crie uma função que receba um array de números e retorne um array com os números em ordem decrescente.
  • Crie uma função que encontre a palavra mais longa em uma string.
  • Use `array_reduce()` para calcular a soma de todos os elementos de um array.
  • Crie uma função que recebe um array e retorne a soma de todos os valores ímpares.
  • Crie uma função que use `array_unique()` para remover elementos duplicados de um array.
  • Crie um programa que use `preg_match()` para verificar se uma string contém uma palavra específica.
  • Crie uma função que use `str_split()` para dividir uma string em um array de caracteres e, em seguida, imprima os caracteres em ordem inversa.
  • Use a função `array_intersect()` para encontrar elementos comuns entre dois arrays e imprima o resultado.
  • Crie uma função que use `array_map()` para aplicar uma função matemática a todos os elementos de um array.
  • Crie uma função que receba um array de strings e retorne a string mais curta.

Desafios

  • Crie uma função que receba um array multidimensional e retorne uma lista plana (flatten) do array.
  • Desenvolva uma função que aceite um array de números e retorne um array com os números multiplicados pelo índice do elemento.
  • Crie uma função que use `preg_replace()` para substituir todas as vogais em uma string por um caractere específico.
  • Desenvolva uma função que receba um array de strings e retorne um array com todas as strings que contêm uma palavra específica.
  • Crie uma função que use `array_diff()` para encontrar a diferença entre dois arrays.
  • Implemente uma função que aceite um array de strings e retorne a mesma string em ordem alfabética.
  • Desenvolva uma função que aceite um array e retorne um array com os elementos duplicados removidos e ordenados.
  • Crie uma função que use `implode()` para formatar uma data como "dd-mm-aaaa" a partir de uma array de partes de data.
  • Implemente uma função que, dado um array de strings, retorne a quantidade de palavras que contêm a letra "e".
  • Crie um programa que use `array_search()` para encontrar o índice de um elemento em um array e imprima o resultado.
