Exercícios de Criação de Coleções.

  • Voltar
  • Início
  • 1. Criar um banco de dados chamado exercicios_db.
  • 2. Verificar se o banco de dados foi criado usando show dbs.
  • 3. Selecionar o banco de dados use exercicios_db.
  • 4. Criar uma coleção chamada usuarios usando db.createCollection("usuarios").
  • 5. Inserir um documento na coleção usuarios com os campos: nome, email e idade.
  • 6. Criar uma coleção chamada produtos e inserir alguns documentos com os campos: nome, preco e quantidade_em_estoque.
  • 7. Criar uma coleção chamada pedidos e inserir um documento com os campos: usuario_id, data_pedido e total.
  • 8. Criar uma coleção chamada itens_pedido e inserir documentos com os campos: pedido_id, produto_id, quantidade e preco_unitario.
  • 9. Criar uma coleção chamada categorias e inserir documentos com os campos: nome e descricao.
  • 10. Inserir um documento na coleção categorias que contenha um array de produtos associados.
  • 11. Criar uma coleção chamada log_acessos com os campos: usuario_id, data_acesso e ip_acesso.
  • 12. Criar uma coleção chamada avaliacoes e inserir documentos com os campos: produto_id, usuario_id, nota e comentario.
  • 13. Criar uma coleção chamada transacoes com os campos: usuario_id, data_transacao, valor e descricao.
  • 14. Inserir documentos na coleção usuarios com um campo de array para armazenar telefones.
  • 15. Criar uma coleção chamada enderecos e inserir documentos com os campos: usuario_id, rua, cidade e cep.
  • 16. Criar uma coleção chamada historico_pedidos e inserir documentos que armazenem as alterações em pedidos.
  • 17. Criar uma coleção chamada inventario e inserir documentos que contemplem informações detalhadas de produtos.
  • 18. Criar uma coleção chamada eventos e inserir documentos que incluam campos como data_evento, descricao e local.
  • 19. Criar uma coleção chamada clientes e adicionar um índice no campo email para consultas rápidas.
  • 20. Criar uma coleção chamada feedbacks e inserir documentos com os campos: usuario_id, comentario, data_feedback e status.
