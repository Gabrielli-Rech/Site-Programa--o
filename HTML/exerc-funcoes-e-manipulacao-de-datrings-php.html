Exercícios de Funções

  • Crie uma função que receba dois números e retorne a soma deles.
  • Implemente uma função que receba um array de números e retorne a média deles.
  • Crie uma função que receba uma string e retorne a mesma string com a primeira letra em maiúsculo.
  • Implemente uma função que receba uma string e um caractere e conte quantas vezes o caractere aparece na string.
  • Crie uma função que receba um número e retorne se ele é par ou ímpar.
  • Implemente uma função que receba um array de strings e retorne o maior comprimento de string.
  • Crie uma função que receba uma string e retorne a string invertida.
  • Implemente uma função que receba uma string e retorne a quantidade de vogais nela.
  • Crie uma função que receba um número e retorne o fatorial desse número.
  • Implemente uma função que receba dois números e retorne o maior deles.

Exercícios de Manipulação de Data e Hora

  • Crie uma função que retorne a data e hora atuais em um formato legível (ex.: "22 de Dezembro de 2024, 15:30").
  • Implemente uma função que receba uma data como string e retorne quantos dias faltam para essa data.
  • Crie uma função que receba uma data e retorne o dia da semana correspondente.
  • Implemente uma função que adicione um número específico de dias a uma data e retorne o resultado.
  • Crie uma função que receba uma data e retorne se é um ano bissexto.
  • Implemente uma função que converta uma data em um formato "YYYY-MM-DD" para "DD/MM/YYYY".
  • Crie uma função que receba duas datas e retorne a diferença em dias entre elas.
  • Implemente uma função que receba uma data e retorne o mês correspondente como texto (ex.: "Janeiro").
  • Crie uma função que retorne a quantidade de dias em um mês específico de um determinado ano.
  • Implemente uma função que retorne o número de dias que se passaram desde uma data específica até a data atual.
