Exercícios Comandos e Atalhos VSCode

  • Voltar
  • Início

Nível Fácil

  • Abrir a paleta de comandos: Use Ctrl + Shift + P (ou Cmd + Shift + P no Mac).
  • Abrir um arquivo: Use Ctrl + P (ou Cmd + P no Mac).
  • Salvar arquivo: Pressione Ctrl + S (ou Cmd + S no Mac).
  • Fechar arquivo: Use Ctrl + W (ou Cmd + W no Mac).
  • Criar um novo arquivo: Use Ctrl + N (ou Cmd + N no Mac).
  • Recarregar a janela do VSCode: Use Ctrl + Shift + P e digite Reload Window.
  • Mudar de aba: Use Ctrl + Tab (ou Cmd + Option + → no Mac).
  • Desfazer ação: Pressione Ctrl + Z (ou Cmd + Z no Mac).
  • Refazer ação: Use Ctrl + Y (ou Cmd + Shift + Z no Mac).
  • Abrir terminal integrado: Pressione Ctrl + ` (ou Cmd + ` no Mac).

Nível Médio

  • Pesquisar em todos os arquivos: Pressione Ctrl + Shift + F (ou Cmd + Shift + F no Mac).
  • Substituir em todos os arquivos: Use Ctrl + Shift + H (ou Cmd + Shift + H no Mac).
  • Ir para linha: Pressione Ctrl + G (ou Cmd + G no Mac) e digite o número da linha.
  • Abrir o painel de extensão: Use Ctrl + Shift + X (ou Cmd + Shift + X no Mac).
  • Recarregar a extensão: Pressione Ctrl + Shift + P e digite Reload Window.
  • Split editor (dividir editor): Use Ctrl + \ (ou Cmd + \ no Mac).
  • Mover linha para cima/baixo: Pressione Alt + ↑/↓ (ou Option + ↑/↓ no Mac).
  • Comentar/Descomentar linha: Use Ctrl + / (ou Cmd + / no Mac).
  • Selecionar tudo: Pressione Ctrl + A (ou Cmd + A no Mac).
  • Ajustar a indentação: Use Ctrl + ]/[ (ou Cmd + ]/[ no Mac).

Nível Difícil

  • Selecionar a palavra atual: Pressione Ctrl + D (ou Cmd + D no Mac).
  • Selecionar todas as instâncias da palavra: Use Ctrl + F2 (ou Cmd + F2 no Mac).
  • Refatorar código: Pressione F2 para renomear a variável ou função.
  • Ativar o modo de visualização de Markdown: Use Ctrl + Shift + V (ou Cmd + Shift + V no Mac).
  • Ativar o modo de edição de Markdown: Pressione Ctrl + Shift + M (ou Cmd + Shift + M no Mac).
  • Debug com breakpoints: Pressione F9 para adicionar ou remover breakpoints.
  • Iniciar depuração: Use F5 para iniciar a depuração.
  • Navegar pelo código: Use Ctrl + T (ou Cmd + T no Mac).
  • Gerar arquivo de configuração do Git: Use Ctrl + Shift + P e digite Git: Init.
  • Alternar entre editores: Use Ctrl + 1/2/3 (ou Cmd + 1/2/3 no Mac).

Desafios

  • Criar atalho customizado: Use Ctrl + K Ctrl + S para abrir configurações de atalhos.
  • Renomear arquivo: Pressione F2 para renomear o arquivo no explorador.
  • Buscar e substituir na linha de comando: Use Ctrl + Shift + P e digite Replace in Files.
  • Conectar-se a um repositório remoto: Use Ctrl + Shift + P, digite Git: Clone e conecte-se.
  • Habilitar modo Zen: Use Ctrl + K Z para entrar no modo de foco.
  • Ver alterações no Git: Use Ctrl + Shift + G para abrir o painel de controle de versões.
  • Ajustar o layout de edição: Use Ctrl + Alt + ←/→ para ajustar o layout de editores.
  • Alternar entre os terminais: Use Ctrl + ` e Ctrl + Shift + ` para alternar entre terminais.
  • Ativar modo de comparação de arquivos: Pressione Ctrl + Shift + P, digite Compare Active File With Saved.
  • Compartilhar terminal: Use Ctrl + Shift + P, digite Share Terminal.
