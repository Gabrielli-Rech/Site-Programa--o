O que é C#?

C# é uma linguagem de programação desenvolvida pela Microsoft, amplamente usada no desenvolvimento de aplicações para Windows, desenvolvimento de jogos com Unity, aplicativos web, e muito mais.

Características e Sintaxe do C#

C# é uma linguagem de programação orientada a objetos que oferece suporte a programação de alto nível, segurança de tipos, e programação assíncrona com o uso de tarefas e async/await.

Exemplo de Código C#

using System;

class Program {
    static void Main(string[] args) {
        Console.WriteLine("Olá, mundo!");
    }
}
