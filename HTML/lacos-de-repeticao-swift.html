O que são Laços de Repetição?

Laços de repetição são estruturas de controle que permitem executar um bloco de código múltiplas vezes enquanto uma condição é verdadeira. Eles são essenciais para automatizar tarefas repetitivas e processar dados em Swift.

O Laço for-in

O laço for-in é usado para iterar sobre coleções como arrays, dicionários e intervalos de números.

let frutas = ["maçã", "banana", "laranja"]
for fruta in frutas {
    print(fruta)
}

Este exemplo imprime cada fruta na lista no console.

Laço for com Intervalos

Você pode usar o laço for com intervalos para repetir um bloco de código um número específico de vezes:

for i in 1...5 {
    print(i)
}

Este exemplo imprime os números de 1 a 5 no console.

O Laço while

O laço while é utilizado quando não se sabe o número de repetições de antemão. O bloco de código é executado enquanto a condição for verdadeira.

var i = 0
while i < 5 {
    print(i)
    i += 1
}

Este exemplo imprime os números de 0 a 4 no console.

O Laço repeat-while

O laço repeat-while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

var i = 0
repeat {
    print(i)
    i += 1
} while i < 5

Este exemplo também imprime os números de 0 a 4 no console.
