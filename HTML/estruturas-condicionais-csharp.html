O que são Estruturas Condicionais?

Estruturas condicionais são usadas para executar blocos de código diferentes dependendo de uma condição. Elas permitem que o programa tome decisões baseadas em testes lógicos e variáveis.

Estrutura Condicional if

A estrutura if permite executar um bloco de código se uma condição for verdadeira:

if (condicao)
{
    // código a ser executado se a condição for verdadeira
}

Exemplo:

int idade = 18;
if (idade >= 18)
{
    Console.WriteLine("Você é maior de idade.");
}

Estrutura Condicional if-else

A estrutura if-else permite que você defina um bloco de código a ser executado quando a condição é falsa:

if (condicao)
{
    // código se a condição for verdadeira
}
else
{
    // código se a condição for falsa
}

Exemplo:

int idade = 16;
if (idade >= 18)
{
    Console.WriteLine("Você é maior de idade.");
}
else
{
    Console.WriteLine("Você é menor de idade.");
}

Estrutura Condicional if-else if-else

A estrutura if-else if-else permite testar múltiplas condições em sequência:

if (condicao1)
{
    // código se a condição1 for verdadeira
}
else if (condicao2)
{
    // código se a condição2 for verdadeira
}
else
{
    // código se nenhuma das condições for verdadeira
}

Exemplo:

int nota = 85;
if (nota >= 90)
{
    Console.WriteLine("Nota A");
}
else if (nota >= 80)
{
    Console.WriteLine("Nota B");
}
else if (nota >= 70)
{
    Console.WriteLine("Nota C");
}
else
{
    Console.WriteLine("Nota D");
}

Estrutura Condicional switch

A estrutura switch permite avaliar uma variável em diferentes casos e executar o código correspondente:

switch (variavel)
{
    case valor1:
        // código a ser executado se a variável for igual a valor1
        break;
    case valor2:
        // código a ser executado se a variável for igual a valor2
        break;
    default:
        // código a ser executado se nenhum dos casos for verdadeiro
        break;
}

Exemplo:

int dia = 3;
switch (dia)
{
    case 1:
        Console.WriteLine("Segunda-feira");
        break;
    case 2:
        Console.WriteLine("Terça-feira");
        break;
    case 3:
        Console.WriteLine("Quarta-feira");
        break;
    default:
        Console.WriteLine("Dia inválido");
        break;
}
