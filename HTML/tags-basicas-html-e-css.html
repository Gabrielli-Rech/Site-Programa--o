O que são as Tags em HTML?

As tags HTML são elementos que formam a estrutura de uma página web. Elas indicam ao navegador como o conteúdo deve ser exibido. As tags básicas são essenciais para a construção de qualquer página web.

Tag <html>

A tag <html> é a raiz de um documento HTML. Ela envolve todo o conteúdo da página.

<html>
    <head> ... </head>
    <body> ... </body>
</html>

Tag <head>

A tag <head> contém metadados da página, como título, links para CSS, e scripts.

<head>
    <title>Título da Página</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>

Tag <body>

A tag <body> contém todo o conteúdo visível da página, como texto, imagens, links, e outros elementos.

<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é um parágrafo de texto.</p>
    <a href="https://www.exemplo.com">Clique aqui</a>
</body>

Tags de Título (<h1> a <h6>)

As tags de título são usadas para criar títulos e subtítulos em uma página. Elas vão de <h1> (mais importante) a <h6> (menos importante).

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Subtítulo Menor</h3>

Tag <p>

A tag <p> é usada para definir parágrafos de texto.

<p>Este é um parágrafo de exemplo.</p>

Tag <a>

A tag <a> é usada para criar links. O atributo href define o destino do link.

<a href="https://www.exemplo.com">Visite o exemplo</a>

Tag <img>

A tag <img> é usada para inserir imagens. O atributo src define a fonte da imagem, e alt fornece um texto alternativo.

<img src="imagem.jpg" alt="Descrição da imagem">
