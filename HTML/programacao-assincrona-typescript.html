Exercícios de Programação Assíncrona

  • Crie uma função assíncrona em TypeScript que faça uma requisição HTTP e imprima a resposta no console.
  • Desenvolva uma função que utilize `async/await` para ler um arquivo e exibir seu conteúdo no console.
  • Implemente um script TypeScript que use `Promise.all` para executar múltiplas operações assíncronas em paralelo e mostrar os resultados.
  • Crie uma função que retorne uma promessa que resolve após um tempo determinado e use `await` para esperar por ela.
  • Desenvolva uma função que use `fetch` para fazer uma chamada a uma API e aguarde a resposta de forma assíncrona.
  • Crie uma função `async` que leia dados de um arquivo JSON e processe essas informações de forma assíncrona.
  • Implemente uma função que use `setTimeout` e `await` para criar um atraso assíncrono antes de executar outra função.
  • Escreva uma função que use `async/await` para buscar e processar dados de uma API com tratamento de erros usando `try/catch`.
  • Desenvolva uma função que utilize `Promise.race` para executar múltiplas promessas e aguarde a que terminar primeiro.
  • Crie uma função que use `fetch` e `async/await` para fazer uma requisição e exibir o status da resposta.
  • Implemente um script que use `async/await` para baixar e salvar imagens de URLs em um diretório local.
  • Desenvolva uma função que utilize `Promise.allSettled` para aguardar várias promessas e exibir o resultado de cada uma delas.
  • Crie uma função que use `async/await` para fazer uma requisição de múltiplas etapas com encadeamento de promessas.
  • Desenvolva uma função que use `fetch` com `async/await` e decodifique a resposta JSON de uma API.
  • Implemente uma função que utilize `await` para esperar por múltiplas promessas e depois agregue os resultados em um array.
  • Crie uma função assíncrona em TypeScript que simule um processo de carga de dados com um tempo de espera.
  • Desenvolva um script que use `async/await` para fazer a consulta a uma API de forma sequencial.
  • Escreva uma função que utilize `async/await` para buscar dados de uma API com um parâmetro de filtro e exiba os resultados.
  • Crie uma função que use `async/await` e `setInterval` para executar uma tarefa repetidamente com um atraso específico.
  • Desenvolva uma função que use `async/await` e `Promise.all` para processar dados de múltiplas fontes em paralelo.
  • Implemente uma função que utilize `async/await` para processar dados de uma API e trate diferentes estados de resposta com `if/else`.
