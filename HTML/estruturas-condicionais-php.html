O que são Estruturas Condicionais?

As estruturas condicionais permitem executar diferentes blocos de código com base em uma condição específica. Em PHP, você pode usar estruturas como if, else, elseif e switch para controlar o fluxo do programa.

Estrutura if

A estrutura if é usada para executar um bloco de código se uma condição for verdadeira.

$idade = 20;
if ($idade >= 18) {
    echo "Você é maior de idade.";
}

Se a condição for falsa, o bloco de código não é executado.

Estrutura if-else

Com a estrutura if-else, você pode executar um bloco de código se a condição for verdadeira e outro se for falsa.

$idade = 16;
if ($idade >= 18) {
    echo "Você é maior de idade.";
} else {
    echo "Você é menor de idade.";
}

Essa estrutura garante a execução de um dos dois blocos de código, dependendo da condição.

Estrutura if-elseif-else

Para testar múltiplas condições, você pode usar a estrutura if-elseif-else.

$nota = 85;
if ($nota >= 90) {
    echo "Nota A";
} elseif ($nota >= 80) {
    echo "Nota B";
} elseif ($nota >= 70) {
    echo "Nota C";
} else {
    echo "Nota abaixo de C";
}

Essa estrutura permite a verificação de múltiplas condições em sequência.

Estrutura switch

A estrutura switch é usada para comparar uma variável com múltiplos casos possíveis e executar o bloco de código correspondente ao caso verdadeiro.

$dia = "terça";
switch ($dia) {
    case "segunda":
        echo "Hoje é segunda-feira.";
        break;
    case "terça":
        echo "Hoje é terça-feira.";
        break;
    case "quarta":
        echo "Hoje é quarta-feira.";
        break;
    default:
        echo "Dia não reconhecido.";
}

O comando break é usado para sair do switch após a execução de um caso. Se omitido, a execução continua para os próximos casos.

Operadores Lógicos

Os operadores lógicos permitem combinar condições em uma estrutura condicional:

  • && (E): Retorna verdadeiro se ambas as condições forem verdadeiras.
  • || (OU): Retorna verdadeiro se pelo menos uma condição for verdadeira.
  • ! (NÃO): Inverte o valor lógico da condição.

Exemplo:

$idade = 20;
$temCarro = true;
if ($idade >= 18 && $temCarro) {
    echo "Você pode dirigir.";
} else {
    echo "Você não pode dirigir.";
}
