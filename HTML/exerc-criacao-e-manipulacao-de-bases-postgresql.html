Exercícios de Criação e Manipulação de Bases

  • Voltar
  • Início
  1. Crie uma nova base de dados chamada empresa.
  2. Crie uma tabela chamada funcionarios com as colunas id, nome, cargo e salario.
  3. Adicione uma nova coluna data_admissao à tabela funcionarios.
  4. Crie uma tabela departamentos com as colunas id, nome e localizacao.
  5. Insira um novo funcionário na tabela funcionarios com nome e cargo específicos.
  6. Atualize o salário de um funcionário específico na tabela funcionarios.
  7. Exclua um funcionário da tabela funcionarios usando o id.
  8. Recupere todos os funcionários que recebem um salário superior a 5000.
  9. Crie uma restrição de chave primária na coluna id da tabela funcionarios.
  10. Adicione uma chave estrangeira na coluna departamento_id da tabela funcionarios que referencia a tabela departamentos.
  11. Liste os nomes de todos os funcionários e o nome do departamento ao qual pertencem.
  12. Crie um índice na coluna nome da tabela funcionarios.
  13. Crie uma visualização chamada funcionarios_view que exibe os nomes e salários dos funcionários.
  14. Crie uma função que retorne o nome completo de um funcionário baseado no id.
  15. Use uma transação para inserir um novo funcionário e, em seguida, excluir um registro da tabela departamentos.
  16. Liste todos os nomes de tabelas no banco de dados empresa.
  17. Crie uma tabela projetos com as colunas id, nome, data_inicio e data_fim.
  18. Adicione um campo status na tabela projetos com os valores possíveis 'ativo' e 'concluído'.
  19. Insira vários funcionários ao mesmo tempo na tabela funcionarios.
  20. Exporte a tabela funcionarios para um arquivo CSV.
  21. Importe dados de um arquivo CSV para a tabela departamentos.
  22. Crie uma tabela salarios_historico com as colunas id, funcionario_id, salario, data_inicio e data_fim.
  23. Atualize a tabela salarios_historico para incluir o salário atual do funcionário.
  24. Verifique o tamanho da tabela funcionarios em termos de número de linhas.
  25. Crie uma tabela temporária temp_funcionarios que contém os mesmos dados que a tabela funcionarios.
  26. Liste os cinco funcionários com maior salário.
  27. Crie uma restrição que impede a inserção de um salario menor que 0 na tabela funcionarios.
  28. Adicione uma coluna email na tabela funcionarios e torne-a única.
  29. Crie um índice de texto completo na coluna nome da tabela funcionarios.
  30. Exclua todas as linhas da tabela projetos que possuem data_fim anterior a uma data específica.
  31. Crie uma trigger que atualize a coluna data_admissao da tabela funcionarios toda vez que um registro for atualizado.
  32. Crie uma tabela clientes com as colunas id, nome, telefone e email.
  33. Adicione uma chave estrangeira da tabela clientes para a tabela departamentos.
  34. Crie uma função que calcule a média de salários da tabela funcionarios.
  35. Exclua a tabela temp_funcionarios.
  36. Crie uma tabela funcionarios_temp com dados de teste usando a cláusula SELECT INTO.
  37. Modifique a tabela departamentos para incluir a coluna orcamento do tipo decimal.
  38. Liste todas as views existentes no banco de dados.
  39. Crie uma trigger que insira um registro de auditoria toda vez que uma linha da tabela funcionarios for excluída.
  40. Verifique as permissões de um usuário específico na base de dados empresa.
