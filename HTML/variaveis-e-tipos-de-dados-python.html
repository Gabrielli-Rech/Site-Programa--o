O que são Variáveis e Tipos de Dados?

Em Python, variáveis são usadas para armazenar valores que podem ser manipulados em um programa. Python é uma linguagem de tipagem dinâmica, o que significa que não é necessário declarar explicitamente o tipo de dado de uma variável.

Declaração de Variáveis

Para declarar uma variável em Python, você simplesmente atribui um valor a um nome de variável:

nome = "Ana"  # string
idade = 28  # inteiro
altura = 1.65  # float
ativo = True  # booleano

Note que não é necessário especificar o tipo da variável, pois Python infere automaticamente o tipo baseado no valor atribuído.

Tipos de Dados em Python

Os tipos de dados em Python podem ser classificados como primitivos e complexos:

  • Primitivos:
    • int: Números inteiros. Exemplo: 42, -10
    • float: Números de ponto flutuante. Exemplo: 3.14, -0.001
    • str: Strings (cadeias de texto). Exemplo: 'Olá, mundo!'
    • bool: Valores booleanos. Exemplo: True, False
    • NoneType: Representa a ausência de valor. Exemplo: None
  • Complexos: Tipos de dados como listas, tuplas, dicionários e conjuntos, que podem conter múltiplos valores e serem manipulados de diversas formas.

Exemplo de diferentes tipos de dados:

nome = 'Carlos'  # string
idade = 30  # int
altura = 1.78  # float
ativo = False  # booleano
endereco = None  # NoneType
