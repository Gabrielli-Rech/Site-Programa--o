O que são Estruturas Condicionais?

Estruturas condicionais são usadas para executar diferentes blocos de código dependendo de uma condição. Em TypeScript, você pode usar as instruções if, else e switch para criar lógica condicional.

Estrutura if-else

A estrutura if-else permite executar um bloco de código se uma condição for verdadeira e outro bloco se for falsa.

let numero: number = 10;
if (numero > 0) {
    console.log("O número é positivo");
} else if (numero < 0) {
    console.log("O número é negativo");
} else {
    console.log("O número é zero");
}

Estrutura switch

A estrutura switch é usada para comparar um valor com múltiplas possibilidades, tornando o código mais legível e organizado.

let diaDaSemana: string = "terça-feira";
switch (diaDaSemana) {
case "segunda-feira":
    console.log("Hoje é segunda-feira");
    break;
case "terça-feira":
    console.log("Hoje é terça-feira");
    break;
case "quarta-feira":
    console.log("Hoje é quarta-feira");
    break;
default:
    console.log("Outro dia da semana");
}

Nota: Em TypeScript, o uso de break em cada case é necessário para evitar a execução de blocos subsequentes.

Operador Ternário

O operador ternário é uma forma concisa de escrever uma estrutura if-else. Ele é ideal para condições simples.

let numero: number = 5;
let resultado: string = numero > 0 ? "Positivo" : "Negativo";
console.log(resultado);

O operador ternário ? avalia a condição e, se for verdadeira, retorna o valor antes do :. Se for falsa, retorna o valor após o :.
