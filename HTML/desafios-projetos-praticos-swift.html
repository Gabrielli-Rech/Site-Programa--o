Desafios Fáceis

  • 1. Criar uma função que soma dois números e retorna o resultado.
  • 2. Implementar um programa que verifica se um número é par ou ímpar.
  • 3. Desenvolver uma função que calcula a área de um círculo dado o raio.
  • 4. Criar um array e exibir todos os números pares nele.
  • 5. Escrever uma função que inverte uma string.
  • 6. Implementar uma estrutura condicional que exibe "Bom dia", "Boa tarde" ou "Boa noite" com base na hora.
  • 7. Criar um programa que calcula o fatorial de um número usando um loop.
  • 8. Desenvolver uma função que converte uma temperatura de Celsius para Fahrenheit.
  • 9. Criar uma estrutura `struct` para representar um ponto em um plano cartesiano.
  • 10. Implementar uma função que verifica se uma palavra é um palíndromo.

Desafios Médios

  • 11. Criar uma classe `Pessoa` com propriedades `nome` e `idade` e um método para apresentar a pessoa.
  • 12. Desenvolver uma função que ordena um array de strings em ordem alfabética.
  • 13. Implementar um programa que lê um dicionário e exibe suas chaves e valores.
  • 14. Criar uma classe `ContaBancaria` com métodos para depósito, saque e saldo.
  • 15. Desenvolver uma função que conta o número de vogais em uma string.
  • 16. Implementar uma função recursiva para calcular a sequência de Fibonacci.
  • 17. Criar um programa que simula o lançamento de um dado e exibe o resultado.
  • 18. Desenvolver uma estrutura `enum` para representar os quatro pontos cardeais.
  • 19. Criar uma classe base `Veiculo` com subclasses `Carro` e `Moto` e métodos específicos para cada um.
  • 20. Escrever uma função que encontra o maior número em um array multidimensional.

Desafios Difíceis

  • 21. Criar uma classe `Animal` com métodos para comer e dormir e heranças para subclasses como `Cachorro` e `Gato`.
  • 22. Desenvolver um programa que utiliza closures para filtrar um array de números maiores que 10.
  • 23. Implementar um programa que usa generics para criar uma pilha de elementos.
  • 24. Criar um jogo simples no terminal onde o jogador adivinha um número aleatório.
  • 25. Desenvolver um programa que utiliza protocolo para definir a funcionalidade básica de formas geométricas.
  • 26. Implementar uma classe que realiza operações matemáticas complexas, como raiz quadrada e exponenciação.
  • 27. Criar um programa que faz o parsing de uma string JSON para um objeto Swift.
  • 28. Desenvolver um algoritmo para resolver o problema da mochila utilizando programação dinâmica.
  • 29. Criar um sistema de gerenciamento de tarefas com CRUD básico usando Swift e arquivos locais.
  • 30. Implementar um aplicativo básico de lista de compras com SwiftUI.

Desafios Avançados

  • 31. Criar um aplicativo que utiliza Core Data para gerenciar informações persistentes.
  • 32. Desenvolver uma API REST simples usando Vapor para gerenciar usuários.
  • 33. Implementar um jogo básico em 2D usando o framework SpriteKit.
  • 34. Criar uma aplicação que utiliza notificações locais e envio de notificações push.
  • 35. Desenvolver um sistema de login utilizando Keychain para armazenamento seguro de credenciais.
  • 36. Criar um programa que utiliza operações assíncronas com Combine para carregar dados de uma API.
  • 37. Implementar um aplicativo que utiliza MapKit para exibir a localização atual do usuário.
  • 38. Desenvolver uma aplicação de monitoramento de saúde utilizando HealthKit.
  • 39. Criar uma biblioteca Swift personalizada para realizar validação de dados.
  • 40. Implementar uma solução para o problema de "produtor e consumidor" usando Swift e multithreading.
