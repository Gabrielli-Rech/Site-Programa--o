Exercícios de Classes e Heranças

Fáceis

  • Crie uma classe `Pessoa` com propriedades `nome` e `idade`. Adicione um método para exibir as informações da pessoa.
  • Adicione um construtor à classe `Carro` para inicializar as propriedades `marca` e `modelo`.
  • Crie uma classe `Animal` com um método `emitirSom` que exibe um som genérico no console.
  • Faça uma classe `Produto` com propriedades `nome` e `preco` e crie um método para calcular um desconto.
  • Implemente uma classe `ContaBancaria` com métodos para depositar e sacar dinheiro.
  • Crie uma classe `Livro` com propriedades `titulo` e `autor` e um método que retorna as informações do livro.
  • Adicione uma propriedade `dataFabricacao` em uma classe `Veiculo` e exiba-a no console.
  • Crie uma classe `Curso` com propriedades `nome` e `duracao`. Adicione um método para exibir os detalhes do curso.
  • Crie uma classe `Relogio` com propriedades `hora` e `minuto`, e um método que exibe o horário no formato HH:MM.
  • Adicione um método `alterarPreco` à classe `Produto` que redefine o valor da propriedade `preco`.
  • Implemente uma classe `Usuario` com métodos para definir e exibir o nome do usuário.
  • Crie uma classe `Calculadora` com métodos básicos de soma, subtração, multiplicação e divisão.
  • Adicione validações simples em uma classe `Conta` para verificar se o saldo é suficiente antes de realizar saques.
  • Implemente uma classe `Jogador` com propriedades como `nome` e `pontuacao` e métodos para alterar a pontuação.
  • Crie uma classe `Loja` com uma propriedade `endereco` e um método para exibir o endereço.
  • Adicione propriedades como `altura` e `largura` em uma classe `Retangulo`, e crie um método para calcular a área.
  • Crie uma classe `Filme` com propriedades `titulo` e `genero`, e um método para exibir o título do filme.
  • Adicione um método estático `converterCelsiusParaFahrenheit` em uma classe `Temperatura`.
  • Crie uma classe `Aluno` com propriedades `nome` e `nota`, e adicione um método que verifica se o aluno foi aprovado.
  • Implemente uma classe `Telefone` com métodos para fazer e receber chamadas.

Médios

  • Crie uma classe `Funcionario` e uma classe derivada `Gerente`, com propriedades e métodos específicos para cada uma.
  • Adicione um método `apresentar` em uma classe `Aluno` que herda de uma classe `Pessoa`.
  • Implemente uma hierarquia com a classe `FormaGeometrica` e classes derivadas como `Quadrado` e `Circulo`, cada uma com seu próprio método para calcular a área.
  • Use `protected` em uma propriedade para permitir o acesso em classes derivadas.
  • Adicione métodos estáticos em uma classe `Util` para realizar operações como soma ou cálculo de porcentagem.
  • Crie uma classe `Pagamento` e herde dela classes específicas como `Pix` e `Boleto`, cada uma com seu método de processamento.
  • Implemente a funcionalidade de carrinho de compras com uma classe `Carrinho` que gerencia uma lista de itens.
  • Crie uma classe `Funcionario` com subclasses `Engenheiro` e `Gerente`, e implemente métodos específicos em cada uma.
  • Adicione validações em propriedades usando métodos `get` e `set` em uma classe `Produto`.
  • Crie uma classe `Veiculo` com subclasses `Carro` e `Moto`, e implemente um método que retorne o tipo de veículo.
  • Implemente um sistema simples de inventário com classes `Item` e `Inventario`.
  • Crie um método genérico em uma classe `Lista` para adicionar e buscar elementos.
  • Use interfaces para garantir que uma classe `Animal` e suas subclasses implementem métodos específicos.
  • Adicione lógica para verificar duplicatas ao adicionar itens em uma classe `Estoque`.
  • Implemente um sistema de controle de acesso com classes `Usuario` e `Admin`, com permissões diferentes.
  • Crie uma classe `Calendario` com métodos para verificar feriados e adicionar eventos.

  • Implemente lógica de persistência de dados em memória usando uma classe `BancoDeDados`.

  • Adicione um método abstrato em uma classe `Documento`, implementado por subclasses como `PDF` e `Word`.

Difíceis

  • Desenvolva uma hierarquia de classes para um sistema de reservas com classes `Reserva`, `Hotel`, e `Voo`, garantindo que todas implementem métodos específicos como `calcularCusto`.
  • Implemente um sistema de transporte com classes base `Veiculo` e subclasses específicas como `Navio` e `Aviao`. Adicione lógica para calcular tempo de viagem com base na velocidade.
  • Crie uma classe abstrata `Conta` e herde dela classes como `ContaCorrente` e `ContaPoupanca`. Adicione regras para cada tipo de conta.
  • Implemente um sistema de pagamento com classes `Pagamento` e `PagamentoOnline`, usando interfaces para garantir métodos padrão.
  • Adicione lógica para verificar duplicatas em uma lista de objetos em uma classe `InventarioAvancado`.
  • Crie uma classe `Tabela` com métodos para adicionar, atualizar e excluir linhas. Implemente relações com uma classe `BancoDeDados`.
  • Desenvolva um jogo simples com classes `Personagem`, `Monstro`, e `Item`. Use herança para diferenciar características.
  • Crie um sistema de notificações com classes `Notificacao`, `Email`, e `SMS`, cada uma implementando o método `enviar` de forma diferente.
  • Desenvolva uma classe `Pedido` e uma classe `Produto`. Associe múltiplos produtos a um pedido e implemente métodos para calcular o total.
  • Implemente um sistema de cadastro de veículos com classes `Veiculo`, `Carro`, e `Caminhao`. Adicione validações específicas para cada tipo.
  • Use interfaces para definir um contrato entre classes de diferentes níveis de herança em um sistema de controle de estoque.
  • Crie uma hierarquia de classes para um sistema de saúde com `Paciente`, `Medico`, e `Hospital`, cada um com funcionalidades específicas.
  • Adicione métodos assíncronos em uma classe `API` para simular chamadas de rede e processamento de dados.
  • Implemente um sistema de aluguel de carros com classes `Carro`, `Aluguel`, e `Cliente`. Adicione lógica para verificar disponibilidade.
  • Desenvolva um sistema de classificação de filmes com classes `Filme` e `Critico`, e implemente um método para calcular a média de avaliações.
  • Crie uma classe `Estoque` com métodos para buscar produtos por nome e categoria.
  • . Desenvolva uma classe `Banco` com métodos para transferir valores entre objetos `Conta`.
  • Crie uma classe `Biblioteca` para gerenciar empréstimos de livros, com validações para disponibilidade.
  • Implemente um sistema de gerenciamento de projetos com classes `Projeto`, `Tarefa`, e `MembroEquipe`.
  • Adicione métodos para gerar relatórios em uma classe `SistemaFinanceiro`, com herança para categorias como `Despesa` e `Receita`.
