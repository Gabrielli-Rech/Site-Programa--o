Exercícios de Coleções e Mapas em Java

Exercícios sobre Coleções

  • 1. Crie uma lista de inteiros e adicione 10 números a ela. Em seguida, imprima todos os elementos da lista.
  • 2. Desenvolva um programa que crie uma lista de strings e remova a segunda palavra da lista.
  • 3. Implemente um exemplo que use `ArrayList` para armazenar nomes de pessoas e imprima o nome com o maior comprimento.
  • 4. Crie um programa que adicione elementos a uma lista e use um `for-each` para imprimir os elementos da lista em ordem inversa.
  • 5. Escreva um programa que crie uma lista de inteiros, remova os elementos ímpares e imprima a lista resultante.
  • 6. Desenvolva um exemplo que use `LinkedList` para adicionar elementos na frente e no final da lista e imprima a lista.
  • 7. Crie um programa que use `HashSet` para armazenar 10 números inteiros e imprima a coleção sem elementos duplicados.
  • 8. Implemente um exemplo em que um `TreeSet` seja usado para armazenar strings em ordem alfabética e imprima a coleção.
  • 9. Escreva um programa que crie uma lista de objetos personalizados e use um `Comparator` para ordenar a lista de acordo com um campo específico.
  • 10. Crie uma lista de objetos e use `Collections.sort()` para classificá-los com base em um critério específico.
  • 11. Implemente um programa que use `PriorityQueue` para armazenar números e imprima-os em ordem crescente.
  • 12. Desenvolva um exemplo que use `Vector` para armazenar e imprimir uma coleção de strings.
  • 13. Crie um programa que use `CopyOnWriteArrayList` para gerenciar uma lista segura para uso em um ambiente multithreaded.
  • 14. Escreva um exemplo de um programa que use `ArrayList` para implementar uma pilha com os métodos `push()` e `pop()`.
  • 15. Implemente um programa que use um `LinkedList` para simular uma fila com os métodos `offer()` e `poll()`.
  • 16. Desenvolva um programa que utilize `Stack` para empilhar e desempilhar elementos e imprima a pilha após cada operação.
  • 17. Crie um exemplo em que você use `HashSet` para encontrar elementos comuns entre duas listas.
  • 18. Escreva um programa que use `TreeSet` para armazenar e buscar elementos de forma eficiente em uma coleção ordenada.
  • 19. Implemente um programa que utilize `LinkedHashSet` para manter a ordem de inserção de elementos e imprima a coleção.
  • 20. Desenvolva um exemplo em que você use `Collections.shuffle()` para embaralhar uma lista de elementos.
  • 21. Crie um programa que use `ListIterator` para percorrer uma lista de strings e modificar seus elementos.
  • 22. Escreva um exemplo em que você use `HashMap` para mapear chaves e valores e imprima o mapa completo.
  • 23. Implemente um programa que use `TreeMap` para armazenar pares de chave-valor em ordem natural de suas chaves.
  • 24. Crie um programa que use `LinkedHashMap` para manter a ordem de inserção de elementos e imprima o mapa.
  • 25. Desenvolva um exemplo que utilize `ConcurrentHashMap` para gerenciar um mapa de dados em um ambiente multithreaded.
  • 26. Escreva um programa que crie um `HashMap` com nomes como chaves e idades como valores e imprima as entradas.
  • 27. Implemente um exemplo em que você use `Hashtable` para armazenar dados e imprima as chaves e valores.
  • 28. Crie um programa que use `EnumMap` para mapear valores associados a um enum e imprima o conteúdo do mapa.
  • 29. Desenvolva um exemplo que use `WeakHashMap` para armazenar pares de chave-valor com referências fracas.
  • 30. Escreva um programa que use `IdentityHashMap` para mapear chaves com base em suas referências de identidade.
  • 31. Crie um exemplo de um programa que use `ConcurrentSkipListMap` para armazenar pares de chave-valor de forma ordenada.
  • 32. Implemente um programa que use `SortedMap` para mapear e ordenar entradas com base na chave.
  • 33. Desenvolva um exemplo de um programa que use `Map.Entry` para iterar sobre os elementos de um `HashMap`.
  • 34. Escreva um programa que use `Collections.synchronizedMap()` para criar um mapa thread-safe e imprima o mapa.
  • 35. Implemente um programa que use `LinkedList` para armazenar elementos em ordem e imprima os elementos após cada inserção.
  • 36. Crie um exemplo em que você use `Collections.reverse()` para inverter a ordem de uma lista de elementos.
  • 37. Desenvolva um exemplo que use `Collections.frequency()` para contar quantas vezes um elemento aparece em uma lista.
  • 38. Escreva um programa que use `Collections.max()` para encontrar o maior elemento em uma lista de números.
  • 39. Implemente um exemplo que use `Collections.min()` para encontrar o menor elemento em uma lista de strings.
  • 40. Crie um programa que utilize `Collections.rotate()` para rotacionar os elementos de uma lista.
  • 41. Desenvolva um exemplo que use `List.of()` para criar uma lista imutável e imprima seus elementos.
  • 42. Escreva um programa que use `Set.of()` para criar um conjunto imutável e imprima o conteúdo do conjunto.
  • 43. Implemente um exemplo que use `Map.of()` para criar um mapa imutável com entradas chave-valor.
  • 44. Crie um programa que use `Collections.addAll()` para adicionar elementos a uma lista existente.
  • 45. Desenvolva um exemplo que utilize `HashMap` e `Stream` para filtrar elementos com base em um critério.
  • 46. Escreva um programa que use `ArrayList` e `Stream` para mapear elementos de uma lista e imprimir os resultados.
  • 47. Implemente um exemplo que use `Set` e `Stream` para remover elementos duplicados de uma lista de strings.
  • 48. Crie um programa que use `HashMap` para armazenar contatos de uma agenda e imprima todos os contatos.
  • 49. Desenvolva um exemplo que use `TreeMap` para armazenar entradas de um dicionário e imprima os elementos em ordem de chave.
  • 50. Escreva um programa que use `LinkedHashMap` para manter a ordem de inserção e imprima os elementos em ordem.
