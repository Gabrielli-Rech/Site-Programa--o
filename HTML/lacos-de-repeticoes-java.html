O que são Laços de Repetição?

Laços de repetição, ou loops, são usados para executar um bloco de código várias vezes de forma repetitiva, enquanto uma condição é verdadeira ou até que um determinado número de iterações seja alcançado. São úteis para automatizar tarefas e processar listas de dados.

Principais Laços de Repetição em Java

Veja os principais tipos de laços de repetição em Java:

  • Laço for: Ideal para quando o número de iterações é conhecido.
  • Laço while: Ideal para quando o número de iterações não é conhecido e depende de uma condição.
  • Laço do-while: Semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

Exemplo de laço for:

for (int i = 0; i < 5; i++) {
    System.out.println("Iteração: " + i);
}

Exemplo de laço while:

int i = 0;
while (i < 5) {
    System.out.println("Iteração: " + i);
    i++;
}

Exemplo de laço do-while:

int i = 0;
do {
    System.out.println("Iteração: " + i);
    i++;
} while (i < 5);
