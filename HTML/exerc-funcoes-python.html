  • 1. Escreva uma função que receba dois números e retorne a soma deles.

  • 6. Implemente uma função que receba um número e retorne se ele é par ou ímpar.
  • 7. Escreva uma função que calcule a área de um retângulo dado o comprimento e a largura.
  • 8. Crie uma função que receba um número e retorne sua raiz quadrada.
  • 9. Implemente uma função que receba um número e retorne "Positivo", "Negativo" ou "Zero".
  • 10. Escreva uma função que receba uma lista e retorne o comprimento dela.
  • 11. Crie uma função que receba uma string e retorne a quantidade de vogais nela.
  • 12. Implemente uma função que receba dois números e retorne a diferença entre eles.
  • 13. Escreva uma função que imprima "Python é incrível!".
  • 14. Crie uma função que receba um número e retorne seu fatorial.
  • 15. Implemente uma função que receba uma lista de números e retorne a soma deles.
  • 16. Escreva uma função que imprima os números de 1 a 10.
  • 17. Crie uma função que receba um número e retorne a sua representação em binário.
  • 18. Implemente uma função que receba um texto e retorne o texto invertido.
  • 19. Escreva uma função que receba um número e retorne "Par" ou "Ímpar".
  • 20. Crie uma função que calcule o comprimento de uma string.

  • 11. Implemente uma função que verifique se um número é primo.

  • 13. Escreva uma função que verifique se uma lista de números está ordenada de forma crescente.
  • 14. Implemente uma função que receba uma lista e retorne a soma dos elementos que são múltiplos de 3.
  • 15. Crie uma função que receba um texto e retorne o texto com a primeira letra de cada palavra em maiúscula.

  • 17. Implemente uma função que receba uma lista e retorne a lista com os elementos em ordem inversa.

  • 20. Implemente uma função que imprima a sequência de Fibonacci até um número dado.

Desafios

  • 1. Crie uma função que implemente a lógica de um jogo de adivinhação em que o usuário tenta acertar um número aleatório.
  • 2. Implemente uma função que encontre o elemento que aparece mais vezes em uma lista.
  • 3. Escreva uma função que calcule o valor de uma série matemática usando laços de repetição.
  • 4. Crie uma função que receba uma lista e retorne a lista com os elementos em ordem crescente de acordo com o número de ocorrências.
  • 5. Implemente uma função que retorne o n-ésimo número de Fibonacci de forma otimizada.
  • 6. Escreva uma função que receba uma lista de listas e retorne uma lista com a soma de cada sublista.
  • 7. Crie uma função que verifique se uma palavra é um anagrama de outra.
  • 8. Implemente uma função que calcule a soma de uma progressão geométrica.
  • 9. Escreva uma função que receba uma lista e retorne a soma de todos os números de índices pares.
  • 10. Crie uma função que calcule a soma de todos os elementos de uma lista usando recursão.
  • 11. Implemente uma função que encontre a palavra mais longa em um texto dado.
  • 12. Escreva uma função que calcule a quantidade de dígitos em um número.
  • 13. Crie uma função que receba um número e retorne a quantidade de vezes que ele aparece na sequência de Fibonacci.
  • 14. Implemente uma função que converta uma string em um formato de título, onde cada palavra começa com maiúscula.
  • 15. Escreva uma função que receba uma lista de números e retorne o produto de todos os números não repetidos.
  • 16. Crie uma função que calcule o somatório de uma lista de números de forma assíncrona.
  • 17. Implemente uma função que receba uma lista de palavras e retorne a quantidade de palavras com mais de 3 letras.
  • 18. Escreva uma função que encontre todos os pares de números em uma lista que somam um número específico.
  • 19. Crie uma função que receba uma lista de números e retorne a lista de forma reversa sem usar a função `reverse()`.
  • 20. Implemente uma função que encontre o menor elemento em uma lista e, se existir, o elemento que vem após ele.
