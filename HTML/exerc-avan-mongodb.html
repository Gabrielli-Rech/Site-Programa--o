Exercícios Avançados de MongoDB

  • 1. Realize uma agregação para calcular a média de uma coluna em uma coleção.
  • 2. Implemente uma busca com `lookup` para unir duas coleções em uma consulta.
  • 3. Crie uma consulta usando o operador `$match` para filtrar documentos por múltiplas condições.
  • 4. Use o operador `$group` para agrupar documentos e contar o número de vezes que um valor aparece.
  • 5. Realize uma atualização com `updateMany` para modificar múltiplos documentos de uma vez.
  • 6. Crie um índice composto em uma coleção para melhorar a performance de consultas complexas.
  • 7. Implemente uma consulta que use o operador `$in` para selecionar documentos com valores de um array específico.
  • 8. Faça uma consulta que utilize `$lookup` e `$unwind` para desmembrar resultados de uma agregação.
  • 9. Utilize o operador `$regex` para buscar documentos que correspondam a um padrão específico de texto.
  • 10. Crie uma consulta para usar o operador `$sort` e ordenar documentos por mais de um campo.
  • 11. Realize uma consulta para usar o operador `$project` e exibir apenas certos campos de um documento.
  • 12. Crie um documento usando o método `insertOne` e um documento com `insertMany`.
  • 13. Use a operação `deleteOne` e `deleteMany` para remover documentos específicos de uma coleção.
  • 14. Utilize o operador `$addFields` para adicionar um novo campo a todos os documentos em uma coleção.
  • 15. Crie uma consulta que use `$lookup` com uma subconsulta para buscar apenas documentos que atendam a critérios específicos.
  • 16. Implemente uma atualização usando o operador `$set` e `$unset` para modificar e remover campos de documentos.
  • 17. Realize uma consulta que utilize `$facet` para dividir uma agregação em diferentes pipelines simultâneos.
  • 18. Crie um trigger no MongoDB para realizar uma ação quando um documento for inserido em uma coleção.
  • 19. Use o operador `$merge` para armazenar o resultado de uma agregação em uma nova coleção.
  • 20. Crie uma consulta com `$map` para aplicar uma transformação em um array dentro de um documento.
  • 21. Realize uma consulta para usar `$lookup` e `$project` em uma única operação para simplificar a exibição de dados.
  • 22. Implemente uma consulta que utilize o operador `$sample` para retornar um número aleatório de documentos.
  • 23. Use a função `db.collection.countDocuments()` para contar o número de documentos que atendem a uma condição.
  • 24. Crie uma consulta usando o operador `$sort` com `$limit` para encontrar os principais resultados.
  • 25. Realize uma consulta com `$text` para buscar documentos em uma coleção com um índice de texto.
  • 26. Implemente uma consulta que utilize o operador `$graphLookup` para realizar buscas recursivas em um grafo de documentos.
  • 27. Crie uma consulta usando `$bucket` para agrupar documentos em faixas de valores.
  • 28. Utilize o operador `$lookup` para combinar informações de duas coleções em uma consulta aninhada.
  • 29. Realize uma atualização condicional usando `updateOne` e um filtro com `if` em JavaScript.
  • 30. Crie uma consulta que utilize o operador `$setWindowFields` para calcular valores de janelas em uma agregação.
  • 31. Implemente uma consulta que use `$out` para armazenar o resultado de uma agregação em uma coleção existente.
  • 32. Realize uma consulta com o operador `$map` para modificar elementos em um array.
  • 33. Use o operador `$mergeObjects` para combinar múltiplos objetos em um único documento.
  • 34. Crie uma consulta com `$lookup` e `aggregate` para agregar e mostrar informações de várias coleções.
  • 35. Implemente uma consulta com `$count` para contar o número de documentos após uma operação de `match`.
  • 36. Realize uma operação para buscar e remover um documento específico usando `findOneAndDelete`.
  • 37. Crie uma consulta que utilize o operador `$map` para aplicar uma operação a todos os elementos de um array.
  • 38. Implemente uma agregação com `$sort` e `$skip` para implementar paginação de resultados.
  • 39. Realize uma consulta para aplicar o operador `$set` e atualizar um campo específico de todos os documentos.
  • 40. Crie uma consulta com `$merge` para armazenar o resultado de uma consulta em uma coleção externa.
  • 41. Use o operador `$unwind` para desfazer a estrutura de um array em documentos.
  • 42. Implemente uma consulta que utilize o operador `$convert` para mudar o tipo de um campo.
  • 43. Crie uma operação que utilize `$lookup` e `$unwind` para retornar somente os elementos que correspondem a critérios específicos.
  • 44. Realize uma atualização usando `$addToSet` para adicionar um valor a um array somente se não existir.
  • 45. Crie uma consulta que utilize `$replaceRoot` para substituir o documento pai por um campo específico.
  • 46. Implemente uma operação que use `$group` para encontrar a soma total de um campo em uma coleção.
  • 47. Realize uma consulta com `$lookup` que utilize `localField` e `foreignField` para definir como as coleções são unidas.
  • 48. Crie uma consulta com o operador `$push` para adicionar elementos a um array em um documento.
  • 49. Implemente uma operação que use `$graphLookup` para fazer buscas de padrões em um conjunto de dados em grafo.
  • 50. Realize uma consulta que utilize `$lookup` e `$project` para mostrar apenas certos campos após a junção.
