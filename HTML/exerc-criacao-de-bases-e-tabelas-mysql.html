Exercícios de Criação de Bases e Tabelas.

  • Voltar
  • Início
  • 1. Criar um banco de dados chamado exercicios_db.
  • 2. Verificar se o banco de dados foi criado com sucesso usando SHOW DATABASES;.
  • 3. Selecionar o banco de dados USE exercicios_db;.
  • 4. Criar uma tabela chamada usuarios com colunas: id (chave primária, auto incremento), nome (VARCHAR), email (VARCHAR único) e data_nascimento (DATE).
  • 5. Criar uma tabela chamada produtos com colunas: id (chave primária, auto incremento), nome (VARCHAR), preco (DECIMAL), quantidade_em_estoque (INT).
  • 6. Criar uma tabela chamada pedidos com colunas: id (chave primária, auto incremento), usuario_id (chave estrangeira referenciando usuarios), data_pedido (DATE) e total (DECIMAL).
  • 7. Criar uma tabela chamada itens_pedido com colunas: id (chave primária, auto incremento), pedido_id (chave estrangeira referenciando pedidos), produto_id (chave estrangeira referenciando produtos), quantidade (INT) e preco_unitario (DECIMAL).
  • 8. Adicionar uma restrição de chave primária na coluna id da tabela usuarios.
  • 9. Adicionar uma restrição de chave única na coluna email da tabela usuarios.
  • 10. Adicionar uma chave estrangeira na coluna usuario_id da tabela pedidos que referencie a tabela usuarios.
  • 11. Adicionar uma chave estrangeira na coluna produto_id da tabela itens_pedido que referencie a tabela produtos.
  • 12. Adicionar uma chave estrangeira na coluna pedido_id da tabela itens_pedido que referencie a tabela pedidos.
  • 13. Criar uma tabela categorias com colunas: id (chave primária, auto incremento), nome (VARCHAR) e descricao (TEXT).
  • 14. Adicionar uma coluna categoria_id à tabela produtos e definir como chave estrangeira que referencie a tabela categorias.
  • 15. Criar uma tabela log_acessos com colunas: id (chave primária, auto incremento), usuario_id (chave estrangeira), data_acesso (TIMESTAMP) e ip_acesso (VARCHAR).
  • 16. Criar uma tabela avaliacoes com colunas: id (chave primária, auto incremento), produto_id (chave estrangeira), usuario_id (chave estrangeira), nota (INT) e comentario (TEXT).
  • 17. Criar um índice na coluna email da tabela usuarios para melhorar a busca.
  • 18. Criar um índice composto nas colunas nome e preco da tabela produtos.
  • 19. Criar uma tabela transacoes com colunas: id (chave primária, auto incremento), usuario_id (chave estrangeira), data_transacao (TIMESTAMP), valor (DECIMAL) e descricao (TEXT).
  • 20. Alterar a tabela usuarios para incluir uma coluna telefone (VARCHAR).
