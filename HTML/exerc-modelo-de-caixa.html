Exercícios Básicos

  1. Crie um elemento `div` e aplique uma borda de 2px sólida preta.
  2. Adicione `padding` de 10px em uma `div` e observe como o conteúdo se afasta das bordas internas.
  3. Defina uma `div` com largura de 200px, altura de 100px e uma borda sólida azul de 5px.
  4. Adicione uma margem de 20px a uma `div` e observe como ela se afasta de outros elementos.
  5. Use a propriedade `box-sizing: border-box;` em uma `div` e veja como a largura e a altura são calculadas.
  6. Crie um elemento com largura de 150px e altura de 150px, adicionando `padding` de 15px e uma borda de 5px. Calcule o tamanho total da caixa.
  7. Defina um elemento com `margin: auto;` e faça com que ele seja centralizado horizontalmente.
  8. Adicione uma sombra à borda de um elemento usando `box-shadow`.
  9. Use a propriedade `outline` para adicionar um contorno externo a uma `div` sem alterar o tamanho da caixa.
  10. Crie um botão com `border-radius` para deixá-lo com bordas arredondadas.

Exercícios Intermediários

  1. Crie uma `div` com `padding` diferente para cada lado (ex.: 10px para cima, 20px para direita, 15px para baixo, 25px para esquerda).
  2. Defina uma `div` com uma borda dupla (`border-style: double`) e largura de 10px.
  3. Use `margin` e `padding` para posicionar dois elementos adjacentes com espaçamento uniforme.
  4. Adicione uma borda pontilhada (`dotted`) de 3px a um elemento e uma margem de 15px.
  5. Crie um layout onde uma `div` tenha bordas diferentes (ex.: borda superior azul e borda inferior verde).
  6. Defina um elemento com largura de 300px e altura de 200px, mas faça com que ele se ajuste ao conteúdo interno usando `min-width` e `min-height`.
  7. Crie um cartão com sombra (`box-shadow`) e espaçamento interno usando `padding`.
  8. Adicione uma borda interna (`inset`) a uma `div` usando a propriedade `box-shadow`.
  9. Experimente com bordas arredondadas aplicando valores diferentes para cada canto de um elemento.
  10. Crie um elemento com `overflow: hidden;` para ocultar conteúdo que ultrapasse os limites da caixa.

Exercícios Avançados

  1. Implemente um layout de cartões, onde cada cartão tenha uma borda, `margin` e `padding` personalizados.
  2. Crie um elemento com múltiplas sombras usando a propriedade `box-shadow`.
  3. Adicione bordas animadas a um elemento usando a propriedade `@keyframes` e `border`.
  4. Experimente com a propriedade `clip-path` para cortar uma `div` em um formato personalizado.
  5. Combine bordas, `margin` e `padding` para criar um layout com espaçamento uniforme entre elementos.
  6. Crie um elemento que tenha bordas personalizadas, como uma imagem de fundo na borda, usando `border-image`.
  7. Use `outline-offset` para criar um efeito de contorno destacado em uma `div` ao passar o mouse.
  8. Crie um elemento com `padding` baseado em porcentagens do tamanho do contêiner.
  9. Experimente com `overflow: scroll;` e `overflow: auto;` em uma `div` que contém muito texto.
  10. Crie uma barra de progresso personalizada usando bordas e sombras.

Exercícios de Desafios

  1. Implemente um layout de galeria de imagens com bordas arredondadas e espaçamento uniforme.
  2. Crie um cartão interativo que mude o tamanho das bordas ao passar o mouse.
  3. Faça uma animação onde a sombra de uma `div` pulsa continuamente.
  4. Implemente um layout com várias `divs` sobrepostas usando `margin`, `padding` e `z-index`.
  5. Crie uma interface de caixa de diálogo com bordas, sombra e espaçamento interno personalizado.
  6. Desenvolva uma animação onde as bordas de uma caixa mudem de cor gradualmente.
  7. Construa um botão com efeito de borda flutuante usando `@keyframes` e `box-shadow`.
  8. Implemente um layout onde caixas tenham `padding` e `margin` que variem dinamicamente com base no tamanho da janela.
  9. Desenvolva um efeito onde o contorno de uma `div` cresce ao passar o mouse, mas sem alterar seu tamanho real.
  10. Crie um design de caixa com bordas que formam um gradiente contínuo usando `border-image`.
