Exercícios Práticos

  • Instale a extensão Prettier - Code formatter e formate um arquivo de código usando o atalho de formatação.
  • Instale a extensão Live Server e use-a para iniciar um servidor local para um projeto HTML/CSS.
  • Pesquise e instale a extensão GitLens para explorar o histórico de commits e descobrir quem alterou o que em um arquivo.
  • Adicione a extensão Bracket Pair Colorizer e observe como ela colore os pares de parênteses, colchetes e chaves para facilitar a leitura do código.
  • Instale a extensão Path Intellisense para ajudar a autocompletar caminhos de arquivos ao importar módulos em seus projetos.
  • Experimente a extensão Debugger for Chrome e depure um aplicativo JavaScript executando-o no navegador Chrome.
  • Instale a extensão Material Icon Theme para alterar os ícones de arquivos na barra lateral e torná-los mais fáceis de identificar.
  • Configure a extensão Settings Sync para sincronizar suas configurações do VSCode em vários dispositivos.
  • Use a extensão Markdown Preview Enhanced para visualizar arquivos Markdown com recursos avançados de formatação.
  • Explore a extensão Jest para integrar a execução de testes unitários em um projeto JavaScript/TypeScript.
  • Experimente a extensão Material Theme para mudar o tema do editor e explorar diferentes paletas de cores.
  • Instale a extensão Todo Tree para visualizar e gerenciar rapidamente todas as tarefas TODO e FIXME em um projeto.
  • Use a extensão Code Spell Checker para identificar e corrigir erros de digitação em seu código.
  • Explore a extensão Settings Cycler para alternar rapidamente entre diferentes configurações de VSCode em um projeto.
  • Instale a extensão Indent-Rainbow para ajudar a visualizar a indentação do código com cores.
  • Teste a extensão Git Graph para visualizar o histórico de commits de uma maneira gráfica e interativa.
  • Configure e use a extensão Docker para integrar e gerenciar seus containers Docker diretamente do VSCode.
  • Explore a extensão PHP Intelephense para obter autocompletar, análise e recursos de refatoração de código PHP.
  • Adicione a extensão Python e use o suporte integrado para linting, depuração e execução de testes em projetos Python.
  • Instale a extensão REST Client para testar suas APIs diretamente do VSCode sem precisar de ferramentas externas.
  • Use a extensão CodeMetrics para visualizar as métricas de complexidade de código e manter o código mais limpo e compreensível.
  • Configure e use a extensão Project Manager para organizar e alternar rapidamente entre diferentes projetos no VSCode.
  • Instale a extensão Settings Sync e teste sua funcionalidade para sincronizar suas configurações e extensões entre diferentes dispositivos.
