O que são Estruturas Condicionais?

Em Java, estruturas condicionais são usadas para tomar decisões no código, executando diferentes blocos de código com base em uma condição. Essas estruturas permitem que o programa siga diferentes caminhos dependendo das condições específicas.

Principais Estruturas Condicionais

Veja as principais estruturas condicionais em Java:

  • Estrutura if: Executa um bloco de código se a condição for verdadeira.
  • Estrutura if-else: Executa um bloco de código se a condição for verdadeira, e outro se for falsa.
  • Estrutura else-if: Verifica múltiplas condições de forma sequencial.
  • Estrutura switch: Permite escolher entre múltiplas opções com base no valor de uma variável.

Exemplo de estrutura if:

int idade = 18;

if (idade >= 18) {
    System.out.println("Você é maior de idade.");
} else {
    System.out.println("Você é menor de idade.");
}

Exemplo de estrutura switch:

int dia = 3;
String nomeDoDia;

switch (dia) {
    case 1:
        nomeDoDia = "Segunda-feira";
        break;
    case 2:
        nomeDoDia = "Terça-feira";
        break;
    case 3:
        nomeDoDia = "Quarta-feira";
        break;
    default:
        nomeDoDia = "Dia inválido";
        break;
}

System.out.println("O dia da semana é: " + nomeDoDia);
