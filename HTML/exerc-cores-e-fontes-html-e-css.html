Exercícios de Cores e Fontes em CSS

Fáceis

  • 1. Altere a cor de fundo de um elemento usando a propriedade `background-color`.
  • 2. Modifique a cor do texto de um elemento usando a propriedade `color`.
  • 3. Use a propriedade `font-family` para aplicar uma fonte específica a um elemento de texto.
  • 4. Altere a cor de um botão quando o mouse passa sobre ele usando `:hover`.
  • 5. Defina uma cor de texto usando a notação hexadecimal, RGB e HSL.
  • 6. Aplique uma cor de fundo degradê linear a um elemento com `background: linear-gradient()`.
  • 7. Use a propriedade `opacity` para ajustar a opacidade de um elemento.
  • 8. Aplique uma sombra de texto usando a propriedade `text-shadow`.
  • 9. Crie uma borda com uma cor específica em um elemento com `border`.
  • 10. Utilize a propriedade `font-size` para ajustar o tamanho da fonte de um elemento.
  • 11. Modifique a cor de fundo de um elemento em uma página inteira com `background-color`.
  • 12. Aplique um gradiente radial com a propriedade `background: radial-gradient()`.
  • 13. Use a propriedade `line-height` para ajustar o espaçamento entre linhas de texto.
  • 14. Defina um estilo de fonte em negrito com `font-weight`.
  • 15. Modifique a cor do texto de um parágrafo com uma classe específica.
  • 16. Aplique uma fonte cursiva usando `font-style: italic`.
  • 17. Crie um botão com uma cor de fundo que muda de cor ao passar o mouse.
  • 18. Use `font-variant` para exibir texto em um estilo alternativo, como pequenas maiúsculas.
  • 19. Altere a cor do texto de um elemento ao clicar com `:active`.
  • 20. Aplique uma sombra externa em um elemento usando `box-shadow`.

  • 15. Crie um layout de caixa com `box-sizing: border-box` e aplique uma borda colorida.
  • 16. Aplique uma fonte e cor diferentes a uma tabela com classes específicas.
