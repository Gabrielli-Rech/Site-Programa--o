Exercícios de Estruturas Condicionais

Exercícios Fáceis

  • Escreva um script que verifica se um número é par ou ímpar.
  • Crie um programa que verifica se um número é positivo, negativo ou zero.
  • Faça um script que exiba "Bom dia", "Boa tarde" ou "Boa noite" dependendo do horário atual.
  • Verifique se um número está entre 1 e 100.
  • Escreva um script que verifica se uma idade é maior ou igual a 18 anos e exiba "Maior de idade" ou "Menor de idade".
  • Crie um programa que verifica se uma variável está vazia e exiba uma mensagem correspondente.
  • Escreva um script que verifica se dois números são iguais.
  • Verifique se uma string contém a palavra "PHP".
  • Escreva um script que verifica se um ano é bissexto.
  • Crie um programa que verifica se um número é divisível por 3.

Exercícios Médios

  • Crie um script que verifica se uma senha tem pelo menos 8 caracteres e contém números.
  • Escreva um programa que classifica uma nota em "Aprovado", "Recuperação" ou "Reprovado".
  • Crie um script que verifica se três números podem formar os lados de um triângulo.
  • Faça um programa que verifica se um CPF é válido com base no número de dígitos.
  • Escreva um script que verifica se um usuário tem permissão de administrador com base em uma variável `role`.
  • Crie um programa que exibe o maior de três números.
  • Escreva um script que verifica se uma string é um palíndromo.
  • Faça um programa que calcula o preço de um produto com base em sua categoria e aplicação de descontos.
  • Escreva um script que verifica se um número está dentro de um intervalo fornecido pelo usuário.
  • Crie um programa que avalia se uma data fornecida está no passado, presente ou futuro.

Exercícios Difíceis

  • Crie um script que valida um endereço de e-mail usando expressões regulares.
  • Faça um programa que verifica se uma senha é forte (mínimo 8 caracteres, contendo letras maiúsculas, minúsculas, números e símbolos).
  • Escreva um script que valida uma data no formato "dd/mm/aaaa".
  • Crie um programa que verifica se um número é primo.
  • Faça um script que simula um sistema de login com três tentativas antes de bloquear o usuário.
  • Escreva um programa que calcula o imposto de renda com base em faixas salariais definidas.
  • Crie um script que verifica se dois intervalos de números têm interseção.
  • Escreva um programa que verifica se um horário fornecido está dentro de um período específico (por exemplo, das 8h às 18h).
  • Crie um script que valida um cartão de crédito usando o algoritmo de Luhn.
  • Faça um programa que determina se duas strings são anagramas.

Desafios

  • Implemente um sistema de validação de formulários que verifica nome, e-mail, CPF e senha.
  • Crie um programa que simula um caixa eletrônico, permitindo saques e exibindo o número de cédulas de cada valor necessário.
  • Escreva um script que determina o vencedor de um jogo de pedra, papel e tesoura com base nas escolhas dos jogadores.
  • Crie um programa que avalia se uma expressão matemática está balanceada (parênteses abertos e fechados corretamente).
  • Faça um script que classifica o IMC (Índice de Massa Corporal) de uma pessoa com base no peso e altura fornecidos.
  • Escreva um programa que verifica se uma lista de tarefas foi concluída (todas as tarefas marcadas como "concluídas").
  • Implemente um sistema de autenticação com múltiplos níveis de acesso (usuário comum, moderador e administrador).
  • Crie um programa que verifica se uma palavra é um heterograma (nenhuma letra se repete).
  • Escreva um script que classifica um conjunto de números em pares, ímpares, positivos e negativos.
  • Faça um programa que analisa os resultados de um campeonato de futebol e determina o time vencedor.
