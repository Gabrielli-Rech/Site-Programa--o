Exercícios

  • 1. Criar um banco de dados chamado empresa.
  • 2. Criar uma tabela chamada funcionarios com as colunas id (inteiro), nome (texto), cargo (texto) e salario (decimal).
  • 3. Inserir 5 registros de funcionários na tabela funcionarios.
  • 4. Selecionar todos os registros da tabela funcionarios.
  • 5. Selecionar apenas os nomes e cargos dos funcionários da tabela funcionarios.
  • 6. Atualizar o salário de um funcionário específico com id 3 para 5000.
  • 7. Deletar um registro da tabela funcionarios onde o id seja 2.
  • 8. Selecionar os funcionários cujo salário seja maior que 3000.
  • 9. Contar o número de registros na tabela funcionarios.
  • 10. Selecionar todos os funcionários ordenados por nome em ordem crescente.
  • 11. Selecionar todos os funcionários que têm um cargo específico, por exemplo, "Analista".
  • 12. Criar uma nova tabela chamada departamentos com as colunas id (inteiro) e nome (texto).
  • 13. Inserir 3 registros na tabela departamentos.
  • 14. Alterar a tabela funcionarios para adicionar uma coluna departamento_id (inteiro) que será uma chave estrangeira referenciando a tabela departamentos.
  • 15. Selecionar os nomes dos funcionários que pertencem a um departamento específico.
  • 16. Selecionar a média de salário dos funcionários da tabela funcionarios.
  • 17. Selecionar os 5 funcionários com os maiores salários.
  • 18. Criar uma consulta para exibir o funcionário com o menor salário.
  • 19. Criar uma consulta para exibir o número total de funcionários por cargo.
  • 20. Criar uma consulta que exiba os funcionários cujo nome começa com a letra "A".
