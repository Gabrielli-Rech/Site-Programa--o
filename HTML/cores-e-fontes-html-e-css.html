O que são Cores e Fontes em CSS?

CSS permite a personalização de cores e fontes para deixar as páginas web mais atraentes e legíveis. As cores podem ser definidas de várias formas, e as fontes podem ser aplicadas para criar diferentes estilos de texto.

Cores em CSS

Em CSS, você pode definir cores de diferentes formas:

  • Nome da cor: Exemplos como red, blue, green.
  • Código hexadecimal: Representado por um código de 6 dígitos, como #FF5733.
  • RGB: Valores de cor em formato RGB, como rgb(255, 87, 51).
  • RGBA: RGB com opacidade, como rgba(255, 87, 51, 0.5).
  • HSL: Representação em formato de matiz, saturação e luminosidade, como hsl(12, 100%, 60%).
  • HSLA: HSL com opacidade, como hsla(12, 100%, 60%, 0.5).

Exemplo de aplicação de cores:

p {
    color: #FF5733; /* Cor hexadecimal */
    background-color: rgb(255, 87, 51); /* Cor RGB */
}

Fontes em CSS

CSS permite a personalização de fontes para definir o estilo e a aparência do texto:

  • Propriedade font-family: Define a fonte do texto. Exemplos: font-family: 'Arial', sans-serif;.
  • Propriedade font-size: Define o tamanho da fonte. Exemplo: font-size: 16px;.
  • Propriedade font-weight: Define a espessura da fonte. Exemplos: font-weight: bold;, font-weight: 400;.
  • Propriedade font-style: Define se a fonte é normal, em itálico ou oblíqua. Exemplo: font-style: italic;.
  • Propriedade line-height: Define a altura da linha, ajustando o espaçamento entre linhas. Exemplo: line-height: 1.5;.

Exemplo de aplicação de fontes:

h1 {
    font-family: 'Times New Roman', serif;
    font-size: 36px;
    font-weight: bold;
    font-style: italic;
    line-height: 1.4;
}

Usando Fontes Externas

Você pode usar serviços como Google Fonts para incluir fontes externas em seu projeto:

Exemplo de importação de fonte do Google Fonts:

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

body {
    font-family: 'Roboto', sans-serif;
}
