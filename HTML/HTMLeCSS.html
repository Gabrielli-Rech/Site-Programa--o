O que são HTML e CSS?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar a estrutura básica de páginas da web. CSS (Cascading Style Sheets) é a linguagem de estilização usada para definir a aparência visual dessas páginas.

Estrutura e Sintaxe do HTML

O HTML é composto por elementos que formam a estrutura de uma página. Cada elemento é composto por tags de abertura e fechamento.

Exemplo de Estrutura HTML

<!DOCTYPE html>
<html>
    <head>
        <title>Exemplo de Página</title>
    </head>
    <body>
        <h1>Olá, mundo!</h1>
        <p>Este é um exemplo de uma página HTML.</p>
    </body>
</html>

Estilizando com CSS

O CSS é usado para aplicar estilos como cores, fontes, espaçamento e layout aos elementos HTML.

Exemplo de CSS

body {
    background-color: #f4f4f4;
    font-family: 'Arial', sans-serif;
}

h1 {
    color: #005f99;
}

p {
    color: #333;
}
