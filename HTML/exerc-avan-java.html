Exercícios Avançados em Java

  • 1. Implemente um sistema de multithreading que simule um servidor atendendo várias conexões simultaneamente.
  • 2. Crie uma aplicação que leia e escreva arquivos em formato CSV usando a biblioteca OpenCSV.
  • 3. Desenvolva um programa que implemente um algoritmo de Dijkstra para encontrar o caminho mais curto em um grafo.
  • 4. Escreva um sistema de gerenciamento de usuários com autenticação e autorização usando Java e Spring Security.
  • 5. Implemente um parser de arquivos JSON em Java usando a biblioteca Jackson ou Gson.
  • 6. Crie um programa que use reflexão para listar todos os métodos e atributos de uma classe específica.
  • 7. Desenvolva um aplicativo de chat em tempo real usando Java e WebSockets.
  • 8. Implemente um algoritmo de ordenação rápido (QuickSort) em Java.
  • 9. Crie uma aplicação de cadastro de produtos com persistência em banco de dados MySQL.
  • 10. Desenvolva um programa que utilize a API de Streams para manipulação e transformação de listas de objetos.
  • 11. Implemente um sistema de leitura e gravação de arquivos XML usando a API DOM e SAX.
  • 12. Crie um jogo de Sudoku em Java que permita ao usuário preencher e validar a solução do quebra-cabeça.
  • 13. Desenvolva um programa que utilize a API de concorrência de Java para simular um baralho de cartas sendo distribuído entre jogadores.
  • 14. Escreva um sistema de autenticação de dois fatores (2FA) em uma aplicação web usando Java e uma API de SMS.
  • 15. Implemente uma classe que simule a execução de um algoritmo de ordenação por heap (HeapSort).
  • 16. Crie um aplicativo de previsão de tempo que consuma uma API de dados meteorológicos e exiba as informações em tempo real.
  • 17. Desenvolva uma aplicação para gerenciar eventos com funcionalidade de calendário em JavaFX.
  • 18. Implemente um sistema de análise de logs com Java para identificar padrões de erros em um arquivo de log.
  • 19. Escreva um programa que utilize a API de Annotations para criar anotações personalizadas em Java.
  • 20. Crie uma aplicação de gerenciamento de estoque com controle de entrada e saída de produtos usando uma interface gráfica em JavaFX.
  • 21. Implemente uma função que detecte e remova duplicatas em uma lista de objetos usando `HashSet`.
  • 22. Desenvolva um sistema de recomendação de livros baseado em análise de dados e preferências do usuário.
  • 23. Crie uma aplicação que utilize as expressões regulares para validar números de telefone e endereços de e-mail.
  • 24. Escreva um algoritmo que implemente a busca em largura (BFS) em um grafo.
  • 25. Implemente um programa de compressão de dados usando o algoritmo de Huffman.
  • 26. Desenvolva um sistema de gerenciamento de contas bancárias com cálculos de juros compostos em Java.
  • 27. Crie uma aplicação de visualização de gráficos em Java usando bibliotecas como JFreeChart.
  • 28. Implemente um sistema de chat de grupo com persistência em banco de dados.
  • 29. Crie uma aplicação de agendamento de tarefas que utilize o agendador de tarefas `ScheduledExecutorService`.
  • 30. Desenvolva um programa que utilize o conceito de programação funcional com `Optional`, `Map`, e `Reduce`.
  • 31. Implemente uma classe para resolver equações diferenciais numéricas usando o método de Runge-Kutta.
  • 32. Crie um sistema de autenticação usando JSON Web Token (JWT) com uma API RESTful em Java.
  • 33. Desenvolva uma aplicação de gerenciamento de clientes em um sistema de vendas com Java e Hibernate.
  • 34. Escreva um programa que use uma API de reconhecimento facial para identificar rostos em imagens.
  • 35. Implemente um sistema de busca textual eficiente com estrutura de árvore de prefixo (Trie).
  • 36. Crie um programa de controle de versões de arquivos em Java, semelhante ao Git.
  • 37. Desenvolva uma aplicação que consuma uma API RESTful com autenticação OAuth2 e exiba os dados em uma interface gráfica.
  • 38. Implemente um sistema de controle de versões com o uso de `Git` e `Java` para criar branches e merge.
  • 39. Crie um jogo de quebra-cabeça com movimentação de peças e validação de solução em Java.
  • 40. Escreva uma aplicação que permita criar e gerenciar tarefas com diferentes prazos e níveis de prioridade.
  • 41. Desenvolva um projeto de um leitor de QR code em Java que utilize bibliotecas externas para ler códigos de barras e QR codes.
  • 42. Implemente um sistema de recomendação de músicas baseado em algoritmos de aprendizado de máquina em Java.
  • 43. Crie uma aplicação em Java que consuma e analise dados de redes sociais usando APIs RESTful.
  • 44. Desenvolva uma aplicação de monitoramento de servidores que utilize a API `JMX` para capturar métricas de desempenho.
  • 45. Implemente um sistema de vendas online com funcionalidades de carrinho de compras e checkout em Java.
  • 46. Crie um sistema de gerenciamento de tickets de suporte com funcionalidade de busca e categorização de chamados.
  • 47. Desenvolva um projeto de leitura de dados em tempo real com Java e uso de WebSockets para comunicação bidirecional.
  • 48. Escreva um algoritmo que utilize a técnica de programação dinâmica para resolver o problema da mochila (Knapsack problem).
  • 49. Crie uma aplicação de visualização de dados de sensores com gráficos e relatórios em Java.
  • 50. Desenvolva um projeto de sistema de votação online com funcionalidades de autenticação e votação anônima em Java.
