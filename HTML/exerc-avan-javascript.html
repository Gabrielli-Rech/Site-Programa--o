Desafios para Dominar JavaScript

Os exercícios avançados de JavaScript são projetados para ajudá-lo a aplicar conceitos mais complexos, como manipulação avançada de objetos, programação funcional, funções assíncronas e padrões de design.

Pratique com os seguintes exercícios:

Manipulação Avançada de Objetos e Arrays

  1. Crie uma função que receba um objeto aninhado e retorne um objeto achatado (flatten).
  2. Implemente uma função que filtre um array de objetos com base em múltiplos critérios.
  3. Crie um sistema de ordenação de objetos que funcione com diferentes propriedades e ordens (ascendente/descendente).
  4. Escreva uma função que combine dois arrays de objetos com base em uma propriedade comum.
  5. Crie uma função que transforme um array em um mapa (Map) onde as chaves sejam propriedades do array.

Programação Funcional

  1. Implemente uma função pipe que permita encadear várias funções e aplique-as sequencialmente.
  2. Crie uma função que use reduce para agrupar itens em um array com base em uma propriedade.
  3. Escreva uma função que simule a funcionalidade de map e filter combinadas.
  4. Crie uma função que receba uma lista de funções e retorne uma função que execute todas em sequência.
  5. Implemente uma versão personalizada de debounce e throttle.

Funções Assíncronas e Promises

  1. Crie uma função que implemente uma fila de tarefas assíncronas com limite de execução simultânea.
  2. Escreva uma função que execute um conjunto de promessas com um intervalo entre elas.
  3. Implemente uma função que reintente uma promessa um número específico de vezes em caso de falha.
  4. Crie uma função que combine os resultados de múltiplas promessas resolvidas em um único objeto.
  5. Desenvolva uma função que use async/await para rastrear o tempo de execução de operações assíncronas.

Padrões de Design e Conceitos Avançados

  1. Implemente o padrão de design *Observer* usando classes ES6.
  2. Crie um módulo Singleton em JavaScript para gerenciar configurações globais.
  3. Desenvolva um sistema de eventos customizado usando o padrão *Publisher/Subscriber*.
  4. Crie uma função *memoize* que otimize o desempenho de cálculos repetitivos.
  5. Implemente uma função *currying* que transforme uma função de múltiplos argumentos em várias funções de um único argumento.

Desafios

  1. Crie um interpretador básico que avalie expressões matemáticas simples como "2 + 3 * 4 - 5".
  2. Implemente um algoritmo que compare dois objetos profundamente e determine se eles são iguais.
  3. Desenvolva uma função que use *proxy* para criar propriedades dinâmicas em objetos.
  4. Crie um gerador de números aleatórios que permita especificar uma semente inicial para gerar resultados previsíveis.
  5. Implemente um sistema de roteamento básico para um aplicativo de página única (SPA).

Desafios de Manipulação de Dados

  • Utilize Lodash para agrupar uma lista de objetos por uma propriedade específica (por exemplo, agrupar produtos por categoria).
  • Implemente uma função que use Lodash para encontrar a diferença entre duas listas e retornar os elementos únicos de cada uma.
  • Crie uma função usando Lodash para combinar e mesclar múltiplos objetos em um único objeto, mantendo a estrutura e valores corretos.
  • Use Lodash para ordenar uma lista de objetos por uma propriedade numérica e, em seguida, divida os objetos em grupos de tamanho específico.
  • Implemente uma função que utilize Lodash para inverter a chave e o valor de um objeto.
  • Crie uma função que use Lodash para filtrar uma lista de objetos com base em múltiplas condições complexas.
  • Use Lodash para calcular a soma dos valores de uma propriedade em um array de objetos e retorne o resultado.
  • Implemente uma função que use Lodash para encontrar o elemento mais frequente em um array de dados.
  • Crie uma função que use Lodash para obter a chave de um objeto que possui o maior valor.
  • Utilize Lodash para criar uma função que remova elementos duplicados de um array de objetos com base em uma chave específica.
  • Implemente uma função que use Lodash para combinar duas listas de objetos de forma que elementos iguais sejam fundidos em um único item.
  • Crie uma função que use Lodash para criar uma cópia profunda de um objeto e modifique o original sem afetar a cópia.
  • Use Lodash para criar uma função que percorre um objeto e retorna todas as chaves com valores que atendem a uma condição específica.
  • Crie uma função que utilize Lodash para criar uma lista de objetos ordenada por um valor calculado (por exemplo, preço com desconto).
  • Implemente uma função que use Lodash para dividir um array em partes iguais e retorne um array de subarrays.

Desafios de Manipulação de Dados com Funções Personalizadas

  • Implemente uma função que faça a união de dois arrays de objetos sem duplicar elementos, comparando uma chave específica.

  • Implemente uma função que use `reduce` para calcular a média dos valores de uma propriedade em um array de objetos.
  • Crie uma função que use `forEach` para iterar por um array de objetos e atualizar uma propriedade com base em uma condição.
  • Implemente uma função que use `sort` para ordenar um array de objetos por múltiplas chaves, com critérios de ordenação diferentes.
  • Crie uma função que use `splice` para remover elementos de um array de objetos e retornar os elementos removidos em um array separado.
  • Implemente uma função que use `concat` para combinar várias listas de objetos em uma única lista, mantendo a ordem original.
  • Crie uma função que use `find` para buscar um objeto em um array com base em uma condição complexa e retorne o objeto encontrado.
  • Implemente uma função que use `map` para criar um novo array de objetos com base em uma transformação específica de uma propriedade.
  • Crie uma função que use `filter` para retornar um array com objetos que contêm uma subpropriedade específica com um valor específico.
  • Implemente uma função que use `reduce` para contar quantas vezes uma propriedade específica aparece em um array de objetos.
  • Crie uma função que use `every` e `some` para validar se todas ou algumas entradas em um array de objetos atendem a uma condição específica.
  • Implemente uma função que use `map` e `reduce` em conjunto para criar um objeto que mapeia as chaves de uma lista de objetos com valores agregados.
  • Crie uma função que use `reverse` para inverter a ordem dos elementos em um array de objetos e retorne o array modificado.
  • Implemente uma função que use `slice` para retornar uma cópia parcial de um array de objetos a partir de um índice específico.

  • Crie uma função que utilize `Object.values()` e `reduce` para encontrar a soma de todos os valores em um objeto.
  • Implemente uma função que use `Object.entries()` para criar um array de pares chave-valor e filtrar apenas os pares com valores numéricos.
  • Crie uma função que use `Object.assign()` para fazer a mesclagem de dois objetos, com a possibilidade de sobrescrever valores.
