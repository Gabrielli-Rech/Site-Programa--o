Exercícios de Navegação

  • Abra o VSCode e navegue para a barra lateral esquerda. Identifique as seções "Explorer", "Source Control" e "Extensions".
  • Use o atalho Ctrl + P (Windows) ou Cmd + P (Mac) para pesquisar e abrir rapidamente um arquivo.
  • Abra o painel "Explorer" e crie uma nova pasta e um novo arquivo dentro dessa pasta.
  • Explore a barra lateral de "Source Control" e faça um commit de um arquivo modificado.
  • Use o atalho Ctrl + ` (Windows) ou Cmd + ` (Mac) para abrir o terminal integrado.
  • Personalize a aparência do VSCode alterando o tema de cores na configuração do editor.
  • Pesquise e instale uma extensão no VSCode usando o painel de extensões.
  • Use a barra de navegação lateral para alternar entre diferentes áreas de trabalho, como "Explorer", "Source Control", "Extensions".
  • Abra a paleta de comandos com Ctrl + Shift + P (Windows) ou Cmd + Shift + P (Mac) e execute um comando como "Format Document".
  • Explore as configurações do VSCode e altere o valor da fonte do editor de código.

Exercícios de Interface

  • Personalize a barra de atividades e adicione ou remova ícones de acordo com sua preferência.
  • Utilize a funcionalidade "Split Editor" para dividir a tela do editor em duas partes e edite arquivos simultaneamente.
  • Use o atalho Ctrl + B (Windows) ou Cmd + B (Mac) para mostrar ou ocultar a barra lateral.
  • Modifique a barra de status do VSCode para mostrar informações adicionais, como a linguagem de programação atual.
  • Use o "Command Palette" para encontrar e executar rapidamente comandos do VSCode, como "Toggle Terminal".
  • Personalize a barra de status para incluir informações sobre o branch atual em um projeto Git.
  • Habilite a opção de "Zen Mode" para uma experiência de codificação sem distrações.
  • Use a barra de pesquisa para localizar e substituir palavras em um arquivo específico.
  • Abra a interface de "Problems" para identificar e corrigir problemas de código no arquivo atual.
  • Configure o VSCode para usar um tema de contraste alto e explore como ele altera a interface.
