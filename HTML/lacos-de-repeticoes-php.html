O que são Laços de Repetição?

Laços de repetição são estruturas de controle que permitem a execução de um bloco de código múltiplas vezes, enquanto uma condição for verdadeira. Em PHP, você pode usar diferentes tipos de laços para controlar a repetição de blocos de código.

Laço while

O laço while executa um bloco de código enquanto a condição especificada for verdadeira.

$contador = 0;
while ($contador < 5) {
    echo "Contador é $contador
";
    $contador++;
}

Se a condição for falsa na primeira verificação, o bloco de código não será executado.

Laço do-while

O laço do-while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez antes de verificar a condição.

$contador = 0;
do {
    echo "Contador é $contador
";
    $contador++;
} while ($contador < 5);

Mesmo que a condição seja falsa, o bloco de código é executado pelo menos uma vez.

Laço for

O laço for é usado para iterar um número específico de vezes, facilitando a contagem e a repetição do bloco de código.

for ($i = 0; $i < 5; $i++) {
    echo "Valor de i: $i
";
}

Esse loop percorre todos os valores de 0 a 4, inclusive.

Laço foreach

O laço foreach é usado para iterar sobre arrays e objetos de forma simples e direta.

$frutas = array("Maçã", "Banana", "Cereja");
foreach ($frutas as $fruta) {
    echo "Fruta: $fruta
";
}

O loop percorre cada elemento do array e o armazena na variável temporária especificada.

Comandos break e continue

Os comandos break e continue podem ser usados para controlar a execução dos laços.

for ($i = 0; $i < 10; $i++) {
    if ($i == 5) {
        break; // Interrompe o loop quando $i for igual a 5
    }
    if ($i % 2 == 0) {
        continue; // Pula a iteração se $i for par
    }
    echo "Valor ímpar: $i
";
}

break interrompe o loop quando a condição é atendida, e continue pula a iteração atual quando a condição é verdadeira.
