O que é MongoDB?

MongoDB é um banco de dados NoSQL de código aberto, orientado a documentos. Ele armazena dados em um formato de documento JSON-like, conhecido como BSON, o que permite uma estrutura de dados flexível e escalável.

Características e Conceitos Básicos

O MongoDB é conhecido por sua escalabilidade horizontal, alta disponibilidade e facilidade de uso. Ele suporta grandes volumes de dados e permite consultas rápidas e poderosas.

Exemplo de Consulta

db.usuarios.find({ idade: { $gt: 18 } });
