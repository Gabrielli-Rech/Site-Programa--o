O que é PostgreSQL?

O PostgreSQL é um sistema de gerenciamento de banco de dados relacional (SGBD) avançado, conhecido por sua robustez, desempenho e conformidade com os padrões SQL.

Veja abaixo os principais comandos SQL usados no PostgreSQL para criar, consultar e manipular bancos de dados.

Comandos Básicos

  • Criar Banco de Dados: 
    CREATE DATABASE nome_do_banco;
  • Conectar ao Banco: 
    \c nome_do_banco
  • Criar Tabela: 
    CREATE TABLE nome_tabela (
    id SERIAL PRIMARY KEY,
    nome VARCHAR(100),
    idade INT
);
  • Inserir Dados: 
    INSERT INTO nome_tabela (nome, idade)
VALUES ('Ana', 25);
  • Consultar Dados: 
    SELECT * FROM nome_tabela;

Comandos de Manipulação de Dados

  • Atualizar Dados: 
    UPDATE nome_tabela
SET idade = 26
WHERE nome = 'Ana';
  • Excluir Dados: 
    DELETE FROM nome_tabela
WHERE nome = 'Ana';
  • Adicionar Coluna: 
    ALTER TABLE nome_tabela
ADD COLUMN email VARCHAR(100);
  • Excluir Tabela: 
    DROP TABLE nome_tabela;

Comandos de Consultas Avançadas

  • Filtrar Dados: 
    SELECT * FROM nome_tabela
WHERE idade > 20;
  • Ordenar Dados: 
    SELECT * FROM nome_tabela
ORDER BY idade DESC;
  • Contar Registros: 
    SELECT COUNT(*) FROM nome_tabela;
  • Juntar Tabelas: 
    SELECT a.nome, b.nome_projeto
FROM tabela_a a
JOIN tabela_b b ON a.id = b.id_usuario;
