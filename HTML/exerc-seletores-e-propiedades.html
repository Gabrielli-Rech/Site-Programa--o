Exercícios Práticos

  • Crie um seletor que aplique uma cor de fundo azul e uma cor de texto branca a todos os parágrafos.
  • Use um seletor de classe para estilizar um botão e adicionar uma sombra ao passar o mouse sobre ele.
  • Crie um seletor de ID que altere a fonte de um título para uma fonte personalizada e adicione um sublinhado.
  • Use um seletor de atributo para estilizar links que contêm o texto "Novo" em azul e em negrito.
  • Estilize todos os elementos div que têm uma classe específica e adicione uma borda de 2px de cor preta.
  • Crie um seletor de pseudo-classe que altere a cor de um botão para vermelho quando ele for pressionado.
  • Use um seletor de pseudo-elemento para adicionar um ícone antes de todos os itens de uma lista.
  • Crie um seletor de filho direto (>) para estilizar elementos li dentro de um ul com uma cor de fundo cinza claro.
  • Utilize um seletor de descendente para aplicar um espaçamento entre elementos div e p dentro de um contêiner.
  • Crie um seletor de pseudo-classe para exibir um botão em uma cor diferente quando o mouse passa sobre ele (:hover).
  • Utilize um seletor de grupo para estilizar diferentes elementos (como h1, h2 e p) com a mesma cor de texto.
  • Crie um seletor de pseudo-classe :nth-child para estilizar o terceiro item de uma lista com uma cor de fundo diferente.
  • Estilize todas as imagens dentro de um contêiner div para que elas tenham uma borda arredondada e uma sombra suave.
  • Use um seletor de pseudo-classe :not para aplicar um estilo especial a todos os elementos p que não possuem uma classe específica.
  • Crie um seletor de pseudo-elemento para adicionar um conteúdo adicional após o texto de um parágrafo com o uso de :after.
  • Estilize um menu de navegação com seletores para alterar a cor de fundo do item ativo e a cor do texto.
  • Utilize um seletor de atributo para estilizar todos os links com um atributo href que contém "https".
  • Use um seletor de pseudo-classe :checked para estilizar um checkbox selecionado com uma cor de fundo diferente.
  • Crie um seletor de pseudo-elemento para mostrar um ícone de aviso em um parágrafo que possui uma classe de "aviso".
  • Use um seletor de grupo para estilizar elementos h1, h2 e h3 com a mesma margem.
  • Crie um seletor que estilize todos os elementos input do tipo texto com uma borda e um preenchimento personalizados.
