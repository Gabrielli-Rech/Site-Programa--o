Exercícios de Programação Assíncrona

  • Crie um script Ruby que utilize a gem `async` para executar duas operações simultaneamente e aguarde a conclusão de ambas.
  • Desenvolva um programa que execute várias tarefas assíncronas usando `Concurrent::Future` para medir o tempo de execução total.
  • Implemente uma função assíncrona que faça requisições HTTP em paralelo usando a gem `Typhoeus`.
  • Crie um script que use `Thread` para ler e processar vários arquivos de texto simultaneamente e depois combine os resultados.
  • Escreva um programa em Ruby que use a gem `EventMachine` para criar um servidor TCP assíncrono que aceite conexões de clientes e envie uma resposta de volta.
  • Desenvolva um script Ruby com `Async::Task` que simule a execução de uma tarefa demorada e outra tarefa de monitoramento em paralelo.
  • Implemente uma função assíncrona que leia de um banco de dados e retorne os resultados sem bloquear a execução de outras operações.
  • Crie um script que faça múltiplas consultas a uma API externa de forma assíncrona e colete os dados em um array.
  • Desenvolva um script que utilize `Promise` para encadear várias operações assíncronas e retorne um resultado quando todas forem concluídas.
  • Escreva um programa Ruby que use `Fiber` para gerenciar a execução de várias funções assíncronas e alternar entre elas de maneira eficiente.
  • Crie um serviço em Ruby que utilize `Async::HTTP` para fazer a chamada de um endpoint de uma API e tratar o resultado de forma assíncrona.
  • Desenvolva uma função em Ruby que use `Semaphore` para limitar o número de threads executadas simultaneamente em um bloco de código assíncrono.
  • Implemente um programa que use `Concurrent::Promise` para processar uma lista de tarefas em paralelo e retorne o tempo total de execução.
  • Crie um script que execute uma tarefa de leitura de arquivo de forma assíncrona usando a gem `async` e imprima o resultado no console.
  • Desenvolva uma aplicação que use `Async::HTTP` para fazer chamadas simultâneas a diferentes URLs e exiba os resultados quando todas as respostas forem recebidas.
  • Escreva um programa em Ruby que use `Thread` e `Queue` para processar dados de uma fila de forma assíncrona.
  • Implemente uma função em Ruby que utilize `EventMachine` para criar um cliente UDP que envie e receba mensagens de forma assíncrona.
  • Crie um script que use a gem `Sidekiq` para executar tarefas assíncronas em segundo plano e monitore o status da execução.
  • Desenvolva uma aplicação em Ruby que utilize a gem `celluloid` para criar e gerenciar tarefas assíncronas com actors.
  • Escreva um programa que use `Async` e `IO` para ler e processar múltiplos arquivos em paralelo sem bloqueio.
