Exercícios Fáceis

  • 1. Escreva um programa que imprima os números de 1 a 10.
  • 2. Crie um script que imprima os números pares de 1 a 20.
  • 3. Implemente um programa que imprima a tabuada de 2.
  • 4. Escreva um programa que imprima todos os números ímpares de 1 a 15.
  • 5. Crie um script que imprima os números de 1 a 50, mas substitua múltiplos de 5 por "Buzz".
  • 6. Implemente um programa que leia um número e imprima sua tabuada.
  • 7. Escreva um programa que conte de 10 a 1 e depois imprima "Feliz Ano Novo!".
  • 8. Crie um script que imprima todos os números múltiplos de 3 de 1 a 30.
  • 9. Escreva um programa que imprima a soma dos números de 1 a 100.
  • 10. Implemente um script que leia um número e imprima "Número par" ou "Número ímpar".
  • 11. Crie um programa que imprima os primeiros 10 termos da sequência de Fibonacci.
  • 12. Escreva um script que imprima os quadrados dos números de 1 a 10.
  • 13. Implemente um programa que imprima a soma dos quadrados dos números de 1 a 5.
  • 14. Crie um script que leia um número e imprima "Contagem de 1 a 10" seguido de cada número até 10.
  • 15. Escreva um programa que use um laço para imprimir os elementos de uma lista de 1 a 5.
  • 16. Implemente um programa que leia um número e imprima a soma de todos os números de 1 até esse número.
  • 17. Crie um script que use um laço para imprimir uma contagem regressiva de 10 a 1.
  • 18. Escreva um programa que imprima uma lista de números de 1 a 20, substituindo múltiplos de 4 por "Quadro".
  • 19. Implemente um programa que leia um número e imprima os divisores desse número.
  • 20. Crie um script que imprima a tabuada de 5 usando um laço.

Exercícios Médios

  • 1. Crie um programa que leia um número e imprima se ele é primo ou não.
  • 2. Implemente um script que leia um número e imprima a soma dos números de 1 até o número lido.
  • 3. Escreva um programa que imprima todos os números de 1 a 100 que são divisíveis por 3 e 5.
  • 4. Crie um script que leia uma lista de números e imprima a soma de todos os números pares.
  • 5. Implemente um programa que leia uma string e imprima sua inversão.
  • 6. Escreva um script que imprima todos os números de 1 a 100 e substitua múltiplos de 7 por "Sete".
  • 7. Crie um programa que use um laço para imprimir a tabuada de 1 a 10.
  • 8. Implemente um script que leia um número e imprima a sequência de Fibonacci até esse número.
  • 9. Escreva um programa que leia um número e imprima todos os divisores desse número.
  • 10. Crie um script que imprima os quadrados dos números de 10 a 20.
  • 11. Implemente um programa que imprima a soma dos elementos de uma lista de números.
  • 12. Escreva um script que calcule a média dos números de uma lista.
  • 13. Crie um programa que leia um número e imprima a série de números Fibonacci até esse número.
  • 14. Implemente um script que conte quantas vezes a letra "a" aparece em uma frase dada pelo usuário.
  • 15. Escreva um programa que imprima todos os números de 1 a 100 e, para múltiplos de 3, imprima "Fizz"; para múltiplos de 5, "Buzz".
  • 16. Crie um script que leia um número e imprima "Número primo" se for primo, caso contrário "Não é primo".
  • 17. Implemente um programa que imprima a soma dos elementos de uma lista de maneira recursiva.
  • 18. Escreva um script que leia uma lista de números e imprima somente os números que são maiores que 10.
  • 19. Crie um programa que use um laço para verificar se um número é perfeito (soma de seus divisores igual ao número).
  • 20. Implemente um script que imprima os números de 100 a 1 de forma decrescente com um "!" após cada número.

Exercícios Difíceis

  • 1. Crie um programa que imprima a soma dos números de uma lista, mas somente os números que são primos.
  • 2. Implemente um script que encontre o menor múltiplo de 7 maior que 100.
  • 3. Escreva um programa que calcule a soma dos elementos de uma lista de forma reversa.
  • 4. Crie um script que leia um número e imprima sua decomposição em fatores primos.
  • 5. Implemente um programa que calcule o produto dos elementos de uma lista de números usando laços.
  • 6. Escreva um script que calcule o fatorial de um número usando laço de repetição.
  • 7. Crie um programa que verifique se um número é um número Armstrong (exemplo: 153).
  • 8. Implemente um script que imprima todos os números que são múltiplos de 2 e 3 ao mesmo tempo até 100.
  • 9. Escreva um programa que leia uma string e imprima a quantidade de vezes que cada caractere aparece.
  • 10. Crie um script que encontre a sequência de números primos em uma lista de 1 a 100.
  • 11. Implemente um programa que leia um número e imprima a sequência de números de 1 até esse número em ordem reversa.
  • 12. Escreva um script que imprima o menor e o maior número de uma lista de números.
  • 13. Crie um programa que imprima a soma dos números em uma lista, mas exclua números negativos.
  • 14. Implemente um script que leia um número e imprima sua representação em binário usando laço de repetição.
  • 15. Escreva um programa que use um laço para imprimir o maior número na sequência de Fibonacci menor que 1000.
  • 16. Crie um script que imprima todos os números de uma lista que são divisíveis por um número dado.
  • 17. Implemente um programa que conte quantas palavras diferentes existem em uma frase dada pelo usuário.
  • 18. Escreva um script que calcule a média dos números de uma lista, mas só considere números maiores que 10.
  • 19. Crie um programa que imprima todos os números de 1 a 100 que não são divisíveis por 2, 3 ou 5.
  • 20. Implemente um script que encontre todos os números de uma lista que são perfeitos e imprima-os.

Desafios

  • 1. Crie um programa que implemente a lógica de um jogo de adivinhação de números com laços de repetição.

  • 3. Escreva um programa que calcule a soma dos elementos de uma lista usando um laço recursivo.

  • 5. Implemente um script que imprima uma pirâmide de números de 1 a 10 usando laços de repetição.
  • 6. Escreva um programa que leia um número e imprima a tabuada de 1 a esse número em ordem inversa.
  • 7. Crie um script que leia um texto e encontre a palavra mais longa usando laços.
  • 8. Implemente um programa que calcule a soma de todos os elementos de uma lista que são números primos.
  • 9. Escreva um script que imprima todos os números de 1 a 100 que são palíndromos.
  • 10. Crie um programa que leia um número e imprima a sequência de Collatz até o número 1.
  • 11. Implemente um script que use um laço para simular o lançamento de um dado até sair um número específico.
  • 12. Escreva um programa que leia uma lista de números e os imprima em ordem crescente sem usar funções de ordenação.
  • 13. Crie um script que leia uma string e imprima todas as substrings possíveis.
  • 14. Implemente um programa que leia um número e calcule a soma dos números pares até esse número.
  • 15. Escreva um script que leia um número e imprima um triângulo de números até esse número.
  • 16. Crie um programa que leia uma lista e imprima a soma dos elementos em índices pares.
  • 17. Implemente um script que use laços de repetição para encontrar a soma de uma progressão aritmética.
  • 18. Escreva um programa que leia uma lista de palavras e imprima a quantidade de palavras que possuem mais de 5 caracteres.
  • 19. Crie um script que imprima a sequência de números primos em um intervalo dado pelo usuário.
  • 20. Implemente um programa que imprima a soma de todos os elementos de uma lista em ordem inversa.
