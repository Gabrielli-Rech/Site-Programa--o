Exercícios Layouts e Responsividade

Exercícios Fáceis

  • Crie uma página com um layout de coluna única usando Flexbox.
  • Crie uma página com um layout de duas colunas usando o modelo de grid CSS.
  • Adicione um cabeçalho e um rodapé fixos a uma página com conteúdo centralizado.
  • Desenvolva um layout responsivo simples que mude de coluna única para duas colunas em telas maiores.
  • Crie uma barra de navegação responsiva que se reorganize para um menu hambúrguer em telas menores.
  • Use `flex-wrap` para criar um layout de cartão em uma página que se reorganize conforme o tamanho da tela.
  • Desenvolva um layout de três colunas usando CSS Grid e faça com que as colunas se empilhem em telas menores.
  • Utilize `min-width` e `max-width` para definir a largura mínima e máxima de um contêiner responsivo.
  • Crie um layout de galeria de imagens com grid CSS e adicione espaçamento entre os itens.
  • Implemente uma tabela responsiva usando a técnica de "scroll horizontal" em telas pequenas.

Exercícios Médios

  • Crie uma página com um layout de flexbox que inclua uma barra lateral e um conteúdo principal, ajustando as proporções em diferentes tamanhos de tela.
  • Adicione um menu de navegação horizontal que se torne vertical em telas menores usando `@media` queries.
  • Use o layout de grid para criar um formulário responsivo com campos de entrada e rótulos alinhados.
  • Desenvolva um layout de cabeçalho com três seções (logo, menu e barra de pesquisa) que se ajusta em telas menores.
  • Implemente uma imagem de fundo responsiva que se ajuste automaticamente a diferentes tamanhos de tela.
  • Crie uma página com um layout de "cards" que se reconfigure em colunas de uma única coluna em telas menores.
  • Desenvolva um layout de página com um cabeçalho fixo e um rodapé fixo que se adapta ao conteúdo da página.
  • Use `flex-direction` e `flex-wrap` para criar um layout de galeria de fotos que se reorganize em uma coluna em dispositivos móveis.
  • Implemente um layout de grid com áreas nomeadas e utilize `grid-template-areas` para definir a estrutura do layout.
  • Adicione uma animação de transição de layout para uma página que muda entre um layout de uma coluna e duas colunas.

Exercícios Difíceis

  • Crie um layout de página de blog com um cabeçalho fixo, uma barra lateral e uma área de conteúdo principal que se reorganize em um layout de coluna única em telas menores.
  • Implemente um layout de "cards" com diferentes tamanhos que se ajustam dinamicamente em telas menores.
  • Desenvolva um layout responsivo de e-commerce com uma grade de produtos que ajusta a quantidade de colunas de acordo com a largura da tela.
  • Crie uma página de portfólio com uma galeria de imagens responsiva e um efeito de zoom ao passar o mouse.
  • Implemente um layout de dashboard com múltiplas seções que se reorganizam em um layout de coluna única em dispositivos móveis.
  • Adicione um efeito de parallax em um layout responsivo usando `background-attachment` e `background-position`.
  • Desenvolva um layout de página com um cabeçalho fixo e um conteúdo que é ajustado automaticamente de acordo com a altura da tela.
  • Crie um layout que utilize a técnica de "mobile-first" para garantir que o design funcione em dispositivos menores antes de ser adaptado para telas maiores.
  • Implemente um layout responsivo com um sistema de grid de 12 colunas e diferentes tamanhos de colunas para diferentes pontos de quebra.
  • Desenvolva uma página responsiva que incorpora a técnica de "flexbox gaps" para criar espaçamento entre os elementos do layout.

Desafios

  • Desenvolva um layout responsivo com múltiplas seções de conteúdo que se ajustam automaticamente em tamanhos de tela diferentes, utilizando `flex` e `grid` em conjunto.
  • Crie um layout de página com uma navegação lateral que se transforma em uma barra de navegação no topo em telas menores.

  • Desenvolva um layout de blog responsivo com imagens de destaque e uma lista de postagens que se reorganiza em uma única coluna em dispositivos móveis.
  • Crie um layout de página de destino com um cabeçalho, uma seção de destaque e uma lista de recursos que se ajusta em dispositivos menores.
  • Implemente um sistema de grid complexo que permita que colunas e linhas sejam combinadas dinamicamente em dispositivos menores.
  • Desenvolva um layout de formulário complexo com vários campos que se reorganiza em uma única coluna em telas menores.
  • Crie um layout de site de notícias com uma grade de artigos que se reorganiza em uma única coluna em dispositivos menores e inclui um efeito de rolagem suave.
