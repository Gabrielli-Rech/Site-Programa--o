Exercícios Práticos

  • Crie uma variável do tipo string chamada nome e atribua seu nome a ela. Imprima na tela.
  • Crie uma variável idade do tipo number e atribua sua idade. Imprima a idade com uma mensagem, como "Eu tenho X anos".
  • Crie uma variável altura do tipo number e atribua um valor de ponto flutuante. Imprima a altura com uma mensagem formatada, como "Minha altura é X metros".
  • Crie uma variável isEstudante do tipo boolean e atribua um valor true ou false. Imprima a variável com uma mensagem, como "Sou estudante?"
  • Crie uma variável frutas do tipo string[] com pelo menos três elementos (por exemplo: "maçã", "banana", "laranja"). Imprima todos os elementos usando um loop.
  • Crie um array de números numeros do tipo number[] e calcule a soma de todos os elementos usando um loop.
  • Crie uma variável dataAtual do tipo Date e imprima a data atual formatada.
  • Crie uma variável objeto do tipo object que contenha uma chave nome do tipo string e uma chave idade do tipo number. Imprima o objeto completo e suas propriedades separadas.
  • Crie uma variável valor do tipo any e atribua a ela diferentes tipos de dados (string, number, boolean). Imprima o tipo de dado usando typeof.
  • Crie uma tupla ponto do tipo [number, number] e atribua valores para representar um ponto em um plano cartesiano. Imprima os valores da tupla.
  • Crie uma variável nulo do tipo null e imprima seu valor.
  • Crie uma variável indefinido do tipo undefined e imprima seu valor.
  • Crie uma variável numeroGrande do tipo bigint e imprima seu valor.
  • Crie um enum chamado DiaSemana com os valores "Segunda", "Terça", "Quarta", etc., e imprima o nome do dia usando um valor do enum.
  • Crie uma variável nomeCompleto do tipo string e use interpolação de strings para imprimir uma frase como "Meu nome é X e eu tenho Y anos".
  • Crie uma função que recebe um parâmetro do tipo number e retorna o dobro desse número. Teste a função com diferentes valores.
  • Crie uma função que recebe um parâmetro do tipo string e imprime a string em letras maiúsculas usando .toUpperCase().
  • Crie uma variável mensagem do tipo string e use o método split() para dividir a string em um array de palavras.
  • Crie uma variável valor e use o método isNaN() para verificar se o valor é um número.
  • Crie uma função que recebe um array de strings como parâmetro e retorna a quantidade de elementos que possuem mais de 3 caracteres.
  • Crie uma variável isValid do tipo boolean e use um operador ternário para atribuir um valor de "Válido" ou "Inválido" com base em uma condição.
  • Crie uma variável nome e use o método replace() para substituir uma parte da string.
  • Crie uma variável numero e use Math.floor() e Math.ceil() para arredondar o número para baixo e para cima, respectivamente.
  • Crie uma variável isTrue e use um operador de negação ! para inverter seu valor.
  • Crie um array frutas e use o método forEach() para imprimir cada elemento com uma mensagem.
  • Crie uma variável nome e use o método charAt() para imprimir o primeiro caractere da string.
  • Crie uma variável telefone e use o método substring() para extrair um número de telefone de uma string.
