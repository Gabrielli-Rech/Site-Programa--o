Exercícios

  • 1. Criar um banco de dados chamado escola.
  • 2. Criar uma tabela chamada alunos com as colunas id (inteiro), nome (texto) e idade (inteiro).
  • 3. Inserir 5 registros de alunos na tabela alunos.
  • 4. Selecionar todos os registros da tabela alunos.
  • 5. Selecionar apenas os nomes dos alunos da tabela alunos.
  • 6. Atualizar a idade de um aluno específico com id 3 para 18.
  • 7. Deletar um registro da tabela alunos onde o id seja 2.
  • 8. Selecionar os alunos cuja idade seja maior que 15.
  • 9. Contar o número de registros na tabela alunos.
  • 10. Selecionar todos os alunos ordenados por nome em ordem crescente.
  • 11. Selecionar todos os alunos com idade entre 15 e 20 anos.
  • 12. Criar uma nova tabela chamada cursos com as colunas id (inteiro), nome (texto) e duracao (inteiro).
  • 13. Inserir 3 cursos na tabela cursos.
  • 14. Selecionar todos os cursos que têm duração maior que 30 dias.
  • 15. Criar uma relação entre as tabelas alunos e cursos usando uma tabela de junção chamada matriculas com colunas aluno_id e curso_id.
  • 16. Inserir 2 registros na tabela matriculas para associar alunos a cursos.
  • 17. Selecionar os nomes dos alunos que estão matriculados em um curso específico.
  • 18. Contar o número de alunos matriculados em cada curso.
  • 19. Alterar a tabela alunos para adicionar uma coluna email (texto).
  • 20. Selecionar os alunos que possuem o email terminado com "@gmail.com".
