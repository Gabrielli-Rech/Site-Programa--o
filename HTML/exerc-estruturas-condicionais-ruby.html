Exercícios de Estruturas Condicionais

Exercícios Fáceis

  • Escreva um programa que verifica se um número é par ou ímpar.
  • Crie um script que verifica se um número é positivo, negativo ou zero.
  • Implemente um programa que determina se um ano é bissexto.
  • Escreva um programa que verifica se a idade de uma pessoa é suficiente para votar.
  • Faça um programa que verifica se um número está entre 10 e 20.
  • Escreva um script que imprime "Bom dia", "Boa tarde" ou "Boa noite" com base em uma hora fornecida.
  • Crie um programa que verifica se duas strings são iguais.
  • Escreva um programa que determina o maior de dois números fornecidos pelo usuário.
  • Faça um script que verifica se um nome começa com uma letra específica.
  • Escreva um programa que verifica se um número é divisível por 5.

Exercícios Médios

  • Crie um script que verifica se três números podem formar um triângulo válido.
  • Escreva um programa que determina o maior de três números fornecidos.
  • Implemente um script que classifica uma nota em "Aprovado", "Recuperação" ou "Reprovado".
  • Crie um programa que verifica se um número está dentro de um intervalo especificado pelo usuário.
  • Escreva um script que determina se uma string é um palíndromo.
  • Implemente um programa que valida uma senha com pelo menos 8 caracteres.
  • Faça um script que verifica se uma pessoa pode dirigir com base em sua idade e se possui habilitação.
  • Escreva um programa que verifica se uma variável está vazia ou nula.
  • Crie um script que valida o formato de um endereço de e-mail.
  • Escreva um programa que calcula descontos com base no valor de um produto e sua categoria.

Exercícios Difíceis

  • Implemente um programa que calcula o imposto de renda com base em faixas salariais.
  • Crie um script que valida um CPF (apenas o número de dígitos).
  • Escreva um programa que verifica se uma data fornecida está no passado, presente ou futuro.
  • Faça um script que simula um sistema de login com três tentativas antes de bloquear o usuário.
  • Implemente um programa que determina se uma senha é forte (letras maiúsculas, minúsculas, números e símbolos).
  • Crie um script que classifica números em pares, ímpares, positivos e negativos.
  • Escreva um programa que valida uma data no formato "dd/mm/aaaa".
  • Faça um script que determina se duas strings são anagramas.
  • Implemente um programa que calcula o fatorial de um número apenas se for positivo.
  • Crie um script que avalia se um conjunto de tarefas foi concluído (todas marcadas como "completadas").

Desafios

  • Implemente um sistema de autenticação com diferentes níveis de acesso (usuário, moderador, administrador).
  • Crie um script que verifica se uma expressão matemática está balanceada em relação aos parênteses.
  • Escreva um programa que simula um caixa eletrônico, determinando a quantidade de notas necessárias para um saque.
  • Faça um script que analisa um campeonato de futebol e determina o time vencedor com base nos pontos.
  • Implemente um programa que classifica um número como primo ou não.
  • Crie um script que simula um jogo de "pedra, papel e tesoura" entre dois jogadores.
  • Escreva um programa que calcula o Índice de Massa Corporal (IMC) e classifica o resultado.
  • Faça um script que determina se um horário fornecido está dentro do expediente (ex: 9h às 18h).
  • Implemente um programa que valida um número de cartão de crédito usando o algoritmo de Luhn.
  • Crie um script que verifica se uma palavra é um heterograma (todas as letras são diferentes).
