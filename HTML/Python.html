O que é Python?

Python é uma linguagem de programação de alto nível, de fácil leitura e escrita. É uma das linguagens mais populares do mundo, amplamente usada para desenvolvimento web, automação, análise de dados, inteligência artificial, aprendizado de máquina e muito mais. Python é conhecida por sua sintaxe limpa e concisa, o que a torna ideal para iniciantes e desenvolvedores experientes.

Principais Características do Python

  • Sintaxe Simples: Fácil de ler e escrever, tornando o código mais legível.
  • Multiplataforma: Funciona em diversos sistemas operacionais, como Windows, macOS e Linux.
  • Versatilidade: Usado em diversas áreas, como desenvolvimento web (Django, Flask), ciência de dados (Pandas, NumPy), e IA (TensorFlow, scikit-learn).
  • Grande Comunidade: Uma comunidade ativa que contribui com bibliotecas e frameworks para facilitar o desenvolvimento.
  • Interpretação: Python é uma linguagem interpretada, o que significa que você pode testar e executar o código rapidamente.

Sintaxe Básica

Exemplo de Declaração de Variáveis

# Declaração de variáveis
nome = "João"  # String
idade = 30  # Inteiro
altura = 1.75  # Float

Exemplo de Estruturas de Controle

# Estrutura de controle condicional
if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

# Estrutura de repetição
for i in range(5):
    print(f"Número: {i}")

Funções Avançadas

Funções Anônimas (Lambda)

# Função lambda para somar dois números
soma = lambda a, b: a + b
print(soma(5, 10))  # Saída: 15

Funções de Ordem Superior

# Função de ordem superior que recebe outra função como argumento
def aplicar_operacao(a, b, operacao):
    return operacao(a, b)

def multiplicar(x, y):
    return x * y

print(aplicar_operacao(5, 3, multiplicar))  # Saída: 15

Manipulação de Dados e Listas

Python é poderoso para manipulação de dados, graças a suas listas, dicionários e outras estruturas de dados integradas.

Exemplo de Listas e Compreensão de Listas

# Lista e iteração
frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(fruta)

# Compreensão de listas
numeros = [x for x in range(10) if x % 2 == 0]
print(numeros)  # Saída: [0, 2, 4, 6, 8]

Programação Orientada a Objetos (POO)

Python suporta programação orientada a objetos, permitindo a criação de classes e objetos para modelar e organizar seu código de maneira eficiente.

Exemplo de Classe e Objeto

# Definição de classe e criação de objeto
class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade
    
    def saudacao(self):
        return f"Olá, meu nome é {self.nome} e tenho {self.idade} anos."

pessoa1 = Pessoa("Carlos", 25)
print(pessoa1.saudacao())  # Saída: Olá, meu nome é Carlos e tenho 25 anos.
