Exercícios Desafios e Projetos Práticos

Desafios Fáceis

  • 1. Criar um banco de dados e uma tabela simples.
  • 2. Inserir registros em uma tabela e exibi-los com um comando SELECT.
  • 3. Criar uma consulta para filtrar dados com a cláusula WHERE.
  • 4. Ordenar dados de uma tabela usando a cláusula ORDER BY.
  • 5. Atualizar valores de uma tabela com o comando UPDATE.
  • 6. Deletar registros de uma tabela com a cláusula DELETE.
  • 7. Criar uma consulta com operadores lógicos (AND, OR, NOT).
  • 8. Usar a cláusula DISTINCT para exibir apenas valores únicos.
  • 9. Implementar funções básicas de agregação (SUM, AVG, MAX, MIN).
  • 10. Criar uma tabela com uma chave primária e exibir sua estrutura.

Desafios Médios

  • 11. Criar relacionamentos entre tabelas usando chaves estrangeiras.
  • 12. Implementar uma consulta com JOIN para combinar dados de duas tabelas.
  • 13. Criar uma tabela com colunas do tipo JSON e inserir dados estruturados.
  • 14. Criar e usar uma função personalizada no banco de dados.
  • 15. Configurar e usar índices para melhorar o desempenho de consultas.
  • 16. Criar um VIEW para exibir dados de forma simplificada.
  • 17. Utilizar transações para inserir múltiplos registros com segurança.
  • 18. Implementar uma consulta com subqueries.
  • 19. Criar uma tabela de logs para rastrear alterações de dados.
  • 20. Utilizar o comando COPY para importar dados de um arquivo CSV para uma tabela.

Desafios Difíceis

  • 21. Criar um sistema de auditoria com triggers para registrar alterações em uma tabela.
  • 22. Implementar uma função que calcula valores complexos com PL/pgSQL.
  • 23. Criar um índice GIN ou GiST para pesquisa em colunas JSONB.
  • 24. Configurar partições de tabela para gerenciar grandes volumes de dados.
  • 25. Implementar uma consulta recursiva usando CTE (Common Table Expressions).
  • 26. Configurar uma réplica de banco de dados para alta disponibilidade.
  • 27. Desenvolver um sistema de permissões detalhado para usuários do banco.
  • 28. Implementar consultas geoespaciais usando extensões como PostGIS.
  • 29. Criar uma função para gerar relatórios automaticamente em formato CSV.
  • 30. Desenvolver um sistema de histórico de alterações usando tabelas de versão.

Desafios Avançados

  • 31. Desenvolver um banco de dados com relacionamentos complexos (muitos-para-muitos).
  • 32. Implementar otimizações de consultas em tabelas com milhões de registros.
  • 33. Criar um sistema de backup automatizado e realizar sua restauração.
  • 34. Implementar consultas com funções de janela (window functions).
  • 35. Configurar replicação lógica para sincronizar dados entre servidores.
  • 36. Criar e gerenciar tabelas temporárias para análises dinâmicas.
  • 37. Desenvolver uma integração com aplicações externas usando LISTEN e NOTIFY.
  • 38. Utilizar particionamento de tabelas e demonstrar seus benefícios em performance.
  • 39. Criar um sistema de gerenciamento de filas com PostgreSQL.
  • 40. Desenvolver um sistema de relatórios dinâmicos com agregações e filtros personalizados.
