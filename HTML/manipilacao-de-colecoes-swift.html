Exercícios Fáceis

  • Crie um array de inteiros e imprima o elemento na posição 2.
  • Crie um dicionário com chaves e valores de strings e adicione uma nova chave-valor.
  • Crie um array de strings e use o método `append` para adicionar um elemento no final.
  • Crie uma tupla com três elementos e imprima cada um deles.
  • Crie um conjunto de números e adicione um novo número ao conjunto.
  • Verifique se um elemento específico está presente em um array e imprima o resultado.
  • Use um laço `for-in` para imprimir todos os elementos de um array.
  • Crie um dicionário e imprima todas as chaves presentes nele.
  • Utilize a função `count` para determinar o número de elementos em um array.
  • Crie um array de strings e utilize a função `sorted` para ordená-lo alfabeticamente.

Exercícios Médios

  • Crie um array de números e use a função `map` para dobrar o valor de cada elemento.
  • Crie um dicionário e use o método `keys` para obter um array com todas as chaves.
  • Filtre os elementos de um array de inteiros para obter apenas os números pares.
  • Crie um conjunto e use o método `intersection` para encontrar a interseção entre dois conjuntos.
  • Use a função `reduce` para calcular a soma de todos os elementos de um array de números.
  • Crie um array de strings e use o método `filter` para obter apenas as palavras que começam com a letra 'A'.
  • Crie uma tupla com diferentes tipos de dados e use a desestruturação para acessar seus valores.
  • Use o método `first` para obter o primeiro elemento de um array e imprima-o.
  • Crie um array de números e use `compactMap` para remover elementos nulos.
  • Implemente uma função que receba um array de strings e retorne um array com apenas as palavras que têm mais de 5 caracteres.

Exercícios Difíceis

  • Crie uma função que receba um array de números e retorne o maior e o menor valor em uma tupla.
  • Implemente um algoritmo que verifique se um array é um palíndromo (lê-se da mesma forma de frente para trás).
  • Crie um array de strings e use a função `reduce` para concatenar todos os elementos em uma única string, separada por espaços.
  • Crie um dicionário de alunos com notas e escreva uma função para calcular a média das notas.
  • Use `enumerated` para iterar sobre um array e imprimir o índice e o valor de cada elemento.
  • Crie um array de números e use o método `filter` para obter todos os números que são divisíveis por 3.
  • Implemente um programa que conte quantas vezes cada elemento aparece em um array e imprima o resultado.
  • Crie uma função que receba um array de strings e retorne um novo array com os elementos em ordem reversa.
  • Implemente uma função que verifique se um array contém elementos únicos (sem repetição).
  • Crie um programa que receba um array de tuplas (nome, idade) e retorne um array apenas com os nomes das pessoas maiores de 18 anos.

Desafios

  • Crie uma função que receba dois arrays de números e retorne um array com os elementos que aparecem em ambos os arrays.
  • Implemente uma função que receba um array de strings e retorne a string mais longa.
  • Crie um array de números e use a função `sorted` para classificá-lo em ordem decrescente.
  • Desenvolva uma função que use um dicionário para armazenar o número de vezes que cada palavra aparece em uma string.
  • Crie um programa que use um array de dicionários para armazenar informações de produtos em um estoque, e implemente uma função que retorne todos os produtos com preço acima de um determinado valor.
  • Implemente uma função que, dado um array de números, retorne o segundo maior número.
  • Crie um programa que use um array de estruturas (structs) para armazenar informações de alunos e imprima o aluno com a maior nota.
  • Desenvolva uma função que use a função `map` para criar um novo array onde cada elemento é o quadrado do elemento correspondente do array original.
  • Implemente uma função que receba um array de strings e retorne um array de strings em ordem alfabética, sem repetir elementos.
  • Crie uma função que utilize `zip` para combinar dois arrays em um dicionário onde as chaves são os elementos do primeiro array e os valores são os elementos do segundo array.
