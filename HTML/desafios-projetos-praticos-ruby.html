Desafios Fáceis

  • 1. Criar um programa que soma dois números e exibe o resultado.
  • 2. Implementar um script que verifica se um número é par ou ímpar.
  • 3. Desenvolver um programa para calcular a área de um triângulo.
  • 4. Criar uma lista de números e exibir apenas os números pares.
  • 5. Escrever um script que exibe "Olá, Mundo!" no terminal.
  • 6. Criar uma função que retorna o maior número em um array.
  • 7. Desenvolver um programa para contar até 10 usando `while`.
  • 8. Implementar um script que converte Celsius para Fahrenheit.
  • 9. Criar uma função que verifica se uma string é um palíndromo.
  • 10. Gerar números aleatórios entre 1 e 10 e exibir no terminal.

Desafios Médios

  • 11. Criar uma classe `Pessoa` com atributos `nome` e `idade` e exibir essas informações.
  • 12. Desenvolver um programa que ordena uma lista de números em ordem crescente.
  • 13. Implementar um script que lê dados de um arquivo CSV e os exibe no terminal.
  • 14. Criar um programa que calcula o fatorial de um número.
  • 15. Desenvolver uma função que calcula o número de vogais em uma string.
  • 16. Criar uma classe `ContaBancaria` com métodos para depositar e sacar dinheiro.
  • 17. Implementar um programa que simula o lançamento de dois dados e exibe o resultado.
  • 18. Desenvolver um script que utiliza hashes para armazenar informações de alunos.
  • 19. Criar uma função que encontra a palavra mais longa em uma frase.
  • 20. Escrever um programa que calcula o MDC (Máximo Divisor Comum) de dois números.

Desafios Difíceis

  • 21. Criar um sistema de gerenciamento de tarefas com opções de adicionar, listar e remover tarefas.
  • 22. Desenvolver um script que realiza operações CRUD em um banco de dados SQLite.
  • 23. Implementar uma classe `Animal` com herança para `Cachorro` e `Gato`, com métodos específicos para cada um.
  • 24. Criar um programa que analisa um texto e exibe a frequência de cada palavra.
  • 25. Desenvolver um sistema de autenticação simples com usuários e senhas armazenados em um hash.
  • 26. Implementar um script que interage com uma API externa para buscar informações de um filme.
  • 27. Criar um programa que simula um jogo da velha no terminal.
  • 28. Desenvolver uma função que encontra todos os números primos em um intervalo.
  • 29. Criar um sistema de reservas de hotel com classes e interações básicas.
  • 30. Implementar um script que gera gráficos simples no terminal usando gemas do Ruby.

Desafios Avançados

  • 31. Criar um sistema de inventário para um jogo, utilizando classes e heranças.
  • 32. Desenvolver uma aplicação web simples com Sinatra que exibe uma lista de tarefas.
  • 33. Implementar um programa que utiliza threads para executar múltiplas tarefas simultaneamente.
  • 34. Criar uma aplicação que monitora alterações em um diretório e exibe logs.
  • 35. Desenvolver uma API REST usando o framework Rails para gerenciar usuários e produtos.
  • 36. Criar um programa que simula uma fila de banco com múltiplos atendentes.
  • 37. Implementar um algoritmo para resolver o problema do caixeiro viajante (TSP).
  • 38. Criar um sistema de gerenciamento de biblioteca com funcionalidades completas de empréstimos.
  • 39. Desenvolver um jogo de texto no terminal, como "Adivinhe a Palavra".
  • 40. Criar um programa que analisa grandes arquivos de texto e exibe estatísticas detalhadas.
