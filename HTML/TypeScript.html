O que é TypeScript?

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É uma superconjunto do JavaScript que adiciona tipagem estática e outros recursos, melhorando a escalabilidade e a manutenção do código.

Características e Sintaxe do TypeScript

TypeScript é conhecido por suas características como tipagem estática, suporte a classes e interfaces, e compilação para JavaScript. Ele ajuda a detectar erros durante o desenvolvimento, tornando o código mais robusto e menos propenso a bugs.

Exemplo de Código TypeScript

let mensagem: string = "Olá, TypeScript!";
console.log(mensagem);
