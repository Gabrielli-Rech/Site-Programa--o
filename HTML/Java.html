O que é Java?

Java é uma linguagem de programação de alto nível, orientada a objetos e amplamente usada em desenvolvimento de aplicações empresariais, aplicações móveis (Android), sistemas corporativos e aplicações de servidor. É conhecida por seu lema "escreva uma vez, execute em qualquer lugar", devido à plataforma Java Virtual Machine (JVM).

Principais Características do Java

  • Portabilidade: O código Java pode ser executado em qualquer sistema que tenha a JVM.
  • Orientação a Objetos: Java é baseada em POO, o que permite a organização de código em classes e objetos.
  • Segurança: Oferece características robustas de segurança para desenvolver aplicações seguras.
  • Multithreading: Java suporta execução de múltiplas threads simultaneamente, tornando-o ideal para aplicações que exigem processamento paralelo.
  • Bibliotecas Amplas: A linguagem tem uma vasta coleção de bibliotecas e APIs para diversas funcionalidades.

Exemplo de Estruturas de Controle

// Estrutura de controle condicional
if (idade >= 18) {
    System.out.println("Você é maior de idade.");
} else {
    System.out.println("Você é menor de idade.");
}

// Estrutura de repetição
for (int i = 0; i < 5; i++) {
    System.out.println("Número: " + i);
}

Programação Orientada a Objetos (POO)

Java é conhecida por sua forte abordagem orientada a objetos, permitindo o desenvolvimento de software modular e reutilizável.

Exemplo de Classe e Objeto

// Definição de classe e criação de objeto
public class Pessoa {
    private String nome;
    private int idade;

    // Construtor
    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }

    // Método
    public String saudacao() {
        return "Olá, meu nome é " + nome + " e tenho " + idade + " anos.";
    }

    public static void main(String[] args) {
        Pessoa pessoa1 = new Pessoa("Carlos", 25);
        System.out.println(pessoa1.saudacao());  // Saída: Olá, meu nome é Carlos e tenho 25 anos.
    }
}
