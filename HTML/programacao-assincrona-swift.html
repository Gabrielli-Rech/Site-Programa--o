Exercícios de Programação Assíncrona

  • Crie uma função assíncrona que faça uma requisição HTTP e imprima a resposta no console.
  • Desenvolva um programa que execute várias tarefas em paralelo usando `DispatchGroup` e imprima o tempo total de execução.
  • Implemente um script Swift que use `async/await` para buscar dados de uma API e exiba os resultados de forma assíncrona.
  • Crie uma função que use `Task` para realizar duas operações simultâneas e aguarde ambas com `await`.
  • Escreva um programa que use `URLSession.shared.dataTask` para fazer uma requisição assíncrona e use `await` para esperar pela resposta.
  • Desenvolva uma função assíncrona que leia e escreva em arquivos de forma assíncrona usando `Task`.
  • Implemente uma função que utilize `async let` para criar variáveis assíncronas em um contexto paralelo e aguarde com `await`.
  • Crie uma aplicação que utilize `DispatchQueue` para executar tarefas em background e exiba os resultados no thread principal.
  • Desenvolva um programa que use a função `Task.detached` para executar uma tarefa independente e imprima uma mensagem após sua conclusão.
  • Escreva um script Swift que use `Task` e `withCheckedContinuation` para integrar código assíncrono com funções baseadas em closures.
  • Crie um aplicativo Swift que utilize `URLSession` com `async/await` para fazer chamadas assíncronas em um loop.
  • Desenvolva uma função que utilize `async/await` para fazer múltiplas requisições HTTP em paralelo e agregue os resultados em um array.
  • Implemente uma função que use `Task` para ler de uma base de dados local de forma assíncrona e exiba os dados em uma interface de usuário.
  • Crie um programa que use `DispatchSemaphore` para sincronizar tarefas assíncronas em um contexto de concorrência.
  • Desenvolva um script que utilize `async/await` para buscar e processar dados de uma API em um aplicativo iOS.
  • Escreva um código que use `Task` e `await` para criar uma função de contagem regressiva que emita um número a cada segundo.
  • Crie uma função assíncrona que receba um número de URLs e faça chamadas `async/await` para processar todas as respostas de forma paralela.
  • Desenvolva uma função que utilize `async let` e `await` para baixar múltiplos arquivos de uma URL em paralelo e exiba uma mensagem de conclusão.
  • Escreva um script que use `URLSession.data(for:delegate:)` e `await` para buscar e decodificar JSON de uma API de forma assíncrona.
  • Crie uma função que use `async/await` para realizar um processamento em série de uma lista de tarefas em um app Swift.
  • Desenvolva uma aplicação de exemplo que utilize a classe `Task` para atualizar a interface de usuário com resultados de uma operação assíncrona.
