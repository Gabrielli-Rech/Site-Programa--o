Exercícios Fáceis

  • 1. Crie uma lista com 5 elementos e imprima cada um deles.
  • 2. Adicione um elemento a uma lista e imprima a lista resultante.
  • 3. Remova o último elemento de uma lista e imprima a lista resultante.
  • 4. Escreva um programa que verifique se um elemento está presente em uma lista.
  • 5. Crie um dicionário com 3 chaves e imprima seus valores.
  • 6. Adicione uma nova chave a um dicionário e imprima o dicionário atualizado.
  • 7. Remova uma chave de um dicionário e imprima o dicionário resultante.
  • 8. Escreva um programa que imprima todas as chaves de um dicionário.
  • 9. Crie uma lista de números de 1 a 10 e imprima apenas os números pares.
  • 10. Escreva um programa que verifique a quantidade de elementos em uma lista.
  • 11. Crie uma lista de strings e imprima a lista em ordem inversa.
  • 12. Implemente um programa que encontre o maior elemento de uma lista.
  • 13. Escreva um programa que receba um dicionário e imprima seus valores de forma ordenada.
  • 14. Crie uma lista de números e calcule a soma dos elementos.
  • 15. Escreva um programa que imprima os valores de um dicionário em uma lista.
  • 16. Adicione um elemento no início de uma lista e imprima a lista.
  • 17. Remova o primeiro elemento de uma lista e imprima a lista.
  • 18. Crie uma lista de palavras e imprima a palavra mais longa.
  • 19. Escreva um programa que verifique se um dicionário está vazio.
  • 20. Crie um dicionário com elementos de uma lista como chaves e imprima-o.

Exercícios Médios

  • 1. Crie um programa que encontre o índice de um elemento em uma lista.
  • 2. Implemente uma função que receba uma lista e retorne a lista sem elementos duplicados.
  • 3. Escreva um programa que inverta a ordem dos elementos de uma lista.
  • 4. Crie um dicionário e adicione chaves com valores iguais ao índice de cada elemento em uma lista.
  • 5. Implemente um programa que calcule o produto dos elementos de uma lista.
  • 6. Escreva um programa que imprima a chave e o valor de um dicionário em uma linha separada.
  • 7. Crie uma função que receba uma lista e imprima o elemento mais frequente.
  • 8. Implemente um programa que verifique se todas as chaves de um dicionário possuem um valor numérico.
  • 9. Crie uma lista de números e use uma função de filtro para imprimir apenas os números ímpares.
  • 10. Escreva um programa que remova todas as chaves com valores nulos de um dicionário.
  • 11. Crie um programa que imprima os elementos de uma lista que estão presentes em outra lista.
  • 12. Implemente uma função que receba duas listas e retorne uma lista com a interseção entre elas.
  • 13. Crie uma função que imprima a quantidade de valores numéricos em um dicionário.
  • 14. Escreva um programa que concatene duas listas e imprima o resultado.
  • 15. Crie um dicionário de listas e imprima os elementos de uma lista específica do dicionário.
  • 16. Implemente um programa que classifique uma lista de strings em ordem alfabética.
  • 17. Escreva uma função que imprima a soma dos valores de um dicionário.
  • 18. Crie um programa que remova elementos que sejam menores que um valor dado em uma lista.
  • 19. Implemente um programa que crie um dicionário com as palavras de uma frase como chaves e a quantidade de vezes que aparecem como valores.
  • 20. Escreva um programa que substitua um elemento em uma lista por outro dado.

Exercícios Difíceis

  • 1. Crie uma função que receba duas listas e retorne a união delas sem elementos duplicados.
  • 2. Implemente um programa que encontre o elemento que ocorre mais vezes em uma lista.
  • 3. Escreva um programa que remova elementos de uma lista com base em seus índices.
  • 4. Crie uma função que inverta as chaves e valores de um dicionário.
  • 5. Implemente um programa que retorne a média dos valores numéricos em um dicionário.
  • 6. Escreva uma função que conte o número de elementos em uma lista que são divisíveis por um número dado.
  • 7. Crie um programa que encontre o índice do elemento mínimo em uma lista.
  • 8. Implemente uma função que verifique se duas listas têm os mesmos elementos, independentemente da ordem.
  • 9. Escreva um programa que crie uma nova lista com os elementos de duas listas, mas apenas aqueles que estão presentes em ambas.
  • 10. Crie uma função que receba uma lista e um dicionário e retorne uma lista de elementos que não estão no dicionário.
  • 11. Implemente um programa que gere um dicionário de frequências de letras em uma string.
  • 12. Escreva uma função que remova todas as chaves de um dicionário que tenham um valor menor que um valor específico.
  • 13. Crie um programa que calcule a soma de todos os valores de uma lista de dicionários com uma chave específica.
  • 14. Implemente uma função que receba um dicionário e uma chave, e imprima o valor da chave ou uma mensagem de erro.
  • 15. Escreva um programa que crie um dicionário de listas e imprima o elemento da lista com o maior número de elementos.
  • 16. Crie uma função que receba uma lista de palavras e retorne um dicionário com a contagem de palavras únicas.
  • 17. Implemente um programa que encontre os elementos comuns em três listas diferentes.
  • 18. Escreva uma função que substitua elementos em uma lista por seus respectivos índices.
  • 19. Crie um programa que verifique se uma lista é um anagrama de outra.
  • 20. Implemente uma função que encontre todos os elementos que são comuns a duas listas de maneira eficiente.

Desafios

  • 1. Crie um programa que leia uma lista de números e retorne o segundo maior elemento.
  • 2. Implemente um programa que crie um dicionário onde as chaves sejam os índices de uma lista e os valores, os elementos da lista.
  • 3. Escreva uma função que receba uma lista e retorne a soma dos elementos em ordem reversa.
  • 4. Crie uma função que crie uma matriz 3x3 a partir de uma lista e imprima a soma de cada linha.
  • 5. Implemente um programa que crie um dicionário com a soma dos elementos de sublistas em uma lista de listas.
  • 6. Escreva uma função que verifique se um dicionário é um anagrama de outro dicionário.
  • 7. Crie um programa que receba duas listas de diferentes tamanhos e retorne os elementos que aparecem em ambas.
  • 8. Implemente uma função que remova todos os elementos que aparecem em uma lista e que não tenham pelo menos duas ocorrências.
  • 9. Crie uma função que crie uma lista de palavras únicas de um texto e imprima a palavra com a maior quantidade de letras.
  • 10. Escreva um programa que crie uma lista de números inteiros a partir de uma lista de strings.
  • 11. Implemente um programa que verifique se uma lista de listas é uma matriz identidade.
  • 12. Crie uma função que inverta a chave e o valor de cada elemento em um dicionário de listas.
  • 13. Escreva um programa que calcule a mediana dos valores em uma lista.
  • 14. Implemente um programa que retorne uma lista com todos os elementos que aparecem em mais de uma lista fornecida.
  • 15. Crie uma função que imprima a tabela de multiplicação de um número usando uma lista de listas.
  • 16. Escreva um programa que encontre a média de todos os valores em um dicionário e imprima as chaves com valores acima da média.
  • 17. Implemente um programa que crie uma lista de pares únicos a partir de uma lista de números.
  • 18. Crie uma função que receba uma lista e um número e imprima a lista, removendo todos os elementos menores que o número.
  • 19. Escreva um programa que verifique se uma lista é um palíndromo.
  • 20. Implemente um programa que crie um dicionário e imprima os elementos cujos valores são listas com mais de 3 elementos.
