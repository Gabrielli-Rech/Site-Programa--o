Exercícios Fáceis

  • 1. Escreva um programa que verifique se um número é positivo, negativo ou zero.
  • 2. Crie um script que imprima "Maior de idade" se a idade for 18 ou mais, e "Menor de idade" caso contrário.
  • 3. Implemente um programa que verifique se um número é par ou ímpar.
  • 4. Crie um programa que leia um número e imprima "Divisível por 3" se for divisível por 3.
  • 5. Escreva um script que verifique se um número é maior que 10 e menor que 50.
  • 6. Implemente um programa que leia uma nota e imprima "Aprovado" se for maior ou igual a 6.
  • 7. Crie um script que leia dois números e imprima o maior.
  • 8. Escreva um programa que verifique se um número é divisível por 5 e por 10 ao mesmo tempo.
  • 9. Implemente um script que verifique se um ano é bissexto.
  • 10. Crie um programa que leia a idade de uma pessoa e imprima "Adulto" se for 18 anos ou mais, caso contrário "Jovem".
  • 11. Escreva um script que verifique se um número é um múltiplo de 7.
  • 12. Implemente um programa que leia uma palavra e verifique se ela é igual a "Python".
  • 13. Crie um script que leia um número e imprima "Positivo" ou "Negativo".
  • 14. Escreva um programa que determine se um número é menor que 10 e maior que 0.
  • 15. Implemente um script que leia um valor e imprima "Número válido" se for entre 1 e 100.
  • 16. Crie um programa que imprima "Passou no teste" se a nota for maior ou igual a 7.
  • 17. Escreva um script que determine se uma pessoa pode votar (idade >= 16).
  • 18. Implemente um programa que verifique se um número é múltiplo de 3 e 5.
  • 19. Crie um script que leia um número e imprima "Positivo", "Negativo" ou "Zero".
  • 20. Escreva um programa que determine se uma pessoa tem idade suficiente para dirigir (idade >= 18).

Exercícios Médios

  • 1. Escreva um programa que imprima todos os números entre 1 e 100 que são divisíveis por 3.
  • 2. Crie um script que leia a idade de uma pessoa e imprima "Infantil", "Adolescente", "Adulto" ou "Idoso".
  • 3. Implemente um programa que leia a temperatura e imprima "Quente", "Agradável" ou "Frio".
  • 4. Crie um script que verifique se um número é primo.
  • 5. Escreva um programa que imprima "Positivo" se um número for maior que 0, "Negativo" se for menor que 0, e "Zero" se for igual a 0.
  • 6. Implemente um script que determine se um dia da semana (representado por número) é um dia útil ou um fim de semana.
  • 7. Crie um programa que leia três números e imprima o maior e o menor.
  • 8. Escreva um script que verifique se uma palavra tem mais de 5 caracteres.
  • 9. Implemente um programa que leia um número e imprima "Par" ou "Ímpar".
  • 10. Crie um script que verifique se um ano é bissexto.
  • 11. Escreva um programa que imprima "É um triângulo" se a soma de três lados for maior que 0.
  • 12. Implemente um script que leia um número e verifique se é um número perfeito (soma dos divisores igual ao número).
  • 13. Crie um programa que leia uma nota e imprima "Aprovado" se for maior que 6, "Reprovado" caso contrário.
  • 14. Escreva um script que verifique se uma palavra é um palíndromo.
  • 15. Implemente um programa que verifique se um número é múltiplo de 3 ou 5.
  • 16. Crie um script que leia o nome de um mês e imprima quantos dias ele possui.
  • 17. Escreva um programa que imprima os números de 1 a 100 e, para múltiplos de 3, imprima "Fizz", e para múltiplos de 5, "Buzz".
  • 18. Implemente um script que leia dois números e imprima "Igual" se forem iguais, "Diferente" caso contrário.
  • 19. Crie um programa que imprima "Positivo" se um número for maior que 0, "Negativo" se for menor, e "Zero" se for igual a 0.
  • 20. Escreva um script que verifique se uma letra é uma vogal ou uma consoante.

  • 5. Crie um programa que imprima os números de 1 a 100, mas para múltiplos de 3 imprima "Fizz", para múltiplos de 5 imprima "Buzz" e para múltiplos de ambos imprima "FizzBuzz".
  • 6. Escreva um script que verifique se uma palavra é um anagrama de outra.
  • 7. Implemente uma função que calcule a mediana de uma lista de números.
  • 8. Crie um programa que leia um número e imprima "Primo" ou "Não Primo".
  • 9. Escreva um script que verifique se um número é um quadrado perfeito.
  • 10. Implemente um script que determine se uma lista está ordenada (crescente ou decrescente).
  • 11. Crie um programa que verifique se um número é um palíndromo.
  • 12. Escreva um script que imprima todos os números entre 1 e 100 que são divisíveis por 7 e não por 3.
  • 13. Implemente um programa que, dado um número, imprima a soma de seus divisores próprios.
  • 14. Crie um script que verifique se um ano é bissexto e imprima "Sim" ou "Não".
  • 15. Escreva um programa que resolva o problema da soma de dois números e verifique se o resultado é positivo, negativo ou zero.
  • 16. Implemente uma função que verifique se um triângulo é equilátero, isósceles ou escaleno.
  • 17. Crie um programa que implemente a lógica de um jogo de pedra, papel e tesoura.
  • 18. Escreva um script que verifique se um número é divisível por 2, 3, 5 e imprima a mensagem correspondente.
  • 19. Implemente um script que encontre o maior e o menor elemento em uma lista.
  • 20. Crie um programa que, dado um mês, imprima se é um mês de inverno, verão, outono ou primavera.
