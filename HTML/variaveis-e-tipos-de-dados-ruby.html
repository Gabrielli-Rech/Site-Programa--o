O que são Variáveis e Tipos de Dados?

Em Ruby, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. Ruby é uma linguagem de tipagem dinâmica, o que significa que o tipo da variável é determinado automaticamente com base no valor atribuído.

Declaração de Variáveis

Em Ruby, você declara uma variável simplesmente atribuindo um valor a ela. Não é necessário especificar o tipo explicitamente:

nome = "João"  # string
idade = 25       # inteiro
altura = 1.75    # float
ativo = true     # boolean

Os nomes das variáveis em Ruby seguem convenções específicas, como começar com uma letra ou sublinhado (_).

Tipos de Dados em Ruby

Ruby possui tipos de dados fundamentais que ajudam a estruturar os programas. Os principais tipos incluem:

  • String: Cadeias de caracteres. Exemplo: "Olá, mundo!"
  • Integer: Números inteiros. Exemplo: 42, -10
  • Float: Números de ponto flutuante. Exemplo: 3.14, -0.001
  • Boolean: Valores booleanos. Exemplo: true, false
  • Nil: Representa a ausência de valor. Exemplo: nil
  • Array: Conjunto ordenado de elementos. Exemplo: [1, 2, 3]
  • Hash: Estrutura de pares chave-valor. Exemplo: { nome: "Ana", idade: 30 }

Exemplo de diferentes tipos de dados em Ruby:

nome = "Maria"       # String
idade = 30         # Integer
ativo = false      # Boolean
altura = 1.68      # Float
dados = nil        # Nil
