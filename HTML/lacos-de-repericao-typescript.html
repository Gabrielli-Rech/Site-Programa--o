O que são Laços de Repetição?

Laços de repetição, também conhecidos como loops, são estruturas de controle que permitem executar um bloco de código repetidamente enquanto uma condição for verdadeira. Eles são fundamentais em programação para automatizar tarefas e processar listas de dados.

O Laço for

O laço for é utilizado quando se sabe o número de vezes que o bloco de código deve ser repetido. A estrutura do for é a seguinte:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Este exemplo imprime os números de 0 a 4 no console.

O Laço while

O laço while é usado quando não se sabe o número de repetições antecipadamente. O bloco de código é executado enquanto a condição especificada for verdadeira.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Este exemplo também imprime os números de 0 a 4 no console.

O Laço do...while

O laço do...while é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez, independentemente da condição.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Este exemplo também imprime os números de 0 a 4 no console.

O Laço for...of

O laço for...of é utilizado para iterar sobre elementos de coleções como arrays, strings e outros objetos iteráveis.

const frutas = ['maçã', 'banana', 'laranja'];
for (const fruta of frutas) {
    console.log(fruta);
}

Este exemplo imprime cada fruta da lista no console.

O Laço for...in

O laço for...in é usado para iterar sobre as propriedades de um objeto. É importante ter cuidado ao usá-lo com arrays, pois ele pode não ter o comportamento esperado.

const pessoa = { nome: 'João', idade: 30, cidade: 'São Paulo' };
for (const chave in pessoa) {
    console.log(chave + ': ' + pessoa[chave]);
}

Este exemplo imprime as propriedades e valores do objeto pessoa.
