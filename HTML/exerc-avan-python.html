Exercícios Avançados de Python

1. Manipulação de Dados e Estruturas

  • 1. Implemente uma função que faça análise estatística de uma lista de números e retorne a média, mediana e moda.
  • 2. Crie uma função que aceite um dicionário de listas e retorne a lista com a maior soma de elementos.
  • 3. Escreva uma função que encontre o menor número em uma lista de listas.
  • 4. Crie um programa que simule um jogo de tabuleiro, onde cada jogador tem uma lista de posições possíveis e realiza jogadas aleatórias.
  • 5. Implemente uma função que receba duas listas de diferentes tamanhos e retorne um dicionário onde as chaves são os elementos da lista menor e os valores, a soma dos elementos correspondentes da lista maior.

2. Programação Orientada a Objetos (POO)

  • 1. Crie uma classe que represente um livro, com atributos como título, autor, ano de publicação, e métodos para exibir as informações e calcular a idade do livro.
  • 2. Implemente um sistema de gerenciamento de banco de dados usando classes para representar tabelas e métodos para inserir, deletar e consultar registros.
  • 3. Escreva uma classe para gerenciar uma fila de tarefas, onde cada tarefa tem um nome e prioridade, e um método para retornar a tarefa de maior prioridade.
  • 4. Crie uma classe `Carro` com atributos de velocidade, modelo e cor, e métodos para aumentar a velocidade e exibir as características do carro.
  • 5. Implemente uma classe `ContaBancaria` com métodos para sacar, depositar e consultar saldo, além de uma classe `Cliente` que possua informações pessoais e uma lista de contas.

3. Manipulação Avançada de Strings e Arquivos

  • 1. Escreva uma função que leia um arquivo de texto e conte a frequência de cada palavra, ignorando pontuações e diferenciação de maiúsculas/minúsculas.
  • 2. Crie uma função que utilize expressões regulares para validar e extrair informações de um endereço de e-mail.
  • 3. Implemente uma função que leia um arquivo CSV e retorne um dicionário onde as chaves são os cabeçalhos e os valores são listas com os dados de cada coluna.
  • 4. Escreva um programa que leia um arquivo de texto, calcule o número total de palavras e imprima as 5 palavras mais frequentes.
  • 5. Crie uma função que receba uma string e retorne uma versão criptografada da string usando um algoritmo de cifra de substituição simples.

4. Programação Funcional

  • 1. Implemente uma função `map` que receba uma lista de números e retorne uma nova lista com cada número elevado ao quadrado.
  • 2. Crie uma função `filter` que retorne todos os números ímpares de uma lista de números.
  • 3. Escreva uma função `reduce` que receba uma lista de números e retorne a soma de todos os elementos.
  • 4. Implemente uma função que utilize a função `lambda` para filtrar elementos de uma lista que sejam divisíveis por 3.
  • 5. Crie um programa que use `map`, `filter` e `reduce` em uma sequência de operações sobre uma lista de palavras, retornando apenas as palavras que começam com uma vogal e somando o comprimento total dessas palavras.

5. Desafios Avançados

  • 1. Escreva uma função que resolva o problema do caixeiro viajante utilizando programação dinâmica.
  • 2. Implemente uma função que simule a execução de um algoritmo de ordenação (como quicksort) e calcule o tempo de execução usando `time`.
  • 3. Crie um gerador de senhas seguras que utilize `itertools` para gerar combinações de caracteres e aplique uma verificação de força de senha.
  • 4. Escreva um programa que faça a compressão de texto usando o algoritmo de Huffman.
  • 5. Implemente uma função que faça uma busca binária em uma lista ordenada e retorne o índice do elemento buscado.
