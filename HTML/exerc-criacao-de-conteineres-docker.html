Exercícios de Criação de Conteineres.

  • Voltar
  • Início
  • 1. Criar um contêiner Docker simples com a imagem do nginx e verificar se está em execução.
  • 2. Criar um contêiner Docker com a imagem do nginx e mapear a porta 80 do contêiner para a porta 8080 do host.
  • 3. Criar um contêiner Docker com a imagem do alpine e executar um comando echo "Olá, Docker!".
  • 4. Criar um contêiner Docker baseado em uma imagem do ubuntu e instalar o curl dentro do contêiner.
  • 5. Criar um contêiner Docker em modo interativo com a imagem do python e executar um script Python simples.
  • 6. Criar e executar um contêiner Docker com a imagem do mysql e inicializar um banco de dados com um script SQL.
  • 7. Criar um contêiner Docker que execute uma aplicação Node.js e expor a porta 3000.
  • 8. Criar um contêiner Docker com a imagem do mongo e realizar uma conexão de teste com o banco de dados.
  • 9. Criar um contêiner Docker a partir de um Dockerfile e compilar uma aplicação Java.
  • 10. Criar um contêiner Docker que execute um servidor web em Apache com arquivos de configuração personalizados.
  • 11. Criar um contêiner Docker que execute uma aplicação PHP com um servidor embutido.
  • 12. Configurar e criar um contêiner Docker que utilize variáveis de ambiente para configurar um contêiner do nginx.
  • 13. Criar um contêiner Docker a partir de uma imagem redis e verificar se o serviço está rodando corretamente.
  • 14. Criar e executar um contêiner Docker que utilize um arquivo docker-compose.yml para configurar múltiplos serviços.
  • 15. Criar um contêiner Docker que utilize uma imagem personalizada construída a partir de um Dockerfile.
  • 16. Criar um contêiner Docker que monte um volume local para persistência de dados.
  • 17. Criar um contêiner Docker com a imagem postgres e configurar um banco de dados com um script de inicialização.
  • 18. Criar e executar um contêiner Docker com a imagem do phpmyadmin para gerenciar um banco de dados MySQL.
  • 19. Criar um contêiner Docker que execute um script de backup de banco de dados e armazene os arquivos em um volume externo.
  • 20. Criar um contêiner Docker que execute um processo de build de uma aplicação React com o comando npm run build.
