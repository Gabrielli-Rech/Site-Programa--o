O que são Variáveis e Tipos de Dados?

Em Java, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. Existem diferentes tipos de dados que uma variável pode conter, e é importante entendê-los para programar de forma eficiente.

Declaração de Variáveis

Em Java, você pode declarar variáveis usando os seguintes tipos de declaração:

  • int: Usado para armazenar números inteiros.
  • double: Usado para armazenar números de ponto flutuante.
  • boolean: Usado para armazenar valores verdadeiro ou falso.
  • char: Usado para armazenar um único caractere.
  • String: Usado para armazenar uma sequência de caracteres (classe, não tipo primitivo).

Exemplo de declaração de variáveis:

String nome = "João"; // String
int idade = 25; // int
double altura = 1.75; // double
boolean ativo = true; // boolean

Tipos de Dados em Java

Java possui tipos de dados primitivos e tipos de dados por referência. Veja abaixo os principais tipos de dados:

  • Primitivos:
    • byte: Representa um número inteiro de 8 bits. Exemplo: 100
    • short: Representa um número inteiro de 16 bits. Exemplo: 32000
    • int: Representa um número inteiro de 32 bits. Exemplo: 123456
    • long: Representa um número inteiro de 64 bits. Exemplo: 1234567890123L
    • float: Representa números de ponto flutuante de 32 bits. Exemplo: 3.14f
    • double: Representa números de ponto flutuante de 64 bits. Exemplo: 3.141592653589
    • char: Representa um único caractere. Exemplo: 'A'
    • boolean: Representa um valor verdadeiro ou falso. Exemplo: true, false
  • Tipos por Referência: São usados para armazenar referências a objetos e incluem classes, interfaces e arrays.

Exemplo de diferentes tipos de dados:

String nome = "Maria"; // String
int idade = 30; // int
boolean ativo = false; // boolean
char inicial = 'M'; // char
