O que são Estruturas Condicionais?

As estruturas condicionais em JavaScript são usadas para executar diferentes blocos de código com base em uma condição. Elas são fundamentais para a lógica de programação, permitindo que você tome decisões com base em condições específicas.

Principais Estruturas Condicionais

Em JavaScript, as estruturas condicionais mais comuns são:

  • if: Executa um bloco de código se uma condição for verdadeira.
  • else: Executa um bloco de código se a condição do if for falsa.
  • else if: Verifica uma condição adicional se a condição do if for falsa.
  • switch: Usado para executar um bloco de código baseado no valor de uma variável.

Exemplo de uma estrutura condicional if:

let idade = 18;
if (idade >= 18) {
    console.log('Você é maior de idade.');
} else {
    console.log('Você é menor de idade.');
}

Exemplo de uma estrutura switch:

let dia = 3;
switch (dia) {
    case 1:
        console.log('Segunda-feira');
        break;
    case 2:
        console.log('Terça-feira');
        break;
    case 3:
        console.log('Quarta-feira');
        break;
    default:
        console.log('Dia inválido');
}
