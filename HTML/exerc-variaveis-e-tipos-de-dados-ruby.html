Exercícios Práticos

  • Crie uma variável chamada nome e atribua seu nome a ela. Imprima a variável na tela.
  • Crie uma variável idade e atribua a sua idade. Imprima a idade com uma mensagem, por exemplo: "Eu tenho X anos".
  • Crie uma variável altura e atribua um valor de ponto flutuante. Imprima a altura com uma mensagem formatada, como "Minha altura é X metros".
  • Crie uma variável is_estudante do tipo booleano e atribua um valor true ou false. Imprima a variável com uma mensagem como "Sou estudante?"
  • Crie um array frutas com pelo menos três elementos (por exemplo: "maçã", "banana", "laranja"). Imprima todos os elementos usando um loop.
  • Crie uma variável pessoa como um hash com chaves "nome" e "idade". Imprima a chave "nome".
  • Crie uma variável preco e atribua um valor monetário a ela. Imprima o valor formatado como moeda (por exemplo, "R$ 100,00").
  • Verifique e imprima se a variável idade é do tipo inteiro usando a função is_a?(Integer).
  • Crie uma variável mensagem do tipo string e use o método length para imprimir o comprimento da string.
  • Crie duas variáveis numéricas, num1 e num2, e imprima a soma, subtração, multiplicação e divisão de ambas.
  • Crie uma variável texto e use o método upcase para imprimir o texto em maiúsculas.
  • Crie uma variável nome_completo usando a concatenação de duas strings. Imprima o resultado com uma mensagem.
  • Crie uma variável data_atual usando o método Time.now para exibir a data atual em formato "dd/mm/aaaa".
  • Crie uma variável numero e use o método abs para imprimir o valor absoluto do número.
  • Crie uma variável temp_celsius e converta para Fahrenheit usando a fórmula: (C × 9/5) + 32.
  • Crie uma variável frase e use o método gsub para substituir uma palavra específica por outra.
  • Crie uma variável senha do tipo string e use o método hash para imprimir o hash MD5 da senha.
  • Crie uma variável var de tipo misto e imprima o tipo de dado usando o método class.
  • Crie uma variável numero_grande do tipo inteiro e imprima o número formatado com o método to_s.reverse para separá-lo em grupos de três dígitos.
  • Crie uma variável nome e use o método split para separar uma string em um array de palavras.
  • Crie uma variável email e use uma expressão regular para validar se é um email válido.
  • Crie uma variável mensagem e use o método slice para imprimir os primeiros 5 caracteres da string.
  • Crie uma variável quantidade e use o método round para arredondar o valor para o inteiro mais próximo.
  • Crie uma variável pessoa e use o método is_a?(Hash) para verificar se a variável é um hash.
  • Crie uma variável mensagem e use o método reverse para imprimir a string ao contrário.
  • Crie uma variável valor e use o método numeric? para verificar se é um número.
  • Crie uma variável vazio e use o método empty? para verificar se a variável está vazia.
  • Crie uma variável divisao e use o método divmod para fazer a divisão inteira de dois números.
