O que são Variáveis e Tipos de Dados?

Em JavaScript, variáveis são usadas para armazenar dados que podem ser manipulados em um programa. Existem diferentes tipos de dados que uma variável pode conter, e é importante entendê-los para programar de forma eficiente.

Declaração de Variáveis

Em JavaScript, você pode declarar variáveis usando var, let e const.

  • var: Usado para declarar variáveis com escopo de função (escopo global fora de funções).
  • let: Introduzido no ES6, tem escopo de bloco e é recomendado para variáveis que podem mudar de valor.
  • const: Também introduzido no ES6, é usado para declarar variáveis cujo valor não muda (constantes).

Exemplo de declaração de variáveis:

let nome = 'João'; // string
const idade = 25; // number
var ativo = true; // boolean

Tipos de Dados em JavaScript

JavaScript possui tipos de dados primitivos e complexos. Veja abaixo os principais tipos de dados:

  • Primitivos:
    • string: Representa uma sequência de caracteres. Exemplo: 'Olá, mundo!'
    • number: Representa números inteiros e de ponto flutuante. Exemplo: 42, 3.14
    • boolean: Representa um valor verdadeiro ou falso. Exemplo: true, false
    • null: Representa um valor nulo. Exemplo: null
    • undefined: Representa uma variável declarada, mas ainda não inicializada. Exemplo: let x;
    • symbol: Introduzido no ES6, representa um valor único e imutável. Exemplo: Symbol('descricao')
  • Objetos: São tipos de dados complexos e podem conter múltiplos valores. Exemplos incluem objetos literais, arrays e funções.

Exemplo de diferentes tipos de dados:

let nome = 'Maria'; // string
let idade = 30; // number
let ativo = false; // boolean
let endereco = null; // null
let usuario; // undefined
