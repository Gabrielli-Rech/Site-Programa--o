Exercícios Básicos

  1. Crie um método que receba dois números como argumentos e retorne a soma deles.
  2. Escreva uma função chamada `saudacao` que receba um nome como argumento e imprima "Olá, [nome]!"
  3. Crie um método que calcule o quadrado de um número recebido como argumento.
  4. Implemente uma função que receba um número como argumento e retorne `true` se for par, ou `false` se for ímpar.
  5. Faça um método chamado `multiplicar` que receba dois números e retorne o produto deles.
  6. Crie um método que converta graus Celsius para Fahrenheit. Fórmula: \( F = C \times 9/5 + 32 \).
  7. Implemente uma função que calcule a área de um círculo dado o raio.
  8. Escreva um método que receba uma string como argumento e retorne o número de caracteres.
  9. Faça uma função chamada `maior_numero` que receba dois números e retorne o maior.
  10. Implemente um método chamado `reverter_string` que receba uma string e retorne ela invertida.

Exercícios Intermediários

  1. Crie um método chamado `fatorial` que calcule o fatorial de um número inteiro.
  2. Faça um método que receba uma lista de números e retorne apenas os números pares.
  3. Implemente uma função que receba uma string e retorne `true` se for um palíndromo.
  4. Crie um método chamado `media` que receba um array de números e retorne a média aritmética.
  5. Faça uma função que receba um array e retorne o maior número contido nele.
  6. Implemente uma função chamada `contar_vogais` que receba uma string e conte o número de vogais.
  7. Crie um método que calcule a soma dos números de uma sequência entre dois valores.
  8. Faça uma função chamada `potencia` que receba dois números (base e expoente) e calcule a potência.
  9. Implemente um método que substitua todas as vogais de uma string por `*`.
  10. Crie um método que receba um número e determine se ele é primo.

Exercícios Avançados

  1. Crie um método que receba um número inteiro e retorne todos os seus divisores.
  2. Faça uma função chamada `ordenar_array` que ordene um array em ordem crescente sem usar métodos prontos.
  3. Implemente uma função que calcule o MDC (máximo divisor comum) entre dois números.
  4. Crie um método chamado `fibonacci` que retorne os `n` primeiros números da sequência de Fibonacci.
  5. Faça um método que receba uma matriz 2x2 e retorne sua transposta.
  6. Implemente uma função chamada `remover_duplicados` que receba um array e remova elementos duplicados.
  7. Crie um método que receba um texto e retorne o número de palavras únicas.
  8. Faça uma função que receba um número inteiro e retorne ele em formato binário.
  9. Implemente um método chamado `ordenar_por_tamanho` que ordene uma lista de strings pelo tamanho.
  10. Crie um método que calcule o valor de um número elevado a outro usando recursão.

Exercícios de Desafios

  1. Implemente uma função que receba um texto e identifique a palavra mais frequente.
  2. Crie um método que resolva uma equação quadrática com base nos coeficientes `a`, `b` e `c`.
  3. Faça uma função que gere todas as permutações possíveis de uma string.
  4. Implemente uma função chamada `somar_diagonais` que receba uma matriz quadrada e calcule a soma das diagonais.
  5. Crie um método que receba uma lista de números e encontre a subsequência com a maior soma.
  6. Implemente um algoritmo que determine se duas strings são anagramas.
  7. Faça uma função que receba um número inteiro e retorne todos os números primos menores que ele.
  8. Crie um método chamado `resolver_hanoi` que resolva o problema das Torres de Hanoi para `n` discos.
  9. Implemente uma função que converta um número em romano para decimal.
  10. Faça um método que calcule o determinante de uma matriz 3x3.
